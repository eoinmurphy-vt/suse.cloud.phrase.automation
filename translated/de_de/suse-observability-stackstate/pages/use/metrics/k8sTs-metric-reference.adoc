== Übersicht

Auf dieser Seite werden allgemeine Kubernetes-bezogene Metriken definiert, die von SUSE Observability erfasst werden, und es wird erklärt, wie sie berechnet werden.

=== container_memory_usage

[horizontal]
Name:: `+container_memory_usage+`
Typ:: Messgerät
Einheit:: Bytes
Quelle:: Direktes Lesen aus der Hierarchie der Container-C-Gruppen.

[cols="1,3"]
|===
|cgroup-Version | Quelldatei

|v1 | `+memory.usage_in_bytes+`
|v2 | `+memory.current+`
|===

Zusammenfassung:: Die Gesamtmenge an Speicher, die derzeit von der cgroup und ihren Nachfolgern verwendet wird. Dies ist eine Summe, die anonymen Speicher (RSS), Seiten-Cache und Kernel-Speicher umfasst. Verwenden Sie eine der folgenden Metriken, um weitere Details über einen bestimmten Speicherbereich zu erhalten.

=== container_memory_kernel

[horizontal]
Name:: `+container_memory_kernel+`
Typ:: Messgerät
Einheit:: Bytes
Quelle:: Direktes Lesen aus der Hierarchie der Container-C-Gruppen.

[cols="1,3"]
|===
|cgroup-Version | Quelldatei

|v1 | `+memory.kmem.usage_in_bytes+`
|v2 | `+memory.stat+` -> `+kernel+` Feld innerhalb der Datei
|===
Zusammenfassung:: Gesamtmenge des Kernelspeichers, einschließlich (kernel_stack, pagetables, percpu, vmalloc, slab)

=== container_memory_rss

[horizontal]
Name:: `+container_memory_rss+`
Typ:: Messgerät
Einheit:: Bytes
Quelle:: Direktes Lesen aus der Hierarchie der Container-C-Gruppen.

[cols="1,3"]
|===
|cgroup-Version | Quelldatei

|v1 | `+memory.stat+` -> `+total_rss+` Feld innerhalb der Datei
|v2 | `+memory.stat+` -> `+anon+` Feld innerhalb der Datei
|===
Zusammenfassung:: Die Resident Set Size (RSS) von Anonymous Memory. Dies ist der nicht dateigebundene Speicher (Heap und Stack), der von den Prozessen innerhalb der cgroup aktiv genutzt wird. Dieser Wert spiegelt oft die tatsächliche "Arbeitsmenge" der Anwendung wider. 

=== container_memory_cache

[horizontal]
Name:: `+container_memory_cache+`
Typ:: Messgerät
Einheit:: Bytes
Quelle:: Direktes Lesen aus der Hierarchie der Container-C-Gruppen.

[cols="1,3"]
|===
|cgroup-Version | Quelldatei

|v1 | `+memory.stat+` -> `+total_cache+` Feld innerhalb der Datei
|v2 | `+memory.stat+` -> `+file+` Feld innerhalb der Datei
|===
Zusammenfassung:: Der Seiten-Cache (Speicher, der zum Zwischenspeichern von Dateisystemdaten verwendet wird, einschließlich tmpfs und Shared Memory).
