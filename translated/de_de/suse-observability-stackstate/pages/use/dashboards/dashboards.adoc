= Dashboards
:revdate: 2025-11-24
:page-revdate: {revdate}
:description: Erstellen und verwalten Sie Dashboards zur Visualisierung von Daten aus Ihren Ressourcen.

Ein Dashboard visualisiert Daten aus Ihren Ressourcen und kann nach Ihren Bedürfnissen erstellt und angepasst werden. Ein Dashboard kann Ressourcen aus verschiedenen Bereichen Ihrer IT-Umgebung enthalten.

== Ein neues Dashboard erstellen

So erstellen Sie Ihr erstes Dashboard:

. Klicken Sie im Hauptmenü auf **Dashboards**.
. Klicken Sie auf die Schaltfläche **Neues Dashboard hinzufügen** in der Mitte des Bildschirms.

So erstellen Sie ein neues Dashboard:

. Klicken Sie im Hauptmenü auf **Dashboards**.
. Klicken Sie oben rechts auf die Schaltfläche **Neues Dashboard hinzufügen**.
. Geben Sie den **Namen** und die **Beschreibung** ein.
. (Optional) Wählen Sie **Öffentlich**, um das Dashboard für alle Benutzer sichtbar zu machen.
. (Optional) Wählen Sie **Sternchen**, um das Dashboard für den schnellen Zugriff zum Hauptmenü hinzuzufügen.
. Klicken Sie auf **Dashboard erstellen**.

Sie werden zum neuen leeren Dashboard navigiert.

== Eigenschaften des Dashboards

Sobald Sie sich in einem Dashboard befinden, können Sie die Eigenschaften eines Dashboards bearbeiten, indem Sie das Dropdown-Menü neben der Schaltfläche *Speichern...* in der oberen Navigationsleiste verwenden und **Dashboard-Eigenschaften bearbeiten** auswählen. Von hier aus können Sie den Namen und die Beschreibung aktualisieren und festlegen, ob das Dashboard öffentlich oder mit einem Sternchen versehen sein soll.

image::dashboards/edit-dashboard-properties-button.png[Standort der Dashboard-Eigenschaften bearbeiten]

image::dashboards/edit-dashboard-properties-drawer.png[Schublade für Dashboard-Eigenschaften bearbeiten]

=== Öffentliche Dashboards
Standardmäßig können nur Sie Ihre Dashboards sehen. Wenn Sie ein Dashboard öffentlich machen, ist es für alle Benutzer sichtbar. Öffentliche Dashbords sind für andere sichtbar, aber nicht bearbeitbar.

=== Mit Sternen versehene Dashboards
Wenn Sie ein Dashboard mit einem Sternchen versehen, wird es im Hauptmenü und beim Hinzufügen von Widgets von anderen Seiten deutlicher hervorgehoben. Markieren Sie ein Dashboard, wenn Sie schnell darauf zugreifen möchten.

Mit Sternen versehene Dashboards sind auf Ihren Benutzer beschränkt. Andere Benutzer haben ihre eigenen mit Sternen versehenen Dashboards.

Sie können ein Dashboard auch mit einem Stern versehen oder den Stern entfernen, wenn Sie das Sternsymbol neben dem Namen des Dashboards sehen.

== Dashboard-Einstellungen

Mit den Dashboard-Einstellungen können Sie festlegen, ob der Zeitbereich, das Datenaktualisierungsintervall und die Variablen im Dashboard gespeichert oder von den Betrachtern angepasst werden sollen. Klicken Sie auf das Zahnradsymbol oben rechts in einem Dashboard, um die Dashboard-Einstellungen zu öffnen.

Wenn Sie eines der Kontrollkästchen unter der Überschrift "Speichern..." aktivieren, z. B. den Zeitbereich, wird das Dashboard nach dem Speichern auf den gewählten Zeitbereich festgelegt. Wenn sie deaktiviert ist, kann der Zeitbereich vom Betrachter des Dashboards gewählt werden; die Änderung wird in der URL widergespiegelt, so dass der Link mit anderen Personen geteilt werden kann, die dann das Dashboard mit demselben Zeitbereich betrachten können.
