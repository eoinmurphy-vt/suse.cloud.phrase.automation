= Benachrichtigungen
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: SUSE Observabilität

Mit Benachrichtigungen in SUSE Observability können Drittanbieteranwendungen benachrichtigt werden, wenn ein xref:/use/alerting/k8s-monitors.adoc[Monitor] in SUSE Observability ausgelöst wird. Benachrichtigungen lassen sich in Anwendungen von Drittanbietern wie Incident-Management-Tools (z. B. PagerDuty oder Opsgenie), Ticketing-Systeme (z. B. JIRA, ServiceNow) oder andere Plattformen für die Zusammenarbeit (z. B. Slack) integrieren. Andere Tools bezeichnen Benachrichtigungen als Alarme.

Benachrichtigungen werden durch Änderungen des Zustands von Monitoren auf SUSE Observability-Komponenten ausgelöst. Über die konfigurierten Benachrichtigungskanäle werden die Zustands-, Komponenten- und Überwachungsinformationen an diese externen Systeme gesendet.
