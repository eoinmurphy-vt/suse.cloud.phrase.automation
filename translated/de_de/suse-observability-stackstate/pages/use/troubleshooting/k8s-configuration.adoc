= YAML-Konfiguration
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: SUSE Observabilität

Treten bei einem Pod Probleme auf, wie z.B. Abstürze, Fehlstarts oder Fehlkonfigurationen, kann eine Überprüfung der YAML-Konfiguration helfen, die Ursache zu identifizieren. Sie können die YAML-Konfiguration einsehen, indem Sie auf die Schaltfläche *Konfiguration anzeigen* klicken.

image::k8s/k8s-configuration.png[]

Dies ergibt die gleiche Ausgabe wie der Befehl `+kubectl describe+`, der auf einen Dienst, einen Pod oder eine andere Ressource angewendet wird.

image::k8s/k8s-configuration-opened.png[]
