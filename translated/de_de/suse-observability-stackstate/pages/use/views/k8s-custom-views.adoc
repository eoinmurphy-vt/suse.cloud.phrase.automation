= Benutzerdefinierte Ansichten
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: SUSE Observabilität

== Übersicht

Während eine xref:/use/views/k8s-views.adoc[Kubernetes-Ansicht] als Ausgangspunkt für die Erkundung eines bestimmten Teils Ihrer IT-Landschaft dient und eine xref:/use/views/k8s-explore-views.adoc[Explore-Ansicht] es Ihnen ermöglicht, eine Teilmenge einer bestimmten Ansicht zu untersuchen, bietet eine *benutzerdefinierte Ansicht* die Möglichkeit, zu jeder dieser Ansichten zurückzukehren.

Mit anderen Worten: Sie können eine benutzerdefinierte Ansicht erstellen, indem Sie eine xref:/use/views/k8s-views.adoc[Kubernetes-Ansicht] oder eine xref:/use/views/k8s-explore-views.adoc[Explore-Ansicht] mit Ihren eigenen Einstellungen (Filter, Visualisierungsoptionen, Ansichtseinstellungen, Zeitleistenkonfiguration) speichern, um einen Teil Ihrer Topologie zu markieren, der für Sie oder Ihr Team von besonderem Interesse ist.

[NOTE]
====
Standardmäßig sind benutzerdefinierte Ansichten für alle Benutzer sichtbar. Diese können bei Bedarf von einem *SUSE Observability-Administrator* gesichert werden.
====


image::k8s/k8s-custom-view.png[Benutzerdefinierte Ansicht]

== Handhabung

Alle benutzerdefinierten Ansichten finden Sie unter dem Eintrag *Ansichten* im Hauptmenü.

Wenn Sie eine bestimmte benutzerdefinierte Ansicht häufig verwenden, können Sie sie zu den *markierten Ansichten* hinzufügen. Alle mit einem Sternchen versehenen Ansichten werden direkt im Abschnitt *Ansichten* des Hauptmenüs aufgelistet, um einen schnellen Zugriff zu ermöglichen - Sie erkennen die mit einem Sternchen versehenen Ansichten an dem gelben Sternsymbol neben ihrem Namen. Fügen Sie eine benutzerdefinierte Ansicht zu den mit Sternen versehenen Ansichten hinzu oder entfernen Sie sie, indem Sie einfach auf das Sternsymbol neben dem Namen klicken.

=== Eine Ansicht erstellen

Um eine neue benutzerdefinierte Ansicht zu erstellen, klicken Sie auf die blaue Schaltfläche *Ansicht speichern unter...* in der oberen Navigationsleiste, wenn Sie sich in einer xref:/use/views/k8s-views.adoc[Kubernetes-Ansicht] oder einer xref:/use/views/k8s-explore-views.adoc[Explore-Ansicht] befinden. Um eine neue Ansicht aus einer geänderten benutzerdefinierten Ansicht zu erstellen, verwenden Sie das Dropdown-Menü neben der Schaltfläche und wählen Sie *Ansicht speichern unter...*.

image::k8s/k8s-custom-view-edit-settings.png[Einstellungen der Ansicht bearbeiten]

Im Dialogfeld *Ansicht speichern unter* können die folgenden Optionen eingestellt werden:

|===
| Feld Name | Beschreibung

| Name anzeigen
| Der Name der Ansicht.

| Beschreibung
| Eine kurze Beschreibung für die Ansicht. Es hilft, sich den Umfang der Ansicht zu vergegenwärtigen.

| Identifier
| Eine eindeutige Referenz für die Ansicht, die hilfreich ist, wenn Sie die Ansicht aus einer exportierten Konfiguration referenzieren möchten, z. B. die exportierte Konfiguration in einem StackPack.
|===

=== Zurücksetzen einer Ansicht

Wenn eine benutzerdefinierte Ansicht erstellt wird, werden alle Filter, Visualisierungsoptionen, Ansichtseinstellungen und die Konfiguration der Zeitleiste in der Ansicht gespeichert. Wenn Sie die benutzerdefinierte Ansicht auf ihren ursprünglichen Zustand zurücksetzen möchten, nachdem Sie einige Änderungen daran vorgenommen haben, verwenden Sie das Dropdown-Menü neben der Schaltfläche *Speichern...* in der oberen Navigationsleiste und wählen Sie *Ansicht zurücksetzen*.

=== Löschen oder Bearbeiten einer Ansicht

[NOTE]
====
Beachten Sie, dass Änderungen, die Sie an einer benutzerdefinierten Ansicht vornehmen, für alle Benutzer übernommen werden.
====


Eine benutzerdefinierte Ansicht kann entweder auf dem Bildschirm *Alle Ansichten* oder auf dem Bildschirm *Benutzerdefinierte Ansicht* bearbeitet oder gelöscht werden.

. *Alle Ansichten* auf dem Bildschirm:
 ** Klicken Sie im Hauptmenü auf *Ansichten*, um den Bildschirm *Alle Ansichten* zu öffnen.
 ** Bewegen Sie den Mauszeiger über die Ansicht, die Sie bearbeiten oder löschen möchten.
 ** Klicken Sie auf die Schaltfläche *Ansichtseinstellungen bearbeiten* oder *Ansicht löschen* auf der rechten Seite.
. *Benutzerdefinierter Ansichtsbildschirm*:
 ** Öffnen Sie die Ansicht.
 ** Öffnen Sie das Dropdown-Menü neben der Schaltfläche *Speichern...* in der oberen Navigationsleiste.
 ** Klicken Sie auf *Ansichtseinstellungen bearbeiten* oder *Ansicht löschen*.

== Struktur

Eine benutzerdefinierte Ansicht hat eine identische Struktur wie die xref:/use/views/k8s-views.adoc[Kubernetes-Ansicht] oder die xref:/use/views/k8s-explore-views.adoc[Explore-Ansicht], aus der sie erstellt wurde: Sie hat die gleichen xref:/use/views/k8s-view-structure.adoc#_filters[Filter] und die gleichen xref:/use/views/k8s-view-structure.adoc#_perspectives[Perspektiven].
