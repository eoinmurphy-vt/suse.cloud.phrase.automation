*Lesen Sie die Anweisungen auf dieser Seite, bevor Sie ein Upgrade durchführen!*

Auf dieser Seite finden Sie spezifische Anweisungen und Details zu den erforderlichen manuellen Schritten für ein Upgrade auf die einzelnen unterstützten Versionen von SUSE Observability. Alle wesentlichen Änderungen, die sich auf die Ausführung von SUSE Observability nach dem Upgrade auswirken können, werden hier beschrieben, z. B. Änderungen der Speicheranforderungen oder der Konfiguration.

*Lesen und befolgen Sie alle Anweisungen für die Version, die Sie derzeit verwenden, bis hin zu der Version, auf die Sie aktualisieren werden.*
