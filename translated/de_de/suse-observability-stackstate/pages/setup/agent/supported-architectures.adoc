= Von SUSE Observability Agent unterstützte Architekturen
:revdate: 2025-07-10
:page-revdate: {revdate}

SUSE Observability wurde für die Ausführung auf bestimmten Hardware-Architekturen entwickelt und getestet. Installation und Betrieb werden auf diesen Systemen unterstützt:

* `+x86_64+` (auch bekannt als AMD64 oder Intel 64)
* `+arm64+` (auch bekannt als AArch64)

Vergewissern Sie sich, dass Ihre Zielumgebung diese Anforderungen erfüllt, bevor Sie mit der Bereitstellung fortfahren. Die Installation oder Ausführung von SUSE Observability auf anderen Architekturen wird nicht unterstützt.
