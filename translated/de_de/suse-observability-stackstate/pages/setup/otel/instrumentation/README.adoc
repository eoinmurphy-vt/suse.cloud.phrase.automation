= README
:description: SUSE Observabilität

Open Telemetry bietet https://opentelemetry.io/docs/languages/[SDKs für viele Sprachen] zur (automatischen) Instrumentierung Ihrer Anwendung. Die Instrumentierung ermöglicht es Ihrer Anwendung, Traces, Metriken und Protokolle an SUSE Observability zu senden. Die automatische Instrumentierung erfasst Traces, Metriken und Protokolle für gängige Bibliotheken und Frameworks, ohne dass ein weiteres manuelles Eingreifen erforderlich ist, es sei denn, Sie binden das SDK in Ihre Anwendung ein.

Open Telemetry hat viel mehr SDKs als hier dokumentiert sind. Wenn Ihre Sprache hier nicht dokumentiert ist, können Sie die Dokumentation unter https://opentelemetry.io/docs/languages/[opentelemetry.io] in Kombination mit der für SUSE Observability erforderlichen xref:/setup/otel/instrumentation/sdk-exporter-config.adoc[SDK-Exporter-Konfiguration] verwenden.
