Dieser Abschnitt dokumentiert das *UI-Präsentationsmodell*, das mit *StackPacks 2.0* eingeführt wurde.

== Übersicht

Das hier dokumentierte Komponenten-Präsentationsmodell legt fest, wie Komponenten (z.B. Pods, VMs, Dienste) in der Benutzeroberfläche {stackstate-product-name} dargestellt werden.
Sie konzentriert sich ausschließlich auf die *Präsentation*, z. B. Benennung, Symbole, Tabellen, Übersichts- und Hervorhebungsseiten, Filter und die Platzierung von Menüpunkten.

Das Ziel dieses Modells ist es, im Vergleich zum bestehenden, eher fragmentierten Ansatz, eine besser komponierbare und erweiterbare Methode zur Definition des UI-Verhaltens für Komponenten zu bieten.
