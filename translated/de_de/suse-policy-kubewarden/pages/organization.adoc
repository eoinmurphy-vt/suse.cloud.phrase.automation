Das meiste Material befindet sich in den vier Abschnitten _Tutorials_, _Erklärungen_,_Anleitungen_ und _Referenzen_. Es kann zu Überschneidungen kommen, da sich die Abschnitte nicht gegenseitig ausschließen. Das Projekt {short-project-name} versucht, demlink:https://diataxis.fr/[Diátaxis] Dokumentationsrahmen zu folgen.

== Navigation

Das Inhaltsmenü auf der linken Seite zeigt immer an, wo sich Ihr Dokument gerade befindet.
Das Menü auf der rechten Seite zeigt die aktuellen Seitenabschnitte an. Es gibt auch eine "Breadcrumbs"-Funktion am oberen Rand des Hauptinhaltsbereichs auf jeder Seite.

== Suche

In der Kopfzeile jeder Seite befindet sich ein Suchfeld.

== Bearbeitung von

Ein Link zum Quelltext der Seite auf GitHub befindet sich am Ende jeder Seite.
