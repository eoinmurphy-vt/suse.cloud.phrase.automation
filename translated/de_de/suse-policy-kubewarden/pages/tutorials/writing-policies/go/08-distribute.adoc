Es war ein langer und hoffentlich unterhaltsamer Weg durch das Go-Tutorial. Ich gratuliere Ihnen, dass Sie es bis hierher geschafft haben 🎉.

In diesem Tutorial haben Sie diehttps://github.com/kubewarden/safe-labels-policy[Richtlinie] neu erstellt.

Es gibt nichts Besonderes zu tun, wenn es darum geht, eine Police zu verteilen.
Wenn Sie diesen Leitfaden befolgt haben, haben Sie Ihre Richtlinie bereits mit der im letzten Kapitel beschriebenen GitHub `+release.yml+` Aktion veröffentlicht.

Das Thema der Verteilung von Richtlinien wird in der Dokumentation von {project-name} im Abschnittxref:/explanations/distributing-policies.adoc["Verteilen von Richtlinien"]ausführlicher behandelt.
