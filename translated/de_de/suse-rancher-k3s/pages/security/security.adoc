Dieser Abschnitt beschreibt die Methodik und die Mittel zur Sicherung eines K3s-Clusters. Sie ist in 2 Abschnitte unterteilt. Diese Anleitungen gehen davon aus, dass k3s mit eingebettetem etcd läuft.

Der Härtungsleitfaden enthält zunächst eine Liste bewährter Sicherheitspraktiken zur Sicherung eines K3s-Clusters.
