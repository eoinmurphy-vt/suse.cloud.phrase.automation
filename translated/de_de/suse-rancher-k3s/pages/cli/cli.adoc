= CLI-Werkzeuge
:page-languages: [en, ja, ko, zh]
:revdate: 2024-09-11
:page-revdate: {revdate}

Die K3s-Binärdatei enthält eine Reihe von zusätzlichen Werkzeugen, die Ihnen bei der Verwaltung Ihres Clusters helfen.

|===
| Befehl | Beschreibung

| `+k3s server+`
| Starten Sie einen K3s-Serverknoten, der die Kubernetes-Komponenten `+apiserver+`, `+scheduler+`, `+controller-manager+` und `+cloud-controller-manager+` sowie einen Datenspeicher und die Agentenkomponenten startet. Weitere Informationen finden Sie in der xref:cli/server.adoc[Dokumentation zum Befehl`+k3s server+` ].

| `+k3s agent+`
| Führen Sie den K3s-Agentenknoten aus, der `+containerd+`, `+flannel+`, `+kube-router+` Netzwerkrichtlinien-Controller und die Kubernetes-Komponenten `+kubelet+` und `+kube-proxy+` startet. Weitere Informationen finden Sie in der xref:cli/agent.adoc[Dokumentation zum Befehl`+k3s agent+` ].

| `+k3s kubectl+`
| Führen Sie den eingebetteten https://kubernetes.io/docs/reference/kubectl[Befehl] Dies ist eine CLI für die Interaktion mit dem Kubernetes-Apiserver.  Wenn die Umgebungsvariable `+KUBECONFIG+` nicht gesetzt ist, wird automatisch versucht, die kubeconfig unter `+/etc/rancher/k3s/k3s.yaml+` zu verwenden.

| `+k3s crictl+`
| Führen Sie den eingebetteten https://github.com/kubernetes-sigs/cri-tools/blob/master/docs/crictl.md[Befehl] Dies ist eine CLI für die Interaktion mit der Container-Laufzeit-Schnittstelle (CRI) von Kubernetes. Nützlich für die Fehlersuche.

| `+k3s ctr+`
| Führen Sie den eingebetteten https://github.com/projectatomic/containerd/blob/master/docs/cli.md[Befehl] Dies ist ein CLI für containerd, den von K3s verwendeten Container-Daemon. Nützlich für die Fehlersuche.

| `+k3s token+`
| Verwalten Sie Bootstrap-Tokens. Weitere Informationen finden Sie in der xref:cli/token.adoc[Dokumentation zum Befehl`+k3s token+` ].

| `+k3s etcd-snapshot+`
| Führen Sie bei Bedarf Backups der Daten des K3s-Clusters durch und laden Sie sie auf S3 hoch. Weitere Informationen finden Sie in der xref:cli/etcd-snapshot.adoc[Dokumentation zum Befehl`+k3s etcd-snapshot+` ].

| `+k3s secrets-encrypt+`
| Konfigurieren Sie K3s so, dass Geheimnisse verschlüsselt werden, wenn sie im Cluster gespeichert werden. Weitere Informationen finden Sie in der xref:cli/secrets-encrypt.adoc[Dokumentation zum Befehl`+k3s secrets-encrypt+` ].

| `+k3s certificate+`
| Verwalten Sie K3s Zertifikate. Weitere Informationen finden Sie in der xref:cli/certificate.adoc[Dokumentation zum Befehl`+k3s certificate+` ].

| `+k3s completion+`
| Shell-Vervollständigungsskripte für k3s generieren

| `+k3s help+`
| Zeigt eine Liste von Befehlen oder die Hilfe zu einem Befehl
|===
