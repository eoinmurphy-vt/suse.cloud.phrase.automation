== Upgrade über die Befehlszeilenschnittstelle

Es gibt zwei Möglichkeiten, Knoten für ein Upgrade auszuwählen. Über ein Cluster-Ziel, das mit ALLEN Knoten in einem Cluster übereinstimmt, die unserem Selektor entsprechen, oder über einen Knotenselektor, der die Knoten auf der Grundlage der Knotenbezeichnungen abgleicht. Mit dem Knotenselektor können wir das Upgrade gezielter durchführen, während der Clusterselektor nur alle Knoten in einem passenden Cluster auswählt.

Die Aktualisierung eines bestehenden `+ManagedOSImage+` löst einen neuen Aktualisierungszyklus aus, um das konfigurierte Image (oder die Image-Version) mit allen Zielknoten abzugleichen.

=== Verwaltung der verfügbaren Versionen

Es ist möglich, xref:references/managedosversion-reference.adoc[ManagedOSVersions] direkt zu erstellen oder xref:references/managedosversionchannel-reference.adoc[ManagedOSVersionChannels] zu abonnieren, um `+ManagedOSVersions+` automatisch mit ihnen zu synchronisieren.

Für weitere Details und eine Liste der verfügbaren Kanäle oder um eigene Kanäle zu erstellen, lesen Sie bitte die xref:operator-operational-tasks/channels.adoc[Dokumentation].

== Upgrade über die Benutzeroberfläche

Um über die Benutzeroberfläche zu aktualisieren, müssen Sie im Menü {elemental-product-name} Erweitert und dann `+Update Groups+` anklicken.

Wählen Sie einen Namen und die anzuvisierenden Cluster aus und entscheiden Sie sich für eine der beiden Upgrade-Möglichkeiten:

image::upgrade-ui-menu.png[{elemental-product-name} Upgrade-Menü]

[tabs]
======
Über Managed OS Version::
+
====
In diesem Fall wird eine `+OS Version Channels+` verwendet, um die Ressourcen von `+OS Versions+` automatisch aufzufüllen. 

Der untenstehende Kanal wird von uns bereitgestellt, aber Sie können auch Ihren eigenen Kanal mitbringen. 

Im Abschnitt <<_managing_available_versions,Verwaltung der verfügbaren Versionen>> erfahren Sie, wie die `+ManagedOSVersion+` verwaltet werden. 

image:upgrade-ui-version-channel.png[OS Version Channel erstellen] 

Nach einer kurzen Synchronisierungszeit wird Ihr `+OS Versions+` im Menü `+OS Versions+` angezeigt. 

image:upgrade-ui-os-version.png[{elemental-product-name} Menü "OS-Version] 

Schließlich können Sie bei der Erstellung Ihrer `+Upgrade Group+` die `+OS Versions+` auswählen, wie im folgenden Screenshot dargestellt: 

image:upgrade-ui-upgrade-group-channel.png[OS-Version in der Upgrade-Gruppe auswählen]
====

Registerkarte Über Bild aus Registrierung::
+
====
Geben Sie einfach im Feld `+Image path+` ein OCI-Image an, auf das aktualisiert werden soll: image:upgrade-ui-image-registry.png[Upgrade über Image Registry]
====
======

Klicken Sie auf die Schaltfläche `+Create+`, um den Upgrade-Prozess zu starten. Wenn Sie mehrere Knoten haben, wird das Upgrade nacheinander Knoten für Knoten durchgeführt.
