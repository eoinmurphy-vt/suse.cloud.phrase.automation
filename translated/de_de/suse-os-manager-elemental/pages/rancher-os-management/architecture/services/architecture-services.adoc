= Dienstleistungen
:revdate: 2025-09-08
:page-revdate: {revdate}

{elemental-product-name} Die Funktionalität wird durch die folgende Liste von Diensten bereitgestellt:

* xref:rancher-os-management/architecture/services/architecture-machineonboarding.adoc[Machine Onboarding]: Ermöglicht die Bereitstellung von Betriebssystemen und die Registrierung von Hosts im {elemental-product-name} Operator-Katalog.
* xref:rancher-os-management/architecture/services/architecture-clusterdeployment.adoc[Kubernetes Cluster Deployment]: löst die Erstellung eines Kubernetes Clusters auf dem im {elemental-product-name} Operator-Katalog registrierten verfügbaren Host aus.
