= MachineInventory
:revdate: 2025-07-03
:page-revdate: {revdate}

Wenn sich ein neuer Host erfolgreich registriert, erstellt {elemental_operator_name} eine MachineInventory-Ressource, die diesen speziellen Host repräsentiert.
Das MachineInventory speichert den TPM-Hash des georteten Hosts, der während des Registrierungsprozesses abgerufen wird, und ermöglicht die Ausführung beliebiger Befehle (Pläne) auf dem Rechner.

Ein MachineInventory hat zwei Bedingungen:

* `+AdoptionReady+`was bedeutet, dass die Maschine von einem Selektor als Teil eines Clusters angenommen wurde.
* `+Ready+`Dies zeigt an, dass der Computer mit einem {elemental-product-name} Betriebssystem registriert und bereitgestellt wurde.
