== Was ist {product_name}?

Die Cluster-API-Erweiterung für Rancher, auch bekannt als {product_name}, ist ein Kubernetes-Operator, der die **Integration zwischen dem Rancher Manager und der Cluster-API (CAPI)** ermöglicht, mit dem Ziel, die vollständige CAPI-Unterstützung in Rancher zu integrieren.

Mit {product_name} können Sie:

* **CAPI-Cluster automatisch in Rancher importieren**, indem der Rancher Cluster Agent in CAPI-bereitgestellten Clustern installiert wird.
* Implementierung eines **GitOps-nativen** Workflows zur Verwaltung von Kubernetes-Clustern in Rancher.
* **Vereinfachte Verwaltung von CAPI-Providern** mit dem CAPI Operator.
