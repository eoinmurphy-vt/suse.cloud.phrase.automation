Die SUSE Rancher Prime Observability Erweiterung verwendet Kubernetes RBAC, um Rancher-Benutzern in SUSE Observability Zugriff zu gewähren.
Wenn Sie Rancher nicht verwenden, sehen Sie sich xref:/setup/security/rbac/rbac_roles.adoc[So richten Sie Rollen ein] in einer eigenständigen Installation an.
