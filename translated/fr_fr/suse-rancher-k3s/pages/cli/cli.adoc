= Outils CLI
:page-languages: [en, ja, ko, zh]
:revdate: 2024-09-11
:page-revdate: {revdate}

Le binaire K3s contient un certain nombre d'outils supplémentaires pour vous aider à gérer votre cluster.

|===
| Commandement | Description

| `+k3s server+`
| Exécutez un nœud de serveur K3s, qui lance les composants Kubernetes `+apiserver+`, `+scheduler+`, `+controller-manager+`, et `+cloud-controller-manager+`, ainsi qu'un magasin de données et les composants de l'agent. Voir la xref:cli/server.adoc[documentation de la commande`+k3s server+` ] pour plus d'informations.

| `+k3s agent+`
| Exécutez le nœud d'agent K3s, qui lance `+containerd+`, `+flannel+`, `+kube-router+` network policy controller, et les composants Kubernetes `+kubelet+` et `+kube-proxy+`. Voir la xref:cli/agent.adoc[documentation de la commande`+k3s agent+` ] pour plus d'informations.

| `+k3s kubectl+`
| Exécutez la intégrée https://kubernetes.io/docs/reference/kubectl[commande] Il s'agit d'un CLI permettant d'interagir avec l'apiserver de Kubernetes.  Si la variable d'environnement `+KUBECONFIG+` n'est pas définie, cela tentera automatiquement d'utiliser le kubeconfig à l'adresse `+/etc/rancher/k3s/k3s.yaml+`.

| `+k3s crictl+`
| Exécutez la intégrée https://github.com/kubernetes-sigs/cri-tools/blob/master/docs/crictl.md[commande] Il s'agit d'un CLI permettant d'interagir avec l'interface d'exécution des conteneurs (CRI) de Kubernetes. Utile pour le débogage.

| `+k3s ctr+`
| Exécutez la intégrée https://github.com/projectatomic/containerd/blob/master/docs/cli.md[commande] Il s'agit d'un CLI pour containerd, le démon de conteneur utilisé par K3s. Utile pour le débogage.

| `+k3s token+`
| Gérer les jetons d'amorçage. Voir la xref:cli/token.adoc[documentation de la commande`+k3s token+` ] pour plus d'informations.

| `+k3s etcd-snapshot+`
| Effectuer des sauvegardes à la demande des données du cluster K3s et les télécharger sur S3. Voir la xref:cli/etcd-snapshot.adoc[documentation de la commande`+k3s etcd-snapshot+` ] pour plus d'informations.

| `+k3s secrets-encrypt+`
| Configurer les K3 pour qu'ils chiffrent les secrets lorsqu'ils sont stockés dans le cluster. Voir la xref:cli/secrets-encrypt.adoc[documentation de la commande`+k3s secrets-encrypt+` ] pour plus d'informations.

| `+k3s certificate+`
| Gérer les certificats K3s. Voir la xref:cli/certificate.adoc[documentation de la commande`+k3s certificate+` ] pour plus d'informations.

| `+k3s completion+`
| Générer des scripts de complétion shell pour k3s

| `+k3s help+`
| Affiche une liste de commandes ou l'aide pour une commande.
|===
