= Mise en réseau
:page-languages: [en, ja, ko, zh]
:revdate: 2024-09-11
:page-revdate: {revdate}

Cette section contient des instructions pour configurer la mise en réseau dans les K3.

Les xref:networking/basic-network-options.adoc[options réseau de base] couvrent la configuration réseau de base de la grappe, telle que les configurations flanelle et de pile simple/double.

xref:networking/distributed-multicloud.adoc[Cluster hybride/multicloud] fournit des indications sur les options disponibles pour étendre le cluster k3s sur des nœuds distants ou hybrides.

xref:networking/multus-ipams.adoc[Multus et les plugins IPAM] fournissent des conseils pour exploiter Multus dans les K3 afin d'avoir plusieurs interfaces par pod.

xref:networking/networking-services.adoc[Services de réseau : dns, ingress, etc] explique comment CoreDNS, Traefik, le contrôleur de politique de réseau et le contrôleur ServiceLB fonctionnent au sein de k3s
