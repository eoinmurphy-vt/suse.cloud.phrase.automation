== Conditions préalables

La version actuelle de l'Assistant IA nécessite les composants suivants:

* https://helm.sh/docs/[Helm installé]
* Rancher Prime/Community Manager 2.13 ou supérieur

Assurez-vous d'avoir les droits pour déployer CRD (cluster_admin) sur le cluster hôte.

== Installez {project-name}

L'installation de {short-project-name} est un processus en deux étapes:

* Déployez l'agent et le MCP via le chart Helm fourni.
* Déployez l'extension Rancher UI.

=== Installez l'Agent et le MCP sur le cluster local

. Créez un fichier de configuration `+values.yaml+` (pour Ollama) :
+
[source,yaml]
----
ollamaLlmModel: "gpt-oss:20b"
ollamaUrl: "http://ollama:11434"
activeLlm: "ollama"
----
+
[NOTE]
====

Vous pouvez modifier ces paramètres et fournisseurs plus tard dans les onglets « Paramètres globaux -> Assistant IA » de Rancher.

====
. Installez le chart rancher-ai-agent
+
[source,bash,subs="attributes+"]
----
helm install rancher-ai-agent \
  --namespace cattle-ai-agent-system \
  --create-namespace \
  -f values.yaml {oci-reg}
----

=== Installez l'extension Rancher UI

Ouvrez l'interface utilisateur de Rancher Manager et accédez à la page 'Extensions'.

. Dans Rancher Manager, cliquez sur Extension dans la barre de menu.
+
image::menubar-extension.png[Bouton d'extension dans la barre de menu]
+
. Utilisez le menu à trois points en haut à droite et sélectionnez 'Ajouter des dépôts Rancher'.
+
image::rancher-repositories.png[Menu pour gérer les dépôts Rancher]
+
. Sélectionnez 'Ajouter le dépôt officiel des extensions Rancher' pour ajouter le dépôt.
+
image::add-extensions-repositories.png[Ajouter le dépôt officiel Rancher]
+
. Cliquez sur 'Ajouter'.
+
. Trouvez la carte de l'Assistant IA et cliquez sur 'Installer'.
. Sélectionnez une version (ou utilisez la dernière par défaut) et cliquez sur 'Installer'.
+
image::config-repo-install.png[Sélectionnez une version ou utilisez la dernière.]
+
. Une fois l'extension installée, cliquez sur le bouton 'Recharger' qui apparaît en haut de la page.
