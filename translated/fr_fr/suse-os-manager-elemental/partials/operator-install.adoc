== Installer {elemental-product-name} Opérateur
:revdate: 2025-09-08
:page-revdate: {revdate}

`+elemental-operator+` est le point final de gestion, qui exécute le cluster de gestion et s'occupe de la création d'inventaires, de l'enregistrement des machines et de bien d'autres choses encore.

Nous allons utiliser le gestionnaire de paquets Helm pour installer la carte elemental-operator dans notre cluster.

[,console]
----
helm upgrade --create-namespace -n cattle-elemental-system --install elemental-operator-crds oci://registry.suse.com/rancher/elemental-operator-crds-chart

helm upgrade --create-namespace -n cattle-elemental-system --install elemental-operator oci://registry.suse.com/rancher/elemental-operator-chart
----

Après quelques secondes, vous devriez voir apparaître le pod de l'opérateur sur l'espace de noms `+cattle-elemental-system+`:

[,console]
----
kubectl get pods -n cattle-elemental-system

NAME                                  READY   STATUS    RESTARTS   AGE
elemental-operator-64f88fc695-b8qhn   1/1     Running   0          16s
----

[IMPORTANT]
.Helm v3.8.0+ requis
----
La carte de l'opérateur {elemental-product-name} est distribuée par l'intermédiaire d'un registre OCI : Helm supporte correctement les registres basés sur l'OCI à partir de la version 3.8.0.
----


[CAUTION]
.Changement de l'ordre d'installation des graphiques lors de la mise à jour de la version < 1.2.4 d'elemental-operator
----
Lors de la mise à jour d'une version d'elemental-operator intégrant les CRDs de {elemental-product-name} (version < 1.2.4), l'installation de la carte elemental-operator-crds échouera.
Vous devrez d'abord mettre à jour la carte elemental-operator, puis installer la carte elemental-operator-crds.
----

=== Installations non stables

Outre les cartes Helm mentionnées ci-dessus, deux autres versions de `+non-stable+`sont disponibles.

* *Staging :* se réfère à la dernière version étiquetée de Github. Ceci est documenté dans les pages xref:quickstarts/quickstart-ui.adoc[suivantes.] 
* *Développement :* se réfère à la "pointe de HEAD" de Github. Il s'agit de la version en cours de développement, qui évolue constamment.

[tabs]
=====
Version d'essai (x86-64, ARM64 (Raspberry Pi 4))::
+
--
[,console]
----
helm upgrade --create-namespace -n cattle-elemental-system --install elemental-operator-crds oci://registry.opensuse.org/isv/rancher/elemental/staging/charts/rancher/elemental-operator-crds-chart 
helm upgrade --create-namespace -n cattle-elemental-system --install elemental-operator oci://registry.opensuse.org/isv/rancher/elemental/staging/charts/rancher/elemental-operator-chart
----
--

Version de développement (x86-64, ARM64 (Raspberry Pi 4))::
+
--
[WARNING]
.Rappel
====
La version de développement n'est pas recommandée pour les environnements de production. Nous accueillons volontiers les commentaires via Slack ou Github issues, mais il peut être instable et contenir des fonctionnalités expérimentales qui peuvent être supprimées sans préavis.
====

[,console]
----
helm upgrade --create-namespace -n cattle-elemental-system --install elemental-operator-crds oci://registry.opensuse.org/isv/rancher/elemental/dev/charts/rancher/elemental-operator-crds-chart
helm upgrade --create-namespace -n cattle-elemental-system --install --set image.imagePullPolicy=Always elemental-operator oci://registry.opensuse.org/isv/rancher/elemental/dev/charts/rancher/elemental-operator-chart
----
--
=====
