{elemental-product-name} La fonctionnalité est assurée par la liste de services suivante :

* xref:rancher-os-management/architecture/services/architecture-machineonboarding.adoc[Machine Onboarding]: permet aux hôtes d'être approvisionnés en système d'exploitation et enregistrés dans le catalogue de l'opérateur {elemental-product-name}.
* xref:rancher-os-management/architecture/services/architecture-clusterdeployment.adoc[Déploiement d'un cluster Kubernetes]: déclenche la création d'un cluster Kubernetes sur l'hôte disponible enregistré dans le catalogue de l'opérateur {elemental-product-name}.
