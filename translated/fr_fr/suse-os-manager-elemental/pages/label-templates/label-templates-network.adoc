= Variables du modèle de réseau
:revdate: 2025-08-15
:page-revdate: {revdate}

Les informations collectées dans les variables du modèle de réseau définissent les attributs des interfaces réseau disponibles dans le système.

Les cartes d'interface réseau sont suivies à la fois par leur numéro (0, 1, 2, ...) et par le nom de l'interface_(eth0_, _virbr0_, ...).

Les adresses IPv4Addresses et IPv6Addresses sont suivies par un numéro (0, 1, ...) et les premières entrées d'adresses IPv4 et IPv6 (c'est-à-dire IPv4Addresses.0 et IPv6Addresses.0) sont suivies dans les variables IPv4Addresses et IPv6Addresses.

Par exemple, si `+${Network/NICs/2/Name}+` est égal à `+eth0+`, l'adresse Mac de l'interface réseau peut être récupérée via les variables `+${Network/NICs/2/MacAddress}+` et `+${Network/NICs/eth0/MacAddress}+`.

`+eth0+` (première) L'adresse IPv4 peut être récupérée à l'aide des variables `+${Network/NICs/eth0/IPv4Address}+` et `+${Network/NICs/eth0/IPv4Addresses/0}+`. L'adresse IPv4 secondaire de `+eth0+` peut être récupérée avec `+${Network/NICs/eth0/IPv4Addresses/1}+`.
