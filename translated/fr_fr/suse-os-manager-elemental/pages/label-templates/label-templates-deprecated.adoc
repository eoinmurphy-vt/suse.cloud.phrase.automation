= Variables dépréciées des modèles d'étiquettes
:revdate: 2025-09-08
:page-revdate: {revdate}

Les familles de variables *SMBIOS* et *Hardware* ont été supprimées dans l'Opérateur élémentaire v1.7.0.

Bien que ces familles de variables soient encore correctement rendues dans les versions actuelles d'Elemental Operator, leur prise en charge sera supprimée dans les prochaines versions d'Elemental Operator.

Si vous utilisez encore les familles de variables de ces anciens modèles d'étiquettes, veuillez passer auxxref:label-templates/label-templates.adoc#_label_templates_variables[nouvelles] famillesxref:label-templates/label-templates.adoc#_label_templates_variables[disponibles] si possible.

Pour simplifier l'adoption des nouvelles variables, consultez les tableaux de correspondance entre les variables obsolètes et les nouvelles variables prises en charge :

* <<_hardware_labels_to_new_label_templates_variables_table,Etiquettes matérielles vers les variables des nouveaux modèles d'étiquettes>>
* <<_smbios_labels_to_new_label_templates_variable_table,Etiquettes SMBIOS vers les variables des nouveaux modèles d'étiquettes>>
