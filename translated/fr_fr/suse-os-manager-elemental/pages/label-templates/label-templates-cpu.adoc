= Variables du modèle de CPU
:revdate: 2025-08-15
:page-revdate: {revdate}

Les informations collectées dans les variables du modèle CPU définissent les attributs des processeurs installés sur le système.

Les données sont exposées par numéro de processeur (0, 1, 2, ...).

Pour faire référence au premier processeur, le numéro du processeur (_`+<Number>+`_) peut être omis (par exemple, la variable `+${CPU/Processor/Model}+` est identique à `+${CPU/Processor/0/Model}+`).

|===
| Variable | Description | de

| `+${CPU/Processor/Capabilities}+`
| Liste séparée par des virgules des capacités du processeur
| 1.7.0

| `+${CPU/Processor/ID}+`
| ID du processeur
| 1.7.0

| `+${CPU/Processor/Model}+`
| Modèle de processeur
| 1.7.0

| `+${CPU/Processor/NumCores}+`
| Nombre de cœurs
| 1.7.0

| `+${CPU/Processor/NumThreads}+`
| Nombre de fils
| 1.7.0

| `+${CPU/Processor/Vendor}+`
| Fournisseur de processeur
| 1.7.0

| `+${CPU/Processor/_<Number>_/Capabilities}+`
| Liste de _numéros de_ processeurs séparés par des virgules
| 1.7.0

| `+${CPU/Processor/_<Number>_/ID}+`
| _Numéro du_ processeur ID
| 1.7.0

| `+${CPU/Processor/_<Number>_/Model}+`
| _Numéro du_ processeur modèle
| 1.7.0

| `+${CPU/Processor/_<Number>_/NumCores}+`
| _Nombre_ de cœurs du processeur
| 1.7.0

| `+${CPU/Processor/_<Number>_/NumThreads}+`
| Nombre de threads du _numéro de_ processeur
| 1.7.0

| `+${CPU/Processor/_<Number>_/Vendor}+`
| _Numéro du_ processeur fournisseur
| 1.7.0

| `+${CPU/TotalCores}+`
| Nombre total de cœurs dans le système
| 1.7.0

| `+${CPU/TotalProcessors}+`
| Nombre de processeurs dans le système
| 1.7.0

| `+${CPU/TotalThreads}+`
| Nombre total de threads dans le système
| 1.7.0
|===
