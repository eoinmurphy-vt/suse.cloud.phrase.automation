= Variables du modèle GPU
:revdate: 2025-08-15
:page-revdate: {revdate}

Les informations collectées dans les variables du modèle GPU définissent les attributs des cartes graphiques installées dans le système.

Les données sont exposées par numéro de carte graphique (0, 1, 2, ...).

Pour faire référence à la première carte graphique, le numéro de la carte graphique (_`+<Number>+`_) peut être omis (par exemple, la variable `+${GPU/GraphicsCards/Driver}+` est identique à `+${GPU/GraphicsCards/0/Driver}+`).

|===
| Variable | Description | de

| `+${GPU/GraphicsCards/Driver}+`
| Pilote de la carte GPU
| 1.7.0

| `+${GPU/GraphicsCards/ProductName}+`
| Modèle de carte GPU
| 1.7.0

| `+${GPU/GraphicsCards/VendorName}+`
| Fournisseur de cartes GPU
| 1.7.0

| `+${GPU/GraphicsCards/_<Number>_/Driver}+`
| Pilote du _numéro de la_ carte GPU
| 1.7.0

| `+${GPU/GraphicsCards/_<Number>_/ProductName}+`
| _Numéro de la_ carte GPU Modèle
| 1.7.0

| `+${GPU/GraphicsCards/_<Number>_/VendorName}+`
| _Numéro de la_ carte GPU Fournisseur
| 1.7.0

| `+${GPU/TotalCards}+`
| Nombre de cartes GPU dans le système
| 1.7.0
|===
