= Variables du modèle de châssis
:revdate: 2025-08-15
:page-revdate: {revdate}

Les informations contenues dans les variables Chassis définissent les attributs de la ou des enceintes mécaniques du système.

Sous-ensemble de SMBIOS System Enclosure or Chassis (type 3).

Pour plus d'informations sur SMBIOS, voir lahttps://www.dmtf.org/sites/default/files/standards/documents/DSP0134_3.7.1.pdf[spécification SMSBIOS sur le site].

|===
| Variable | Description | de

| `+${Chassis/AssetTag}+`
| Étiquette d'actif du châssis
| 1.7.0

| `+${Chassis/SerialNumber}+`
| Numéro de série du châssis
| 1.7.0

| `+${Chassis/TypeDescription}+`
| Type de châssis (description)
| 1.7.0

| `+${Chassis/Type}+`
| Type de châssis (numéro/code)
| 1.7.0

| `+${Chassis/Vendor}+`
| Fournisseur de châssis
| 1.7.0

| `+${Chassis/Version}+`
| Version du châssis
| 1.7.0
|===
