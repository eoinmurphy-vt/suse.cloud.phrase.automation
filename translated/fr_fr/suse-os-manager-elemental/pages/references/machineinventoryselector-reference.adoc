= Référence du sélecteur d'inventaire de machines
:revdate: 2024-09-24
:page-revdate: {revdate}

Un MachineInventorySelector sélectionne les inventaires de machines sur la base des sélecteurs appliqués (généralement la recherche de motifs sur les valeurs des étiquettes des inventaires de machines).

Les sélecteurs MachineInventorySelectors sont soumis à deux conditions :

* `+InventoryReady+`devient vrai si le MachineInventorySelector a trouvé un MachineInventory correspondant et s'est défini avec succès comme propriétaire du MachineInventory.
* `+Ready+`permet de savoir si le sélecteur a déjà adopté une machine et démarré le processus de provisionnement kubernetes (node bootstrap).
