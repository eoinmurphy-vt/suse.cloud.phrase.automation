== {product_name} とは？

Cluster API Extension for Rancher（別名：{product_name} ）は、**Rancherに**完全なCAPIサポートをもたらすことを目的として、**Rancher ManagerとCluster API（CAPI）の統合を**提供するKubernetes Operatorである。

{product_name} 、あなたはできる：

* CAPIでプロビジョニングされたクラスタにRancher Cluster Agentをインストールすることで、**CAPIクラスタを自動的にRancherにインポート**します。
* RancherでKubernetesクラスタを管理する**GitOpsネイティブな**ワークフローを実装する。
* CAPI Operatorを使用した**CAPIプロバイダの管理の簡素化**。
