= README
:description: SUSE Observability

Open telemetry は、あなたのアプリケーションを（自動）計測するために、 https://opentelemetry.io/docs/languages/[多くの言語用の]提供します。Instrumentationを使用すると、アプリケーションからSUSE Observabilityにトレース、メトリクス、ログを送信できます。自動計測は、アプリケーションにSDKを含める以外、手動で操作することなく、一般的なライブラリやフレームワークのトレース、メトリクス、ログをキャプチャします。

Open Telemetry には、ここに記載されている以外にも多くの SDK があります。ご使用の言語がここに記載されていない場合は、https://opentelemetry.io/docs/languages/[opentelemetry.ioの]ドキュメントとSUSE Observabilityに必要なxref:/setup/otel/languages/sdk-exporter-config.adoc[SDKエクスポーター設定を組み合わせて]使用できます。
