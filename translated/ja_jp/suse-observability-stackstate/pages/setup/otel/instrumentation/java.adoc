= Javaアプリケーション
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: SUSE Observability

Java SDKは、JVM上のアプリケーションのインスツルメンテーションをサポートしている。その結果、Javaだけでなく、KotlinやScalaのような他のJVM言語もサポートしている。

== 自動計装

Javaの自動インスツルメンテーションでは、JavaエージェントJARが使用され、Java 8+のアプリケーションに添付することができる。いくつかのKotlinやScalaライブラリを含む、多くのhttps://github.com/open-telemetry/opentelemetry-java-instrumentation/blob/main/docs/supported-libraries.md[、人気のあるライブラリやフレームワークから]テレメトリをキャプチャするために、バイトコードを動的に注入する。インバウンドリクエスト、アウトバウンドHTTPコール、データベースコールなど、アプリやサービスの&quot;`+edges+`&quot; 、テレメトリーデータをキャプチャするために使用できる。

自動計測は、アプリケーションを変更する必要はない。設定方法は以下の通り：

. リポジトリのhttps://github.com/open-telemetry/opentelemetry-java-instrumentation/releases[opentelemetry-java-instrumentation]からhttps://github.com/open-telemetry/opentelemetry-java-instrumentation/releases/latest/download/opentelemetry-javaagent.jar[Releases]ダウンロードし、アプリケーションの docker イメージに JAR ファイルを含めます。JARファイルには、エージェント・ライブラリとインスツルメンテーション・ライブラリが含まれている。
. docker イメージのエントリポイントまたはコマンドを更新するか、アプリケーションの Kubernetes マニフェスト内の`+command+` を更新して、Java エージェントをロードするようにアプリケーションを起動するコマンドを更新します。`+-javaagent:/path/to/opentelemetry-javaagent.jar+` を追加する：
+
[,bash]
----
java -javaagent:/path/to/opentelemetry-javaagent.jar -jar myapp.jar
----

. 環境変数を追加してアプリケーションをデプロイxref:/setup/otel/instrumentation/sdk-exporter-config.adoc[し、サービス名とエクスポータ・エンドポイントを設定]します。
. サービス名のメトリクス/トレースエクスプローラでメトリクス/トレースを検索して、SUSE Observabilityがトレースやメトリクスを受信していることを確認します。

詳細については、https://opentelemetry.io/docs/languages/java/automatic/[Open Telemetry のドキュメントを]参照してください。

== 手動計装

手動インストルメンテーションは、自動インストルメンテーションではサポートされていないコード部分からのメトリクス、トレース、またはログが必要な場合に使用できます。例えば、サポートされていないライブラリ、社内コード、ビジネスレベルのメトリクスなどだ。

そのデータを取得するには、アプリケーションを修正する必要がある。

. 依存関係として Open Telemetry SDK を含める
. アプリケーションにコードを追加して、必要に応じてメトリクス、スパン、ログをキャプチャします。

これについては、https://opentelemetry.io/docs/languages/java/instrumentation/[Open Telemetry Java SDK doc pagesに]詳細なドキュメントがあります。

OTLPエクスポータ（これがデフォルト）とhttps://opentelemetry.io/docs/languages/java/instrumentation/#_autoconfiguration[自動コンフィギュレーションを]使用していることを確認してください。アプリケーションをデプロイする際、サービス名とエクスポーターはxref:/setup/otel/instrumentation/sdk-exporter-config.adoc[環境変数によって設定される]。
