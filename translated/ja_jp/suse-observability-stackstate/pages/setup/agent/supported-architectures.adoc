= SUSE Observability Agentがサポートするアーキテクチャ
:revdate: 2025-07-10
:page-revdate: {revdate}

SUSE Observabilityは、特定のハードウェアアーキテクチャ上で動作するように設計され、テストされています。インストールと操作はこれらのシステムでサポートされている：

* `+x86_64+` (AMD64またはインテル64とも呼ばれる）
* `+arm64+` (AArch64としても知られる）

配備を進める前に、ターゲット環境がこれらの要件を満たしていることを確認してください。その他のアーキテクチャへのSUSE Observabilityのインストールや実行はサポートされていません。
