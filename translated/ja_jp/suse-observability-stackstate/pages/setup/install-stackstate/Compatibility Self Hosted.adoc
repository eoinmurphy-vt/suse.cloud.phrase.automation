= RancherおよびKubernetesとの互換性

{stackstate-product-name} 2.6.3は、RKE2/Kubernetesを実行しているRancher管理Kubernetesクラスタでサポートされています。

以下の表は、互換性のあるRancherのバージョンと、対応するRKE2/Kubernetesのバージョンの一覧です。

|===
| 牧場主バージョン | RKE2 / Kubernetes Versions

| 2.11.x
| v1.30.11+rke2r1

| 2.12.x
| v1.30.11+rke2r1

| 2.13.x
a|
v1.30.11+rke2r1

v1.31.13+rke2r1

v1.32.10+rke2r1

|===

{stackstate-product-name} SUSEが提供するHelmチャートを使用して、KubernetesまたはOpenShiftクラスタにインストールできます。インストールにはHelm v3.xが必要で、チャートは以下のプラットフォームでサポートされています：

* *Kubernetes：*1.25から1.33
* *OpenShift:*4.14から4.19
