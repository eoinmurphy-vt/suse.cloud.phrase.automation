== デフォルトのユーザー名とパスワード

[tabs]
====
Kubernetes::
+
--
SUSE Observabilityは、デフォルトで以下の管理者アカウントで設定されています：

* *ユーザー名* `+admin+`
* *のパスワードが必要だ：*インストール時に設定する。これは、`+generate_values.sh+` スクリプトによって収集され、MD5 ハッシュ形式で `+values.yaml+`
--

OpenShift::
+
--
SUSE Observabilityは、デフォルトで以下の管理者アカウントで設定されています：

* *ユーザー名* `+admin+`
* *のパスワードが必要だ：*インストール時に設定する。これは、`+generate_values.sh+` スクリプトによって収集され、MD5 ハッシュ形式で `+values.yaml+`
--
