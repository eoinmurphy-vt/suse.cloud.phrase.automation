== 概要

xref:use/security/k8s-service-tokens.adoc[サービストークンを]使用して、関連するユーザーアカウントを持たずにSUSE Observabilityを認証します。CI/CD から、あるいは、エージェントや Open Telemetry から遠隔測定データを送信するときに、サービス・トークンを認証するのが望ましい方法です。サービス・トークンの作成方法と使用方法については、xref:use/security/k8s-service-tokens.adoc[Service Tokensのドキュメントに]記載されています。
