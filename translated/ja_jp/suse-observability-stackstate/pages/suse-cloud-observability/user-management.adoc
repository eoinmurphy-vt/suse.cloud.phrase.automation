= ユーザー管理
:description: SUSE Cloud Observabilityのユーザー管理

SUSE Cloud Observabilityテナント(SUSE Cloud Observabilityインスタンス)のユーザーは、https://www.keycloak.org/[Keycloakで]管理されます。各顧客（テナント）には専用のKeycloakレルムがあります。ユーザー作成時のウェルカムメッセージにKeycloakコンソールへのリンクが送信されます。

SUSE Cloud Observabilityは、認証のためにユーザーをKeycloakにリダイレクトします。ユーザーは1つ以上のKeycloakグループのメンバーであることが期待されます。

あらかじめ定義されたキークロークのグループ：

* *realm-admin*:このグループのメンバーは、Keycloakレルムコンソールにログインし、ユーザー管理操作を行うことができます。
* *stackstate-k8s-troubleshooter*：このグループのユーザーには、`+stackstate-k8s-troubleshooter+` Keycloakクライアントロールが割り当てられます。このロールは、同名のSUSE Cloud Observabilityロールにマッピングされます。このロールは、通常のSUSE Cloud Observability権限を付与します。
* *stackstate-k8s-admin*：このグループのユーザーには、`+stackstate-k8s-admin+` Keycloakクライアントロールが割り当てられます。このロールは、同名のSUSE Cloud Observabilityロールにマッピングされます。このロールは、SUSE Cloud Observabilityの特権権限を付与します。

== ユーザー管理URL

新規ユーザーが作成されると、Keycloakユーザー管理ページへのリンクを含むウェルカムメッセージを受け取ります。このリンクは*テナント管理者専用で*、デフォルトではテナント管理者が最初のユーザーとなります。URLの形式は次の通り：`+https://<keycloak_FQDN>/realms/<your_dedicated_Keycloak_realm>/account+`.

== ユーザー管理

* キークローク管理コンソールにログインします。

image::keycloak_admin_console.png[キークローク管理コンソール]

=== ユーザー管理

* 左側のメニューで、`+Manage+` セクションの下にある`+Users+` を選択する。

==== 新規ユーザーの追加

`+Add user+` 」ボタンをクリックし、「ユーザー名」、「Eメール」、「名」、「姓」などのユーザー情報を入力する。

. `+Required users actions+` フィールドは空のままにしてください。
. ユーザーを必要なグループに追加する。
. `+Save+` をクリックする。
+
サインアップリンクとSUSE Cloud Observabilityテナント、Keycloak管理コンソール、およびアカウントコンソールへのリンクが記載されたウェルカムメッセージがユーザーに電子メールで送信されます。

NOTE: アカウントの有効化（Eメールによる確認とパスワードの再設定を含む）には、サインアップリンクに従う必要があります。

image::keycloak_create_user.png[キークローク ユーザー作成]

==== ユーザー情報の更新

*ユーザーの詳細を編集*するには、ユーザー名をクリックしてユーザーを選択します。

. 必要に応じて詳細を変更する。
. 例えば、ユーザーにパスワードの更新を強制したり、ワンタイムパスワードを設定したりするために、1つ以上の`+Required user actions+` を設定する。
. 完了したら`+Save+` ボタンを押す。

image::keycloak_update_user.png[キークローク・アップデート・ユーザー]

==== ユーザーの削除

. *1人または複数のユーザーを削除*するには、必要なユーザーを選択し、`+Delete user+` ボタンを押します。

=== グループメンバー

. キークローク管理コンソールにログインします。
. `+Groups+` 、管理したいグループを検索する。
. グループ名をクリックしてグループの詳細を開き、`+Members+` タブを開きます。
. 新しいグループメンバーを追加するには、`+Add Member+` ボタンを押し、必要なユーザーを選択します。
. グループからユーザーを削除するには、リストからユーザーを選択し、"↪Sm_22EE" と表示された`+Add member+` ボタンと同じ行のメニューから`+Leave group+` を選択します。
