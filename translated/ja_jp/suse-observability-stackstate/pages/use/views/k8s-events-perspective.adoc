= イベントの視点
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: SUSE Observability

== 概要

イベント］パースペクティブには、現在のxref:/use/views/k8s-view-structure.adoc[ビュー]またはフィルタリングされたトポロジー内の要素のイベントと変更が表示されます。

image::k8s/k8s-events-perspective.png[イベントの視点]

== 表示されたイベントをフィルタリングする

表示されるイベントは、ビューUIの右上にある*フィルター*メニューでフィルターすることができます。イベントフィルターは、イベントパースペクティブ、ハイライトパースペクティブ、タイムラインなどのビューに表示されるすべてのイベントに適用されます。

=== プロパティで絞り込む

*フィルター・*メニューに設定されたxref:/use/views/k8s-filters.adoc#_filter_events[イベント・フィルターを]使用して、イベント・カテゴリー、タイプ、ソース・システム、タグに基づいて表示されるイベントを絞り込むことができます。

=== ソースコンポーネントによるフィルタリング

*filters*メニューで設定するxref:/use/views/k8s-filters.adoc#_filter_topology[トポロジーフィルターは]、イベントが表示されるエレメント（コンポーネントとリレーション）を定義する。適用されたトポロジーフィルターまたはビュー自体に一致する要素に関連するイベントのみが表示されます。トポロジーフィルタを更新することで、イベントを表示するコンポーネントを調整できる。

=== タイムスタンプによるフィルタリング

イベント パースペクティブとイベント リストには、ビュー UI の下部にあるタイムラインで選択されたxref:/use/stackstate-ui/k8sTs-timeline-time-travel.adoc#_telemetry_interval[Telemetry インターバルに]一致するイベントが表示されます。遠隔測定間隔を調整し、その時間に発生したイベントのみを表示する。

== タイムトラベル

イベント］パースペクティブには、生成されたイベントが一覧表示されます：

* xref:/use/stackstate-ui/k8sTs-timeline-time-travel.adoc#_topology_time[トポロジー時間によって]指定された時点に存在したトポロジー要素によって。
* 選択したxref:/use/stackstate-ui/k8sTs-timeline-time-travel.adoc#_telemetry_interval[テレメトリ間隔]内

これにより、履歴の特定の時点で利用可能なトポロジーエレメントによって生成されたイベントと、特定のタイムウィンドウ内で生成されたイベントの両方にタイムトラベルできる。新しいトポロジーの時間を選択すると、その時点で利用可能であったトポロジーにタイムトラベルする。イベントリストは、その時点で利用可能で、選択された遠隔測定間隔内のトポロジー要素のイベントを含むように更新される。

例えば、こうだ：

* *テレメトリ間隔を*調整して、表示されるイベント数を増減する。
* *トポロジーの時間を*調整し、その時点で利用可能なトポロジーにタイムトラベルする。トポロジー時間に存在したトポロジーエレメントによって、選択したテレメトリーインターバル内に生成されたイベントが表示される。
* イベントの*プロパティから*タイムスタンプをクリックすると、このトポロジー時間にジャンプします。これにより、イベント・リストが更新され、生成されたイベントが表示される：
 ** その時点で存在していたトポロジー要素によって。
 ** 指定されたテレメトリー間隔内（必要であれば、選択されたタイムスタンプに合わせて調整される）。
