= トレース・パースペクティブ
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: SUSE Observability

Traces Perspective（トレース・パースペクティブ）には、コンポーネントのスパンとそれに関連するメトリッ クが表示されます。
これにより、ITインフラストラクチャのアプリケーションのパフォーマンスをSUSE Observabilityで直接監視できます。

リスト内の任意のスパンをクリックすると、そのスパンに属する子孫スパンが表示されます。
拡大すると、スパンのタイムラインは、各子孫がいつ開始し、いつ完了したかを表示します。
同じトレース内の親スパンは、デフォルトでは折りたたまれていますが、必要に応じて展開することができます。 +
スパンは、OpenTelemetry ServiceName によって色分けされています。
トレースを検査し、そのスパンのリストを見るとき、任意のスパンをクリックして、さらに詳細を見ることができる。

image::k8s/k8s-traces-perspective.png[トレース・パースペクティブ]

== フィルターの痕跡

トレースフィルターでは、スパンステータス（Error、Ok、Unset）、親タイプ（External、Internal、Root）、期間に基づいて表示されるトレースを絞り込むことができます。

これらのフィルタに加えて、トレースはSUSE Observability UIの下部にあるタイムラインコントロールで選択した*タイムウィンドウに*一致します。
時間ウィンドウを調整し、その時間からのトレースのみを表示する。

=== 持続時間のフィルタリング

2番目のスクリーンショットに示すように、ヒストグラムをブラッシングすることで、スパンDurationの継続時間間隔を選択することができる。
ズームアウトするには、すべてのフィルターをクリアするか、全範囲にブラシをかける。

== スパンの詳細

SUSE Observabilityでは、xref:/use/views/k8s-view-structure.adoc[ビューに]、特定のKubernetesリソースのITインフラストラクチャのサブセレクションが表示されます。
トレース・パースペクティブは、リソースに関連するスパンとその子孫を表示します。
子孫スパンは、他のリソースから発生することができるため、スパンの詳細からそのリソースに移動することが可能である。
parent spans "リンクをクリックすると、トレース全体が表示されます。

image::k8s/k8s-traces-perspective-span-details.png[子孫のスパンの詳細]

上の2つの画像は、カートから注文を作成することを主な役割とするチェックアウト・サービスを示すことで、これらのコンセプトを説明しています。
注文リクエストのトレースとそのスパンの例を見ることができる。
子孫スパンがハイライトされ、その詳細が表示される。

同様に、トレースをブラッシングすると、トレースの特定の時間セクションがズームインする。
選択は後でクリックすることでリセットできる。

== タイムトラベル

トレース・パースペクティブを使用する場合、他のパースペクティブと同様に、ライブモードにするか、xref:/use/stackstate-ui/k8sTs-timeline-time-travel.adoc#_time_travel[過去にタイムトラベル]するかのどちらかを選ぶことができる。
