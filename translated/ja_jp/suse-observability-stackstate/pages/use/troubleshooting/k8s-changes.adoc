== 前回のデプロイメントとの差分を見る

Kubernetesの設定に変更を加えた後に問題が発生した場合は、最後のデプロイメント変更を見てみよう。

image::k8s/k8s-show-diff.png[]

現在のコンフィギュレーションと以前のコンフィギュレーションを比較することで、問題の原因となった可能性のある特定の変更を特定することができます。

image::k8s/k8s-show-diff-opened.png[]
