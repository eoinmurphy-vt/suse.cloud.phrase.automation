= 測定基準を探る
:revdate: 2025-07-10

:description: SUSE Observability

SUSE Observabilityのメインメニューの一番下に、メトリクス・エクスプローラがあります。PromQLクエリを実行し、結果の時系列を視覚化するために使用します。クエリー結果は、選択された時間範囲のチャートと、時系列のラベルとともに最後の値を示す表に表示される。

image::k8s/k8s-metrics-explorer.png[メトリクス・エクスプローラー]

== PromQLクエリ

ifdef::ss-ff-stackpacks2_enabled[]
クエリ入力フィールドには、メトリック名、ラベル名、値、およびサポートされるPromQL関数の自動提案があります。完全なhttps://prometheus.io/docs/prometheus/latest/querying/basics/[PromQL ガイドとリファレンスは] Prometheus ドキュメントを参照してください。SUSE Observabilityには、どのクエリでも使用できる2つのデフォルトパラメータも追加されています:`+${__interval}+` と`+${__rate_interval}+` 。これらは、チャートの解像度に合わせて集計間隔を自動的に拡大縮小するために使用できるxref:/setup/custom-integrations/metric-bindings/writing-promql.adoc[（詳細]）。
endif::[]

ifndef::ss-ff-stackpacks2_enabled[]
