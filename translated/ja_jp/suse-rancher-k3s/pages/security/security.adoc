このセクションでは、K3sクラスタを保護する方法と手段について説明します。2つのセクションに分かれている。これらのガイドは、k3sが組み込みetcdで動作していることを前提としている。

最初に、ハードニング・ガイドはK3sクラスタをセキュアにするためのセキュリティ・ベスト・プラクティスのリストを提供している。

* xref:security/hardening-guide.adoc[硬化ガイド]

第二に、ハード化されたクラスタを検証するための自己評価である。現在、2種類の査定を用意している：

* xref:security/self-assessment-1.9.adoc[CIS 1.9ベンチマーク自己評価ガイド]、K3sバージョンv1.27-v1.29用
* xref:security/self-assessment-1.10.adoc[CIS 1.10ベンチマーク自己評価ガイド]、K3sバージョンv1.28-v1.31用
* xref:security/self-assessment-1.11.adoc[CIS 1.11ベンチマーク自己評価ガイド]、K3sバージョンv1.29-v1.34用
