= K3sのアンインストール
:page-languages: [en、ja、ko、zh]。
:revdate: 2026-01-05
:page-revdate: {revdate}

[CAUTION]
====
K3sをアンインストールすると、データが失われる可能性があります！
====

インストールスクリプトを使用してK3sをインストールした場合、インストール中にK3sをアンインストールするスクリプトが生成されます。

アンインストールスクリプトを実行すると、K3sと実行中のすべてのPodが停止し、ローカルのクラスタデータストア、xref:add-ons/storage.adoc#_setting_up_the_local_storage_provider[Local Storage]Persistent Volumeデータ、ノード構成、およびすべてのスクリプトとCLIツールが削除されます。

外部のデータストアからデータを削除したり、外部のKubernetes Persistent Volumesを使用してPodが作成したデータを削除することはありません。

ノードをアンインストールして再インストールした後に既存のクラスタに再参加させる場合は、必ずクラスタからノードを削除して、ノードのパスワードの秘密が削除されるようにしてください。詳細については、xref:architecture.adoc#_how_agent_node_registration_works[ノード登録の]ドキュメントを参照してください。

== サーバーのアンインストール

サーバーノードからK3sをアンインストールするには、以下を実行します：

[,bash]
----
/usr/local/bin/k3s-uninstall.sh
----

== エージェントのアンインストール

エージェントノードからK3sをアンインストールするには、以下を実行します：

[,bash]
----
/usr/local/bin/k3s-agent-uninstall.sh
----
