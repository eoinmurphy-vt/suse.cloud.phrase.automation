= インストール
:page-languages: [en、ja、ko、zh]。
:revdate: 2025-09-11
:page-revdate: {revdate}

このセクションでは、K3を様々な環境に設置するための手順を説明します。K3sのインストールを開始する前に、xref:installation/requirements.adoc[必要条件を満たして]いることを確認してください。

xref:installation/configuration.adoc[設定オプションは]、K3をインストールする際に利用可能なオプションに関するガイダンスを提供します。

xref:installation/private-registry.adoc[プライベートレジストリの構成では]、`+registries.yaml+` を使用してコンテナイメージレジストリの認証とミラーリングを構成します。

xref:installation/registry-mirror.adoc[Embedded Registry Mirror]は、ノード間で画像をピアツーピアで共有するために、組み込みの分散画像レジストリミラーを有効にする方法を示しています。

xref:installation/airgap.adoc[エアギャップインストールでは]、インターネットに直接アクセスできない環境でK3をセットアップする方法を詳しく説明します。

xref:installation/server-roles.adoc[サーバーロールの管理は]、専用の`+control-plane+` または`+etcd+` サーバーでK3をセットアップする方法を詳細に説明します。

xref:installation/packaged-components.adoc[パッケージ化されたコンポーネントの管理]では]、パッケージ化されたコンポーネントを無効にする方法、または自動展開マニフェストを使用して独自のコンポーネントをインストールする方法について詳しく説明します。

xref:installation/uninstall.adoc[K3sをアンインストールする]ホストからK3sを削除する方法を詳しく説明します。
