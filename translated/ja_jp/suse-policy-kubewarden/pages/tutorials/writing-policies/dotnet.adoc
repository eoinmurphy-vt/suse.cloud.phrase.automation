現在、.NET CoreはWebAssembly WASIプラットフォームを実験的にサポートしている。
これはhttps://github.com/SteveSandersonMS/dotnet-wasi-sdk[`+dotnet-wasi-sdk+`]プロジェクトからのものです。

[NOTE]
====

.NET Coreのコードを書いたり実行したりするのに、Windowsのインストールは必要ない。
すべてはLinuxやmacOSマシンでも可能だ。
====


== 現状

ポリシー作成者は以下のリソースを利用できる：

* https://github.com/kubewarden/policy-sdk-dotnet[{project-name}.NET Core SDK]: ポリシーの記述プロセスを簡素化するオブジェクトと関数のセットを提供します。
* https://github.com/kubewarden/policy-sdk-dotnet/tree/main/example[{short-project-name} ポリシーの例]：これは実用的なポリシーの例である。

制限はない。
SDKは、検証ポリシーと変異ポリシーの両方を書くことができる。

Kubernetesのオブジェクトを扱うためにhttps://www.nuget.org/packages/KubernetesClient.Models[`+KubernetesClient.Models+`]ライブラリを使ってKubernetesのオブジェクトを扱うことができる。
