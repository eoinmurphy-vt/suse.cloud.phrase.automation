= メモリテンプレート変数
:revdate: 2025-08-15
:page-revdate: {revdate}

メモリ・テンプレート変数で収集された情報は、システムで利用可能な（揮発性）メモリの属性を定義します。

|===
| 可変 | 説明 | より

| `+${Memory/Modules}+`
| メモリーモジュール
| 1.7.0

| `+${Memory/SupportedPageSizes}+`
| 許容メモリ・ページ・サイズ
| 1.7.0

| `+${Memory/TotalPhysicalBytes}+`
| 総メモリ量
| 1.7.0

| `+${Memory/TotalUsableBytes}+`
| 使用可能なメモリの合計
| 1.7.0
|===
