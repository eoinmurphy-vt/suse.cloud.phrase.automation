= MachineInventorySelector リファレンス
:revdate: 2024-09-24
:page-revdate: {revdate}

MachineInventorySelectorは、適用されたセレクタ（通常はMachineInventoryラベル値のパターンマッチング）に基づいてMachineInventoryを選択する。

MachineInventorySelectorsには2つの条件がある：

* `+InventoryReady+`MachineInventorySelectorが一致するMachineInventoryを見つけ、MachineInventoryの所有者として自身を正常に設定した場合、trueになる。
* `+Ready+`セレクタがすでにマシンを採用し、kubernetesのプロビジョニングプロセス（ノードのブートストラップ）を開始しているかどうかを追跡します。
