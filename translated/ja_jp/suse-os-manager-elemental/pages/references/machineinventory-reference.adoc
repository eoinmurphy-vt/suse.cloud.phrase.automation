= MachineInventory
:revdate: 2025-07-03
:page-revdate: {revdate}

新しいホストが正常に登録されると、{elemental_operator_name} 、その特定のホストを表すMachineInventoryリソースが作成される。
MachineInventoryは、登録プロセス中に取得した追跡対象ホストのTPMハッシュを保存し、マシン上で任意のコマンド（プラン）を実行できるようにする。

MachineInventoryには2つの条件がある：

* `+AdoptionReady+`これは、マシンがセレクタによってクラスタの一部に採用されたことを示す。
* `+Ready+`これは、マシンが{elemental-product-name} OSで登録され、プロビジョニングされたことを示す。
