= 建築概要
:revdate: 2025-11-06
:page-revdate: {revdate}

{elemental-product-name} という2つの主要プロジェクトの組み合わせである。link:{elemental_toolkit_url}[{elemental_toolkit_name}]とlink:{elemental_operator_url}[{elemental_operator_name}].

これによりlink:{elemental_toolkit_url}[{elemental_toolkit_name}]は、OCIコンテナ・イメージからのOSインストールとアップデートを可能にするため、コンテナ・レジストリを介してOSを配布したり取得したりできる。

そのlink:{elemental_operator_url}[{elemental_operator_name}]は、{elemental-product-name} Toolkit を活用して、OS プロビジョニングと OS 管理機能で Rancher を拡張する。
とのギャップを埋める。link:{elemental_toolkit_url}[{elemental_toolkit_name}]とhttps://documentation.suse.com/cloudnative/rancher-manager/latest/en/cluster-deployment/launch-kubernetes-with-rancher.html[RKE2/K3s Clusterのプロビジョニングの] Rancherギャップを埋め、OSのないホストから完全に構成されたKubernetes Clusterまでシームレスな体験を提供する。

これは、{elemental-product-name} Toolkitと{elemental-product-name} Operatorプロジェクトから借用したxref:rancher-os-management/architecture/architecture-components.adoc[コンポーネントの]おかげで可能となった、{elemental-product-name} が提供するxref:rancher-os-management/architecture/services/architecture-services.adoc[一連のサービスによって]達成される。
