= サービス
:revdate: 2025-09-08
:page-revdate: {revdate}

{elemental-product-name} 機能は以下のサービスリストを通じて提供される：

* xref:rancher-os-management/architecture/services/architecture-machineonboarding.adoc[マシン・オンボーディング]：ホストを OS プロビジョニングし、{elemental-product-name} Operator カタログに登録することができます。
* xref:rancher-os-management/architecture/services/architecture-clusterdeployment.adoc[Kubernetes Cluster Deployment]:{elemental-product-name} Operatorカタログに登録されている利用可能なホスト上でKubernetes Clusterの作成をトリガーします。
