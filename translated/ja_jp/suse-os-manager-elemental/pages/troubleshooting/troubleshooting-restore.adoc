= リストアのトラブルシューティング
:revdate: 2025-07-03
:page-revdate: {revdate}

[WARNING]
====
リストアが実行された場合、`+rancher-system-agent+` をエレメンタル・ノード上で再起動しないでください。 が古くなり、以下のエラーで終了する可能性があります：

`+panic: error while connecting to Kubernetes cluster: the server has asked for the client to provide credentials+`

この問題に直面した場合は、以下の手順に従ってください。
====


リストアを開始する前に、`+/var/lib/rancher/agent/rancher2_connection_info.json+` をエレメンタルノードから Rancher UI でアクセスできる場所にコピーする必要があります。

ファイルをコピーしたら、https://github.com/rancher/elemental[{elemental-product-name} リポジトリから] `+rancher-agent-token-update.sh+` スクリプトをダウンロードする：

[,shell]
----
wget -q https://raw.githubusercontent.com/rancher/elemental/main/scripts/rancher-agent-token-update && chmod +x rancher-agent-token-update
----

追加オプションなしでスクリプトを実行する：

[,shell]
----
./rancher-agent-token-update
----

リストアが正常に完了したら、`+rancher2_connection_info.json+` をエレメンタル・ノードにコピーし、パス`+/var/lib/rancher/agent/rancher2_connection_info.json+` に戻す。最後に、`+rancher-system-agent+` サービスを再起動する：

[,shell]
----
systemctl restart rancher-system-agent
----
