= Rancherアップグレードのトラブルシューティング
:revdate: 2025-07-03
:page-revdate: {revdate}

[WARNING]
====
{elemental-product-name} クラスタが定義されている場合、Rancher v2.7.2 へのアップグレードは失敗します。rancherポッドがクラッシュループにはまり( https://github.com/rancher/rancher/issues/41145).
====

この問題は、少なくとも1つの{elemental-product-name} クラスタが定義されている場合にのみ発生します。

この問題を回避するには、`+machineinventoryselectortemplate+` という名前の空の`+dynamicschemas.management.cattle.io+` リソースを作成します：

[,shell]
----
kubectl apply -f - <<EOF
kind: DynamicSchema
apiVersion: management.cattle.io/v3
metadata:
  name: machineinventoryselectortemplate
EOF

----

クラッシュループは停止し、Rancherのアップグレードは正常に完了します。
