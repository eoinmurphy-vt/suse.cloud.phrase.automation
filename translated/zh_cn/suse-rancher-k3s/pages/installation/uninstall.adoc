= 卸载 K3s
:page-languages: [EN, JA, KO, ZH]
:revdate: 2026-01-05
:page-revdate: {revdate}

[CAUTION]
====
卸载 K3s 可能会导致数据丢失！
====

如果使用安装脚本安装 K3s，则在安装过程中会生成卸载 K3s 的脚本。

运行卸载脚本会停止 K3s 和所有正在运行的 pod，并删除本地群集数据存储、xref:add-ons/storage.adoc#_setting_up_the_local_storage_provider[本地存储]持久卷数据、节点配置以及所有脚本和 CLI 工具。

它不会删除外部数据存储中的任何数据，也不会删除使用外部 Kubernetes 持久卷的 pod 创建的任何数据。

如果计划在卸载和重新安装后将节点重新加入现有群集，请务必从群集中删除该节点，以确保删除节点密码秘 密。更多信息，请参阅xref:architecture.adoc#_how_agent_node_registration_works[节点注册]文档。

== 卸载服务器

要从服务器节点卸载 K3s，请运行

[,bash]
----
/usr/local/bin/k3s-uninstall.sh
----

== 卸载代理

要从代理节点卸载 K3，请运行

[,bash]
----
/usr/local/bin/k3s-agent-uninstall.sh
----
