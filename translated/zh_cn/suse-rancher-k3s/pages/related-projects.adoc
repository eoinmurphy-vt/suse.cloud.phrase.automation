= 相关项目
:page-languages: [EN, JA, KO, ZH]
:revdate: 2025-03-31
:page-revdate: {revdate}

欢迎实施 K3s 分配的项目加入，以帮助扩大社区。这些项目有助于

* 探索 K3 的功能和潜在应用。
* 在不同的平台和环境中扩展 K3 的功能。
* 简化创建大型高可用性（HA）K3s 集群的过程。

== k3s-ansible

对于希望启动多节点 K3s 集群并熟悉 ansible 的用户，请访问https://github.com/k3s-io/k3s-ansible[] k3s-io/k3s-ansible 存储库。这套 ansible playbooks 提供了在节点上安装 K3 的便捷方法，让您可以专注于集群的配置而不是安装过程。

== k3sup

https://github.com/alexellis/k3sup[] k3sup 是另一个简化 K3s 集群设置过程的项目。该项目由 golang 编写，只需 ssh 访问节点即可。它还提供了一种方便的方法，可将 K3 与外部数据存储（而不仅仅是嵌入式 etcd）一起部署。

== autok3s

另一个配置工具https://github.com/cnrancher/autok3s[autok3s] 提供了一个图形用户界面，用于在一系列云提供商、虚拟机和本地机器上配置 k3s 集群。对于喜欢使用图形界面来证明 K3s 群集的用户来说，该工具非常有用。

== hetzner-k3s

对于在 Hetzner Cloud 上部署的用户，请访问link:https://github.com/vitobotta/hetzner-k3s[hetzner-k3s]。该 CLI 工具由 Crystal 编写，可处理在 Hetzner Cloud 上建立 K3s 集群所需的其他步骤。
