本节介绍确保 K3s 集群安全的方法和手段。它分为两个部分。这些指南假设 k3s 使用嵌入式 etcd 运行。

首先，加固指南提供了一份确保 K3s 集群安全的最佳安全实践清单。

* xref:security/hardening-guide.adoc[硬化指南]

其次是验证加固集群的自我评估。我们目前有两种不同的评估方法：

* xref:security/self-assessment-1.9.adoc[CIS 1.9 基准自我评估指南》]，K3s 版本 v1.27-v1.29
* xref:security/self-assessment-1.10.adoc[CIS 1.10 基准自我评估指南》]，适用于 K3s 版本 v1.28-v1.31
* xref:security/self-assessment-1.11.adoc[CIS 1.11 基准自我评估指南》]，适用于 K3s 版本 v1.29-v1.34
