== 先决条件
:revdate: 2025-11-06
:page-revdate: {revdate}

* 配置了 Rancher 服务器（版本 2.7.0 或更高版本）（设置了服务器URL）
 ** 要配置 Rancher`+server-url+` ，请查看https://documentation.suse.com/cloudnative/rancher-manager/latest/en/rancher-admin/rancher-admin.html#_first_log_in[文档] Rancher
* 配备 TPM 2.0 的机器（裸机或虚拟化机器
 ** 提示 1：Libvirt 允许为虚拟机设置虚拟 TPM，xref:operator-operational-tasks/tpm.adoc#_add_tpm_module_to_virtual_machine[示例如下]
 ** 提示 2：您可以在缺少 TPM 2.0 模块xref:operator-operational-tasks/tpm.adoc#_add_tpm_emulation_to_bare_metal_machine[示例的]裸机上启用 TPM 仿真。
 ** 提示 3：确保在 x86-64 上使用 UEFI（而非 BIOS），否则 ISO 无法启动
 ** 提示 4：建议容量至少为 25 GB。详情请查看xref:installation/installation.adoc#_deployed_partition_table[{elemental-product-name} 分区表]
 ** 提示 5：对 CPU 和内存的要求取决于所安装的 Kubernetes 版本，例如https://documentation.suse.com/cloudnative/k3s/latest/en/installation/requirements.html#_hardware[] K3s 或https://documentation.suse.com/cloudnative/rke2/latest/en/install/requirements.html#_hardware[] RKE2
* 舵手软件包管理器 (https://helm.sh/)
* 用于 ARM (aarch64) - 一张 SD 卡（32 GB 或更大，必须是*高速*卡 - 40MB/s 的写入速度也可）和一个 USB 盘用于安装
