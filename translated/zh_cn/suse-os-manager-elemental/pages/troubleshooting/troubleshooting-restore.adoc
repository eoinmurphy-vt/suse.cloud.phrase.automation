= 故障排除还原
:revdate: 2025-07-03
:page-revdate: {revdate}

[WARNING]
====
执行还原时，请勿重新启动元素节点上的`+rancher-system-agent+` ，因为它可能会过时，并最终出现以下错误：

`+panic: error while connecting to Kubernetes cluster: the server has asked for the client to provide credentials+`

如果您遇到这个问题，请按照以下步骤操作。
====


启动还原之前，需要将`+/var/lib/rancher/agent/rancher2_connection_info.json+` 从元素节点复制到可以通过 Rancher UI 访问的地方。

文件复制完成后，从https://github.com/rancher/elemental[{elemental-product-name} 资源库]下载`+rancher-agent-token-update.sh+` 脚本：

[,shell]
----
wget -q https://raw.githubusercontent.com/rancher/elemental/main/scripts/rancher-agent-token-update && chmod +x rancher-agent-token-update
----

执行脚本，不带任何其他选项：

[,shell]
----
./rancher-agent-token-update
----

还原成功后，将`+rancher2_connection_info.json+` 复制回元素节点的路径`+/var/lib/rancher/agent/rancher2_connection_info.json+` 。最后，重新启动`+rancher-system-agent+` 服务：

[,shell]
----
systemctl restart rancher-system-agent
----
