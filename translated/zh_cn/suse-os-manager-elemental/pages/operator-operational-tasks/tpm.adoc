= 可信平台模块 2.0（TPM）
:revdate: 2024-09-24
:page-revdate: {revdate}

可信平台模块（TPM，又称 ISO/IEC 11889）是安全加密处理器的国际标准，是一种专用微控制器，旨在通过集成加密密钥确保硬件安全。该术语也可指符合标准的芯片。

== 为虚拟机添加 TPM 模块

将 TPM 添加到虚拟机的简单方法是使用带有 Virt-manager 的 Libvirt

=== 创建虚拟机

启动 virt-manager 后创建新虚拟机

image::tpm1.png[创建新虚拟机]

=== 验证和编辑硬件模块列表

在 "硬件配置 "页面，验证模块列表并单击 *_添加硬件_*按钮

image::tpm2.png[设备列表]

=== 为虚拟机添加 TPM 模块

从仿真设备列表中选择 TPM 模块并将其添加到虚拟机中

image::tpm3.png[添加 TPM 模块]

=== 完成虚拟机配置

在最后一个屏幕上再次验证是否正确添加了 TPM 模块

image::tpm4.png[Verify TPM]

== 为裸机添加 TPM 仿真

在应用`+#!yaml MachineRegistration+` 时，在 yaml 中添加以下密钥 `+config:elemental:registration:emulate-tpm: true+`

[IMPORTANT]
====
如果计划在多台机器上部署 TPM 仿真，请确保设置`+config:elemental:registration:emulated-tpm-seed: -1+`，以便每台机器上用于 TPM 仿真的种子是随机的。否则，所有部署的机器都将获得相同的 TPM 哈希值，只有最后注册的机器才有效。
====
