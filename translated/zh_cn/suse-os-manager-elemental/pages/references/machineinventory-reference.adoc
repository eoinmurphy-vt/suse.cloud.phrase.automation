= MachineInventory
:revdate: 2025-07-03
:page-revdate: {revdate}

新主机注册成功后，{elemental_operator_name} 会创建一个代表该特定主机的 MachineInventory 资源。
MachineInventory 存储在注册过程中检索到的被跟踪主机的 TPM 哈希值，允许在机器上执行任意命令（计划）。

机器清单有两个条件：

* `+AdoptionReady+`表示该机器已被一个选择器采纳为群集的一部分。
* `+Ready+`表示该机器已注册并配置了{elemental-product-name} 操作系统。
