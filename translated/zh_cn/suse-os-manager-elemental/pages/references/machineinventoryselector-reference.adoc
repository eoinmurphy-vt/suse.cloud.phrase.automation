= 机器清单选择器参考
:revdate: 2024-09-24
:page-revdate: {revdate}

机器清单选择器（MachineInventorySelector）根据应用的选择器（通常是机器清单标签值的模式匹配）选择机器清单。

MachineInventorySelectors 有两个条件：

* `+InventoryReady+`，如果 MachineInventorySelector 找到了匹配的 MachineInventory，并成功将自己设置为 MachineInventory 所有者，则变为 true。
* `+Ready+`，跟踪选择器是否已经采用了一台机器并启动了 kubernetes 配置流程（节点引导）。
