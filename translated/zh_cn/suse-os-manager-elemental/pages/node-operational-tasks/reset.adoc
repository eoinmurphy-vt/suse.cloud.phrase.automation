= 机器重置
:revdate: 2025-09-08
:page-revdate: {revdate}

有两种方法可以将{elemental-product-name} 机器重置为原始状态或退出运行：

. 删除群集时，将重置所有相关机器
+
image::reset-cluster-deletion.png[删除群集以重置所有机器]

. 管理群集时，只需删除需要重置的节点即可
+
image::reset-single-node-deletion.png[删除单个节点以重置它]

== 重置工作流程

一旦相关`+MachineInventory+` 被标记为删除，机器上运行的`+elemental-system-agent+` 将执行重置计划。

如果机器仍在运行，该计划将：

. 在恢复模式下重启机器。
. 执行`+systemctl start elemental-register-reset+` 。 +
这将获取远程`+MachineRegistration+` ，并应用`+spec.config.elemental.reset+` 选项重置机器。 +
将创建一个新的`+MachineInventory+` ，并再次应用`+MachineRegistration+` 中定义的`+spec.config.cloud-config+` 。

请注意，`+MachineRegistration+` 引用将*不会*更改，机器将*不会*重新安装，如果重置为`+enabled+` ，则`+COS_PERSISTENT+` 和`+COS_OEM+` 分区将被默认清除。有关详细信息，请查阅xref:installation/installation.adoc#_deployed_partition_table[分区表]。

由于`+cloud-config+` 会在重置工作流程中重新应用，因此您可以重置机器以应用`+MachineRegistration+` 定义中的更新，例如轮换`+users+` 凭据和授权密钥。强烈建议启用`+reset-oem+` 选项，以避免与之前配置的云配置冲突。

如果需要将机器绑定到不同的`+MachineRegistration+` 并触发新的完整安装，则需要使用新的镜像重新进行配置。

== 启用机器复位

为了让机器自动重置，应切换`+MachineRegistration+` 的`+spec.config.elemental.reset.enabled+` 标志。 +
默认情况下是关闭的，但一旦激活，所有新创建的`+MachineInventory+` 都将自动继承这一设置。 +
例如

[,yaml]
----
apiVersion: elemental.cattle.io/v1beta1
kind: MachineRegistration
metadata:
  name: fire-nodes
  namespace: fleet-default
spec:
  config:
    elemental:
      reset:
        enabled: true
        reset-persistent: true
        reset-oem: true
        # These cloud-init configs will be created during reset and will persist on the system after
        config-urls:
          - "https://my.cloud.init/reset-plan-1.yaml"
          - "https://my.cloud.init/reset-plan-2.yaml"
        # You can select a different image to run the reset.
        # Note that this image will not be installed on the system.
        system-uri: "my.oci.registry/reset-image:latest"
        power-off: false
        reboot: true
----

也可以在`+MachineInventory+` 级别启用重置功能，例如，在某些情况下，一些机器是物理机，将受益于自动重置，而另一些机器是虚拟机，只需根据需要销毁和重新配置即可。 +
为了标记单个`+MachineInventory+` 以允许重置，可以使用`+elemental.cattle.io/resettable: true+` 注释。 +
例如

[,yaml]
----
apiVersion: elemental.cattle.io/v1beta1
kind: MachineInventory
metadata:
  annotations:
    elemental.cattle.io/resettable: "true"
----
