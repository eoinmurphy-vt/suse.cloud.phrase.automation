= 服务
:revdate: 2025-09-08
:page-revdate: {revdate}

{elemental-product-name} 功能通过以下服务清单提供：

* xref:rancher-os-management/architecture/services/architecture-machineonboarding.adoc[机器入驻]：允许主机在{elemental-product-name} Operator 目录中进行操作系统配置和注册。
* xref:rancher-os-management/architecture/services/architecture-clusterdeployment.adoc[Kubernetes 群集部署]：触发在{elemental-product-name} Operator 目录中注册的可用主机上创建 Kubernetes 群集。
