= 结构概述
:revdate: 2025-11-06
:page-revdate: {revdate}

{elemental-product-name} 是两个主要项目的结合：link:{elemental_toolkit_url}[{elemental_toolkit_name}]和link:{elemental_operator_url}[{elemental_operator_name}].

操作系统link:{elemental_toolkit_url}[{elemental_toolkit_name}]可以从 OCI 容器映像安装和更新操作系统，因此操作系统可以通过容器注册表进行分发和检索。

Rancherlink:{elemental_operator_url}[{elemental_operator_name}]利用{elemental-product-name} 工具包扩展了 Rancher 的操作系统配置和操作系统管理功能。
它弥补了link:{elemental_toolkit_url}[{elemental_toolkit_name}]和https://documentation.suse.com/cloudnative/rancher-manager/latest/en/cluster-deployment/launch-kubernetes-with-rancher.html[RKE2/K3s 集群配置] Rancher之间的桥梁，提供从无操作系统的主机到完全配置的 Kubernetes 集群的无缝体验。

这是通过{elemental-product-name} 提供的xref:rancher-os-management/architecture/services/architecture-services.adoc[一系列服务]实现的，这些xref:rancher-os-management/architecture/services/architecture-services.adoc[服务]借用了{elemental-product-name} 工具包和{elemental-product-name} Operator 项目xref:rancher-os-management/architecture/architecture-components.adoc[的组件]。
