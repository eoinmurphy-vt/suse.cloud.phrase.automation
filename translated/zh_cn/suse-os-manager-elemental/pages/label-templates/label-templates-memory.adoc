= 内存模板变量
:revdate: 2025-08-15
:page-revdate: {revdate}

内存模板变量中收集的信息定义了系统可用（易失）内存的属性。

|===
| 可变 | 说明 | 从

| `+${Memory/Modules}+`
| 内存模块
| 1.7.0

| `+${Memory/SupportedPageSizes}+`
| 允许的内存页面大小
| 1.7.0

| `+${Memory/TotalPhysicalBytes}+`
| 内存总量
| 1.7.0

| `+${Memory/TotalUsableBytes}+`
| 可用内存总量
| 1.7.0
|===
