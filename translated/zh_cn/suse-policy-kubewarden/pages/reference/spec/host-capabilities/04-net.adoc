{project-name} 策略不能在 WebAssembly 执行上下文中发出网络请求。

网络操作可以利用主机提供的一系列功能来完成。

== DNS 主机查询

该函数从策略给出的 FQDN 开始执行 DNS 查找。

=== 缓存

查询结果缓存 1 分钟。

=== 通信协议

这是用于公开此功能的 waPC 协议的说明：

==== waPC 功能 -`+v1/dns_lookup_host+` 输入

[,hcl]
----
# hostname - JSON encoded string
string
----

==== waPC 功能 -`+v1/dns_lookup_host+` 输出

[,hcl]
----

{
  # list of IPs
  "ips": [string]
}
----

与给定 FQDN 相关联的所有 IP 地址都将以字符串形式在响应中返回。
IPv4 和 IPv6 条目将作为同一响应的一部分返回。

例如，在请求`+busybox:latest+` 图像的清单摘要时，有效载荷将是

* 输入有效载荷： `+"google.com"+`
* 输出有效载荷： `+{ "ips": ["127.0.0.1"]}+`
