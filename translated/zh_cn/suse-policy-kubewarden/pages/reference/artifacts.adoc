== 二进制文件

== OCI 手工艺品

对于 Open Container Initiative (OCI) 的工件，项目会将所有内容发布在`+ghcr.io+` 下。 https://github.com/orgs/kubewarden/packages.

- https://github.com/orgs/kubewarden/packages/policies[`+kubewarden/packages/policies/*+`]
  是政策 Wasm OCI 工具。
- https://github.com/orgs/kubewarden/packages/charts[`+kubewarden/packages/charts/*+`]
  是作为 OCI 人工制品的 Helm 图表。他们遵循 SLSA 标准。
- 作为 OCI 图像发布的容器图像：`+kubewarden-controller+`,`+policy-server+`,`+audit-scanner+`,`+kubectl+` 。
- 还有`+packages/tests/*+` ，其中包含用于 e2e 测试的人工制品。

== 通过 HTTPS 获取舵手图表

{short-project-name} 在https://charts.kubewarden.io. 的 Helm 图表 HTTPS 资源库中发布 Helm 图表。这些图表没有 SLSA 或签名，但这类 Helm 资源库提供搜索功能。
