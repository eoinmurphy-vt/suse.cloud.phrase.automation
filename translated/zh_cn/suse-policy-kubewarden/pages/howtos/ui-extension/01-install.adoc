== 安装{short-project-name} UI 扩展

您可以将{short-project-name} UI 作为全局扩展进行安装，但也可以通过 Rancher UI 将{short-project-name} 控制器作为群集范围内的资源进行安装。

[NOTE]
====

对于气隙式安装，请按照xref:howtos/airgap/02-install.adoc[以下步骤]操作。

====


在扩展页面中，选择 "启用 "按钮，并选择添加 Rancher 扩展库的选项。启用后，"{short-project-name}"扩展项会自动出现。选择该项以安装扩展。安装完成后，即可将{short-project-name} 安装到所需的群集中。

=== 安装 {short-project-name}

按照前面的步骤，在群组内的{short-project-name} 侧菜单中会出现一个新项目。该仪表板页面将引导您完成安装过程，并完成先决条件。

[NOTE]
====

在安装向导的 "应用程序安装 "步骤中，"安装{short-project-name}"按钮可能保持灰色。如果出现这种情况，请刷新页面并返回到此步骤。

====


== 安装后

完成安装后，仪表板页面和侧边菜单现在包含新项目，即策略服务器、群集准入策略和准入策略。在这里，您可以创建策略服务器和策略来控制群集内的行为。

*_仪表板视图_*image:ui_dashboard.png[]UI

=== 启用默认策略服务器和策略

在仪表板页面中，您可以选择 "安装图表 "按钮来安装https://github.com/kubewarden/helm-charts/tree/main/charts/kubewarden-defaults[`+kubewarden-defaults+`]Helm 图表。该图表包括默认策略服务器和一些经过优化的策略。

安装图表后，可在可排序的表格中查看默认策略服务器的详细信息和相关策略。

*_策略服务器详细视图_*image:ui_policyserver_detail.png[]UI

=== 制定政策

创建策略时，最初会从策略网格中提供一个 "自定义策略 "选项。提供保单名称、单元和规则的必要信息。建议添加政策目录存储库，以访问{short-project-name} 的官方政策。

*_创建自定义策略_*image:ui_policy_custom.png[]UI

=== 附加功能

按照说明加入xref:howtos/ui-extension/02-metrics.adoc[监控]或xref:howtos/ui-extension/03-tracing.adoc[跟踪]。

== 气隙式安装

由于{short-project-name} 是 Rancher 官方扩展，Rancher 团队提供了自动生成扩展目录映像的机制。https://ranchermanager.docs.rancher.com/getting-started/installation-and-upgrade/other-installation-methods/air-gapped-helm-cli-install/publish-images#1-find-the-required-assets-for-your-rancher-version[安装 Rancher Manager] 时，这将被添加到`+rancher-images.txt+` 文件中。

将镜像镜像到可访问空中加油群集的注册表后，就可以在 Rancher UI 中导入镜像。这将创建一个本地 Helm 资源库，其中包含{short-project-name} UI 图表，以供安装。

=== 安装步骤

. https://ranchermanager.docs.rancher.com/how-to-guides/new-user-guides/kubernetes-resources-setup/secrets[创建]在`+cattle-ui-plugin-system+` 命名空间内注册表秘密。在*注册表域名*字段中输入图像地址的域名。
. 返回*扩展*页面（例如，`+https://cluster-ip/dashboard/c/local/uiplugins+` ）。
. 在右上角选择 "*管理扩展目录"*。image:ui_airgap_01.png[管理目录]
. 选择 "*导入扩展目录 "*按钮。 图像:ui_airgap_02.png[导入目录］
. 在*目录图像参考*字段中输入图像地址。
. 从 "*拉动秘密*"*下拉菜单*中选择您刚刚创建的秘密。image:ui_airgap_03.png[输入目录信息]
. 单击*加载*。现在扩展将处于*待定*状态。
. 返回*扩展*页面。
. 选择 "*可用*"选项卡，然后单击 "*重新加载*"按钮，以确保扩展程序列表是最新的。 image:ui_airgap_04.png[安装{short-project-name}]
. 找到刚刚添加的{short-project-name} 扩展，然后选择*安装*按钮。
