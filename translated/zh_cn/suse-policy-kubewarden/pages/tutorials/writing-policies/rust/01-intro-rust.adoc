https://www.rust-lang.org/[Rust]是可以生成 WebAssembly 模块的最成熟的编程语言。
WebAssembly 是 Rust 世界中的一等公民，因此 Rust 生态系统中的许多工具和板条箱都能开箱即用。

{project-name} 提供了一个https://crates.io/crates/kubewarden-policy-sdk[SDK] Rust，简化了编写策略的过程。
还有一个https://github.com/kubewarden/rust-policy-template[模板项目]，为使用https://github.com/cargo-generate/cargo-generate[`+cargo-generate+`]工具。

本文档介绍了如何使用这些项目来使用 Rust 编写{short-project-name} 策略。
{short-project-name} 的 Rust SDK 的细节。
详细信息请参阅https://docs.rs/kubewarden-policy-sdk/0.1.0[crate 文档]。

== 获取 Rust 依赖项

使用https://github.com/rust-lang/rustup[rustup] 安装 Rust 编译器及其工具。
请参阅 rustuphttps://rust-lang.github.io/rustup/installation/index.html[安装文档]。

安装`+rustup+` 后，添加 WebAssembly System Interface (WASI) 目标：

[subs="+attributes",console]
----
rustup target add wasm32-wasip1
----

== OSX 依赖项

要使用`+cargo-generate+` ，您需要添加 Xcode 工具集。
如果没有通过 Xcode 安装，以下命令会提供所需的依赖项：

[subs="+attributes",console]
----
xcode-select --install
----
