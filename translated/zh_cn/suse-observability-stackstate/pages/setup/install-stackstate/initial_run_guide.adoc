== 默认用户名和密码

[tabs]
====
Kubernetes::
+
--
SUSE Observability 默认配置为以下管理员帐户：

* *用户名* `+admin+`
* *密码*安装时设置。`+generate_values.sh+` 脚本收集这些信息，并以 MD5 哈希格式存储在 `+values.yaml+`
--

OpenShift::
+
--
SUSE Observability 默认配置为以下管理员帐户：

* *用户名* `+admin+`
* *密码*安装时设置。`+generate_values.sh+` 脚本收集这些信息，并以 MD5 哈希格式存储在 `+values.yaml+`
--
====
