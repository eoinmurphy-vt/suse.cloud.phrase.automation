== 概述

使用xref:use/security/k8s-service-tokens.adoc[服务令牌]验证 SUSE Observability 的身份，而无需关联用户账户。服务令牌是通过 CI/CD 或从代理或 Open Telemetry 发送遥测数据时进行身份验证的首选方式。xref:use/security/k8s-service-tokens.adoc[服务]令牌xref:use/security/k8s-service-tokens.adoc[文档]中介绍了如何创建和使用服务令牌。
