== 在进程代理中禁用特定网络协议的跟踪功能

要在进程代理中禁用特定网络协议的跟踪功能，请配置 Helm 图表设置`+processAgent.disabledProtocols+` 。下面的示例禁用了`+http2+` 和`+postgres+` 协议：

----
--set "processAgent.disabledProtocols[0]=http2" \
--set "processAgent.disabledProtocols[1]=postgres"
----

== 禁用进程代理

如果您的环境不支持进程代理，请通过 Helm 图表设置`+nodeAgent.containers.processAgent.enabled+` 将其禁用：

----
--set nodeAgent.containers.processAgent.enabled=false
----
