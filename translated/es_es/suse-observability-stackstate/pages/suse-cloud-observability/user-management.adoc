= Gestión de usuarios
:description: Gestión de usuarios de SUSE Cloud Observability

Los usuarios de los tenants de SUSE Cloud Observability (instancias de SUSE Cloud Observability) se gestionan con https://www.keycloak.org/[Keycloak]. Cada cliente (inquilino) tiene un dominio Keycloak dedicado. Se envía un enlace a la consola Keycloak en el mensaje de bienvenida cuando se crea un usuario.

SUSE Cloud Observability redirige a los usuarios a Keycloak para la autenticación. Se espera que los usuarios sean miembros de uno o más grupos de Keycloak.

Los grupos predefinidos de Keycloak:

* *realm-admin*: Los miembros de este grupo pueden iniciar sesión en la consola del dominio Keycloak y realizar operaciones de gestión de usuarios.
* *stackstate-k8s-troubleshooter*: A los usuarios de este grupo se les asigna la función de cliente de `+stackstate-k8s-troubleshooter+` Keycloak, que se asigna a la función de SUSE Cloud Observability con el mismo nombre. La función concede los permisos habituales de SUSE Cloud Observability.
* *stackstate-k8s-admin*: A los usuarios de este grupo se les asigna la función de cliente de `+stackstate-k8s-admin+` Keycloak, que se asigna a la función de SUSE Cloud Observability con el mismo nombre. La función otorga permisos privilegiados a SUSE Cloud Observability.

== URL de gestión de usuarios

Cuando se crea un nuevo usuario, éste recibe un mensaje de bienvenida que contiene un enlace a la página de gestión de usuarios de Keycloak. Este enlace es *exclusivo para el administrador del inquilino*, que es el primer usuario por defecto. El formato de la URL es: `+https://<keycloak_FQDN>/realms/<your_dedicated_Keycloak_realm>/account+`.

== Gestión de usuarios

* Inicie sesión en Keycloak Admin Console.

image::keycloak_admin_console.png[Consola de administración de Keycloak]

=== Gestionar usuarios

* En el menú de la izquierda, seleccione `+Users+` en la sección `+Manage+`.

==== Añadir un nuevo usuario

Haga clic en el botón `+Add user+` e introduzca la información del usuario, como "Nombre de usuario", "Correo electrónico", "Nombre" y "Apellidos".

. Deje vacío el campo `+Required users actions+`.
. Añada el usuario a los grupos necesarios.
. Haga clic en `+Save+`.
+
El mensaje de bienvenida con el enlace de registro y los enlaces al inquilino de SUSE Cloud Observability y a las consolas Keycloak Admin y Account se envían por correo electrónico al usuario.

NOTE: Para activar la cuenta, que incluye la confirmación por correo electrónico y el restablecimiento de la contraseña, el usuario debe seguir el enlace de registro.

image::keycloak_create_user.png[Keycloak Crear usuario]

==== Actualizar los datos del usuario

Para *editar los datos del usuario*, selecciónelo haciendo clic en Nombre de usuario.

. Cambie los detalles según sea necesario.
. Establezca una o más `+Required user actions+`, por ejemplo, para obligar a los usuarios a actualizar la contraseña o configurar contraseñas de un solo uso.
. Pulse el botón `+Save+` cuando haya terminado.

image::keycloak_update_user.png[Usuario de la actualización de Keycloak]

==== Eliminar un usuario

. Para *eliminar uno o varios usuarios*, seleccione los usuarios deseados y pulse el botón `+Delete user+`.

=== Miembros del grupo

. Inicie sesión en la consola de administración de Keycloak.
. En la sección `+Groups+`, busque el grupo que desea gestionar.
. Haga clic en el nombre del grupo para abrir los detalles del grupo y vaya a la pestaña `+Members+`.
. Para añadir un nuevo miembro al grupo, pulse el botón `+Add Member+` y seleccione los usuarios necesarios.
. Para eliminar usuarios del grupo, seleccione los usuarios de la lista y, a continuación, en el menú que aparece en la misma línea que el botón `+Add member+` marcado como "⋮", seleccione `+Leave group+`.
