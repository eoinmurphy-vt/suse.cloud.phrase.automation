== Ver diff de la última implementación

Cuando surjan problemas después de aplicar cambios a sus configuraciones de Kubernetes, eche un vistazo al último cambio de despliegue.

image::k8s/k8s-show-diff.png[]

Comparar la configuración actual con la anterior puede ayudarle a identificar los cambios específicos que podrían haber causado el problema.
