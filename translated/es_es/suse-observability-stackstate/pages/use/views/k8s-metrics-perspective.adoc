= Perspectiva métrica
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: Observabilidad de SUSE

La Perspectiva de métricas muestra las métricas del recurso seleccionado.

image::k8s/k8s-metrics-perspective.png[Perspectiva métrica]

== Gráficos

Los gráficos muestran los datos métricos de los componentes seleccionados casi en tiempo real: los datos se obtienen cada 30 segundos. Si se detiene un proceso y no se reciben más datos, el proceso acabará abandonando el gráfico, ya que los datos se desplazan hacia la izquierda al menos cada 30 segundos. Si llegan más datos durante el intervalo de 30 segundos, se enviarán a un gráfico.
