== Desactivar el seguimiento de un protocolo de red específico en el agente de proceso

Para desactivar el rastreo para un protocolo de red específico en el process-agent, configure el ajuste del gráfico Helm `+processAgent.disabledProtocols+`. El siguiente ejemplo desactiva los protocolos `+http2+` y `+postgres+`:

----
--set "processAgent.disabledProtocols[0]=http2" \
--set "processAgent.disabledProtocols[1]=postgres"
----

== Desactivar el agente de procesos

Si el agente de procesos no está soportado en su entorno, desactívelo a través de la configuración del cuadro de Helm `+nodeAgent.containers.processAgent.enabled+`:

----
--set nodeAgent.containers.processAgent.enabled=false
----
