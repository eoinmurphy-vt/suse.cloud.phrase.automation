Esta sección documenta el modelo de *presentación de la interfaz de usuario* introducido como parte de *StackPacks 2.0*.

== Visión general

El modelo de Presentación de Componentes documentado aquí declara cómo se presentan los componentes (por ejemplo, pods, VMs, servicios) en la interfaz de usuario {stackstate-product-name}.
Se centra exclusivamente en *cuestiones de presentación*, como la nomenclatura, los iconos, las tablas, las páginas de resumen y destacadas, los filtros y la colocación de los elementos de menú.

El objetivo de este modelo es proporcionar una forma más componible y extensible de definir el comportamiento de la interfaz de usuario para los componentes, en comparación con el enfoque existente, más fragmentado.
