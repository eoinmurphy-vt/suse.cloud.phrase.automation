== ¿Qué es {product_name}?

El Cluster API Extension for Rancher, también conocido como {product_name}, es un Operador Kubernetes que proporciona **integración entre Rancher Manager y Cluster API (CAPI)** con el objetivo de llevar el soporte completo de CAPI a Rancher.

Con {product_name}, puede:

* **Importe automáticamente clusters CAPI en Rancher**, instalando el Agente de Cluster Rancher en clusters aprovisionados CAPI.
* Implemente un flujo de trabajo **nativo de GitOps** para gestionar clústeres Kubernetes en Rancher.
* **Gestión simplificada de los proveedores CAPI** mediante el Operador CAPI.
