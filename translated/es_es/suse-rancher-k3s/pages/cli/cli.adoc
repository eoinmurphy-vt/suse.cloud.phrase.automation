= Herramientas CLI
:page-languages: [en, ja, ko, zh]
:revdate: 2024-09-11
:page-revdate: {revdate}

El binario K3s contiene una serie de herramientas adicionales que le ayudarán a gestionar su clúster.

|===
| Comando | Descripción

| `+k3s server+`
| Ejecutar un nodo servidor K3s, que lanza los componentes Kubernetes `+apiserver+`, `+scheduler+`, `+controller-manager+`, y `+cloud-controller-manager+`, además de un datastore y los componentes del agente. Consulte la xref:cli/server.adoc[documentación del comando`+k3s server+` ] para obtener más información.

| `+k3s agent+`
| Ejecute el nodo agente K3s, que lanza `+containerd+`, `+flannel+`, `+kube-router+` controlador de políticas de red, y los componentes Kubernetes `+kubelet+` y `+kube-proxy+`. Consulte la xref:cli/agent.adoc[documentación del comando`+k3s agent+` ] para obtener más información.

| `+k3s kubectl+`
| Ejecute el integrado https://kubernetes.io/docs/reference/kubectl[comando] Esta es una CLI para interactuar con el apiserver de Kubernetes.  Si la variable de entorno `+KUBECONFIG+` no está establecida, se intentará utilizar automáticamente el kubeconfig en `+/etc/rancher/k3s/k3s.yaml+`.

| `+k3s crictl+`
| Ejecute el integrado https://github.com/kubernetes-sigs/cri-tools/blob/master/docs/crictl.md[comando] Se trata de una CLI para interactuar con la interfaz de ejecución de contenedores (CRI) de Kubernetes. Útil para la depuración.

| `+k3s ctr+`
| Ejecute el integrado https://github.com/projectatomic/containerd/blob/master/docs/cli.md[comando] Este es un CLI para containerd, el demonio contenedor usado por K3s. Útil para la depuración.

| `+k3s token+`
| Gestionar tokens de arranque. Consulte la xref:cli/token.adoc[documentación del comando`+k3s token+` ] para obtener más información.

| `+k3s etcd-snapshot+`
| Realice copias de seguridad bajo demanda de los datos del clúster K3s y súbalos a S3. Consulte la xref:cli/etcd-snapshot.adoc[documentación del comando`+k3s etcd-snapshot+` ] para obtener más información.

| `+k3s secrets-encrypt+`
| Configure los K3 para cifrar los secretos cuando los almacene en el clúster. Consulte la xref:cli/secrets-encrypt.adoc[documentación del comando`+k3s secrets-encrypt+` ] para obtener más información.

| `+k3s certificate+`
| Gestionar los certificados K3s. Consulte la xref:cli/certificate.adoc[documentación del comando`+k3s certificate+` ] para obtener más información.

| `+k3s completion+`
| Generar scripts de finalización de shell para k3s

| `+k3s help+`
| Muestra una lista de comandos o la ayuda de un comando
|===
