K3s es un proyecto sandbox oficial https://www.cncf.io[del CNCF] que ofrece una distribución de Kubernetes certificada, ligera pero potente, diseñada para cargas de trabajo de producción en ubicaciones remotas con recursos limitados o en dispositivos IoT. Cuando se utiliza con SUSE® Rancher Prime, K3s es una distribución de Kubernetes fácil de instalar, ligera pero de alta disponibilidad, que se puede gestionar fácilmente dentro de la plataforma de orquestación Rancher.

== Características principales

* Distribuido como un único binario o una imagen mínima de contenedor.
* Almacén de datos ligero basado en sqlite3 como backend de almacenamiento predeterminado. etcd3, MySQL y Postgres también están disponibles.
* Envuelto en un sencillo lanzador que maneja gran parte de la complejidad de TLS y opciones.
* Seguro por defecto con valores predeterminados razonables para entornos ligeros.
* El funcionamiento de todos los componentes del plano de control de Kubernetes se encapsula en un único binario y proceso, lo que permite a K3 automatizar y gestionar operaciones de clúster complejas como la distribución de certificados.
* Se han minimizado las dependencias externas; los únicos requisitos son un kernel moderno y montajes cgroup.
* Empaqueta las dependencias necesarias para facilitar la creación de clusters "baterías-incluidas":
 ** containerd / cri-dockerd tiempo de ejecución de contenedores (CRI)
 ** Flannel Interfaz de red de contenedores (CNI)
 ** CoreDNS Cluster DNS
 ** Controlador Traefik Ingress
 ** Controlador del equilibrador de carga ServiceLB
 ** Controlador de políticas de red Kube-router
 ** Local-path-provisioner Controlador de volumen persistente
 ** Espejo de registro de imágenes de contenedores distribuidos Spegel
 ** Utilidades de host (iptables, socat, etc.)
