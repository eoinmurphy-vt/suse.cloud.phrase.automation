= Seguridad
:page-languages: [en, ja, ko, zh]
:revdate: 2025-12-31
:page-revdate: {revdate}

Esta sección describe la metodología y los medios para asegurar un cluster K3s. Está dividido en 2 secciones. Estas guías asumen que k3s funciona con etcd integrado.

En primer lugar, la guía de endurecimiento proporciona una lista de las mejores prácticas de seguridad para proteger un clúster K3s.

* xref:security/hardening-guide.adoc[Guía de endurecimiento]

En segundo lugar, está la autoevaluación para validar un clúster reforzado. Actualmente disponemos de dos evaluaciones diferentes:

* xref:security/self-assessment-1.9.adoc[Guía de autoevaluación de CIS 1.9 Benchmark], para K3s versión v1.27-v1.29
* xref:security/self-assessment-1.10.adoc[Guía de autoevaluación de CIS 1.10 Benchmark], para K3s versión v1.28-v1.31
* xref:security/self-assessment-1.11.adoc[Guía de autoevaluación de CIS 1.11 Benchmark], para K3s versión v1.29-v1.34
