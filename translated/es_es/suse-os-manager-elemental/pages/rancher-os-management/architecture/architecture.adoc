= Arquitectura general
:revdate: 2025-11-06
:page-revdate: {revdate}

{elemental-product-name} es la combinación de dos proyectos principales: el link:{elemental_toolkit_url}[{elemental_toolkit_name}] y el link:{elemental_operator_url}[{elemental_operator_name}].

La página link:{elemental_toolkit_url}[{elemental_toolkit_name}] permite la instalación y actualización del SO a partir de imágenes de contenedor OCI, de modo que el SO puede ser distribuido y recuperado a través de los registros de contenedor.

El sitio link:{elemental_operator_url}[{elemental_operator_name}] amplía Rancher con funcionalidades de aprovisionamiento y gestión de SO, aprovechando el {elemental-product-name} Toolkit.
Tiende un puente entre el link:{elemental_toolkit_url}[{elemental_toolkit_name}] y elhttps://documentation.suse.com/cloudnative/rancher-manager/latest/en/cluster-deployment/launch-kubernetes-with-rancher.html[aprovisionamiento de clústeres RKE2/K3s de Rancher de], para ofrecer una experiencia fluida al pasar de hosts sin SO a clústeres Kubernetes totalmente configurados.

Esto se consigue a través de un xref:rancher-os-management/architecture/services/architecture-services.adoc[conjunto de servicios] ofrecidos por {elemental-product-name} que son posibles gracias a los xref:rancher-os-management/architecture/architecture-components.adoc[componentes] prestados por los proyectos {elemental-product-name} Toolkit y {elemental-product-name} Operator.
