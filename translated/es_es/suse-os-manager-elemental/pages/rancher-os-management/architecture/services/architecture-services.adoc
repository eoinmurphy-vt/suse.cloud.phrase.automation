= Servicios
:revdate: 2025-09-08
:page-revdate: {revdate}

{elemental-product-name} La funcionalidad se proporciona a través de la siguiente lista de servicios:

* xref:rancher-os-management/architecture/services/architecture-machineonboarding.adoc[Machine Onboarding]: permite que los hosts se aprovisionen de SO y se registren en el catálogo {elemental-product-name} Operator.
* xref:rancher-os-management/architecture/services/architecture-clusterdeployment.adoc[Kubernetes Cluster Deployment]: activa la creación de un Kubernetes Cluster en el host disponible registrado en el catálogo {elemental-product-name} Operator.
