= Módulo de plataforma segura 2.0 (TPM)
:revdate: 2024-09-24
:page-revdate: {revdate}

Trusted Platform Module (TPM, también conocido como ISO/IEC 11889) es un estándar internacional para un criptoprocesador seguro, un microcontrolador dedicado diseñado para asegurar el hardware mediante claves criptográficas integradas. El término también puede referirse a un chip conforme a la norma.

== Añadir módulo TPM a la máquina virtual

Una forma sencilla de añadir TPM a una máquina virtual es utilizar Libvirt con Virt-manager

=== Crear máquina virtual

Después de iniciar virt-manager cree una nueva máquina virtual

image::tpm1.png[Crear nueva VM]

=== Verificar y editar la lista de módulos de hardware

En la pantalla de configuración de hardware, verifique la lista de módulos y haga clic en *_Añadir hardware_* en

image::tpm2.png[Lista de dispositivos]

=== Añadir módulo TPM a VM

De la lista de dispositivos emulados elija el módulo TPM y añádalo a la VM

image::tpm3.png[Añadir módulo TPM]

=== Finalizar la configuración de la máquina virtual

En la última pantalla verifique una vez más si el módulo TPM fue agregado correctamente

image::tpm4.png[Verificar TPM]

== Añadir emulación TPM a máquina bare metal

Durante la aplicación de `+#!yaml MachineRegistration+` añadir la siguiente clave para el yaml `+config:elemental:registration:emulate-tpm: true+`

[IMPORTANT]
====
Si planea desplegar más de 1 máquina con emulación TPM, asegúrese de configurar `+config:elemental:registration:emulated-tpm-seed: -1+`para que la semilla utilizada para la emulación TPM sea aleatoria por máquina. De lo contrario, obtendrá el mismo TPM Hash para todas las máquinas desplegadas y sólo será válido el último en registrarse.
