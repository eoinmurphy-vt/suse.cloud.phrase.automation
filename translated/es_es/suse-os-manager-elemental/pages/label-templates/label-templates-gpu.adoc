= Variables de plantilla GPU
:revdate: 2025-08-15
:page-revdate: {revdate}

La información recogida en las variables de la plantilla GPU define los atributos de las tarjetas gráficas instaladas en el sistema.

Los datos se exponen por número de tarjeta gráfica (0, 1, 2, ...)

Para hacer referencia a la primera tarjeta gráfica, puede omitirse el número de tarjeta gráfica (_`+<Number>+`_) puede omitirse (por ejemplo, la variable `+${GPU/GraphicsCards/Driver}+` es la misma que `+${GPU/GraphicsCards/0/Driver}+`).

|===
| Variable | Descripción | de

| `+${GPU/GraphicsCards/Driver}+`
| Controlador de tarjeta GPU
| 1.7.0

| `+${GPU/GraphicsCards/ProductName}+`
| Modelo de tarjeta GPU
| 1.7.0

| `+${GPU/GraphicsCards/VendorName}+`
| Vendedor de tarjetas GPU
| 1.7.0

| `+${GPU/GraphicsCards/_<Number>_/Driver}+`
| Controlador del _número de_ tarjeta GPU
| 1.7.0

| `+${GPU/GraphicsCards/_<Number>_/ProductName}+`
| _Número de_ tarjeta GPU Modelo
| 1.7.0

| `+${GPU/GraphicsCards/_<Number>_/VendorName}+`
| _Número de_ tarjeta GPU Proveedor
| 1.7.0

| `+${GPU/TotalCards}+`
| Número de tarjetas GPU del sistema
| 1.7.0
|===
