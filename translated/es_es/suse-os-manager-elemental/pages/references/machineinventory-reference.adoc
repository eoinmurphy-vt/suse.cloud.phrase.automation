= Inventario de máquinas
:revdate: 2025-07-03
:page-revdate: {revdate}

Cuando un nuevo host se registra correctamente, {elemental_operator_name} crea un recurso MachineInventory que representa a ese host en particular.
El MachineInventory almacena el hash TPM del host rastreado, recuperado durante el proceso de registro, y permite ejecutar comandos arbitrarios (planes) en la máquina.

Un MachineInventory tiene dos condiciones:

* `+AdoptionReady+`que indica que la máquina ha sido adoptada por un selector para formar parte de un cluster.
* `+Ready+`que indica que la máquina se ha registrado y aprovisionado con un sistema operativo {elemental-product-name}.
