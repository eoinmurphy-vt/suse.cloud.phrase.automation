== Requisitos previos
:revdate: 2025-11-06
:page-revdate: {revdate}

* Un servidor Rancher (v2.7.0 o posterior) configurado (server-url set)
 ** Para configurar Rancher `+server-url+`, consulte https://documentation.suse.com/cloudnative/rancher-manager/latest/en/rancher-admin/rancher-admin.html#_first_log_in[la documentación de]
* Una máquina (de metal desnudo o virtualizada) con TPM 2.0
 ** Pista 1: Libvirt permite configurar TPMs virtuales para máquinas virtuales xref:operator-operational-tasks/tpm.adoc#_add_tpm_module_to_virtual_machine[ejemplo aquí]
 ** Pista 2: Puede habilitar la emulación TPM en máquinas bare metal omitiendo el xref:operator-operational-tasks/tpm.adoc#_add_tpm_emulation_to_bare_metal_machine[ejemplo] del módulo TPM 2. xref:operator-operational-tasks/tpm.adoc#_add_tpm_emulation_to_bare_metal_machine[0 aquí]
 ** Pista 3: Asegúrese de que está utilizando UEFI (no BIOS) en x86-64, o la ISO no arrancará.
 ** Pista 4: Se recomienda un volumen mínimo de 25 GB. Para más información, consulte la xref:installation/installation.adoc#_deployed_partition_table[tabla de particiones{elemental-product-name} ] 
 ** Pista 5: Los requisitos de CPU y RAM dependen de la versión de Kubernetes instalada, por ejemplo https://documentation.suse.com/cloudnative/k3s/latest/en/installation/requirements.html#_hardware[] K3s o https://documentation.suse.com/cloudnative/rke2/latest/en/install/requirements.html#_hardware[] RKE2
* Gestor de paquetes Helm (https://helm.sh/)
* Para ARM (aarch64) - Una tarjeta SD (32 GB o más, debe ser *rápida* - se acepta una velocidad de escritura de 40 MB/s) y una memoria USB para la instalación.
