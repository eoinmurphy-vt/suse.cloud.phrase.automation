Os registros de contêineres podem ser usados para distribuir muitos tipos de objetos OCI.
De imagens de contêineres conhecidas a artefatos genéricos da OCI.
Os artefatos da OCI são usados para armazenar objetos, como políticas do {project-name}, gráficos do Helm e muito mais.

Esses são os recursos expostos pelo host {short-project-name} para interagir com os registros de contêineres.

== Obter o resumo do manifesto da OCI

Essa chamada de retorno calcula o resumo de um manifesto OCI. O resumo pode ser usado para identificar um objeto armazenado em um registro da OCI. Esse é um modo imutável, ao contrário do `+tags+`, que é mutável.

=== Armazenamento em cache

A computação do resumo envolve uma série de solicitações de rede entre o host da política {short-project-name} e o registro remoto.
Essas operações podem ser demoradas, portanto, os resultados são armazenados em cache por 1 minuto.

=== Autenticação

As interações com registros privados exigem que o host da política {short-project-name} se autentique no registro remoto.

O host da política usará o mesmo conjunto de credenciais usadas para obter políticas do registro remoto.

=== Protocolo de comunicação

Esta é a descrição do protocolo waPC usado para expor esse recurso:

==== Função waPC - `+v1/manifest_digest+` input

[,hcl]
----
# OCI URI - JSON encoded string
string
----

==== Função waPC - `+v1/manifest_digest+` output

[,hcl]
----
{
  # digest of the OCI object
  "digest": string
}
----

Por exemplo, ao solicitar o resumo do manifesto da imagem `+busybox:latest+`, o payload seria:

* Carga útil de entrada: `+"busybox:latest"+`
* Carga útil de saída: `+{ "digest": "sha256:69e70a79f2d41ab5d637de98c1e0b055206ba40a8145e7bddb55ccc04e13cf8f"}+`

== Manifesto da OCI

Essa chamada de retorno busca o manifesto dos objetos OCI. Quando disponíveis, essas informações podem ser usadas para identificar manifestos de imagens específicas, para uma ou mais plataformas. Ou um único manifesto de imagem para a imagem.

=== Armazenamento em cache

A obtenção dessas informações envolve solicitações de rede entre o host de política {short-project-name}e o registro remoto. Essas operações podem ser demoradas, portanto, os resultados são armazenados em cache por 1 minuto.

=== Autenticação
