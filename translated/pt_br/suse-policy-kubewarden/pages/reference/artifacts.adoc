== Binários

Nossa ferramenta `+kwctl+` CLI é publicada por meio de https://github.com/kubewarden/kwctl/releases[versões] do GitHub.

== Artefatos OCI

Para artefatos da Open Container Initiative (OCI), o projeto publica tudo em `+ghcr.io+` em https://github.com/orgs/kubewarden/packages.

- https://github.com/orgs/kubewarden/packages/policies[`+kubewarden/packages/policies/*+`]
  são os artefatos da política Wasm OCI.
- https://github.com/orgs/kubewarden/packages/charts[`+kubewarden/packages/charts/*+`]
  são os gráficos do Helm como artefatos OCI. Eles seguem os padrões da SLSA.
- as imagens de contêineres publicadas como imagens OCI: `+kubewarden-controller+`, `+policy-server+`, `+audit-scanner+`, `+kubectl+`.
- Há também o site `+packages/tests/*+` que contém artefatos usados em testes e2e.

== Gráficos do Helm via HTTPS
