= Segurança
:page-languages: [en, ja, ko, zh]
:revdate: 2025-12-31
:page-revdate: {revdate}

Esta seção descreve a metodologia e os meios de proteger um cluster K3s. Ele é dividido em duas seções. Esses guias pressupõem que o k3s esteja sendo executado com o etcd incorporado.

Primeiro, o guia de fortalecimento fornece uma lista de práticas recomendadas de segurança para proteger um cluster do K3s.

* xref:security/hardening-guide.adoc[Guia de endurecimento]

Em segundo lugar, está a autoavaliação para validar um cluster reforçado. Atualmente, temos duas avaliações diferentes disponíveis:

* xref:security/self-assessment-1.9.adoc[Guia de autoavaliação de benchmark do CIS 1.9], para K3s versão v1.27-v1.29
* xref:security/self-assessment-1.10.adoc[Guia de autoavaliação de benchmark do CIS 1.10], para K3s versão v1.28-v1.31
* xref:security/self-assessment-1.11.adoc[Guia de autoavaliação de benchmark do CIS 1.11], para K3s versão v1.29-v1.34
