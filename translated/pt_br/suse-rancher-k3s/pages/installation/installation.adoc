= Instalação
:page-languages: [en, ja, ko, zh]
:revdate: 2025-09-11
:page-revdate: {revdate}

Esta seção contém instruções para instalar os K3s em vários ambientes. Certifique-se de que você atendeu aos xref:installation/requirements.adoc[requisitos] antes de iniciar a instalação do K3s.

xref:installation/configuration.adoc[Opções de configuração] fornece orientação sobre as opções disponíveis ao instalar o K3s.

O xref:installation/private-registry.adoc[Private Registry Configuration] abrange o uso do site `+registries.yaml+` para configurar a autenticação e o espelhamento do registro de imagens de contêineres.

xref:installation/registry-mirror.adoc[O Embedded Registry Mirror] mostra como ativar o espelho de registro de imagem distribuído incorporado, para compartilhamento ponto a ponto de imagens entre nós.

xref:installation/airgap.adoc[O Air-Gap Install] detalha como configurar o K3s em ambientes que não têm acesso direto à Internet.

xref:installation/server-roles.adoc[O gerenciamento de funções de servidor] detalha como configurar os K3s com servidores dedicados `+control-plane+` ou `+etcd+`.

xref:installation/packaged-components.adoc[O gerenciamento de componentes] empacotados detalha como desativar componentes empacotados ou instalar seus próprios componentes usando manifestos de implantação automática.

xref:installation/uninstall.adoc[A desinstalação do K3s] detalha como remover o K3s de um host.
