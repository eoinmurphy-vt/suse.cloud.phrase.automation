= Desinstalação do K3s
:page-languages: [en, ja, ko, zh]
:revdate: 2026-01-05
:page-revdate: {revdate}

[CAUTION]
====
A desinstalação do K3s pode causar perda de dados!
====

Se você instalou o K3s usando o script de instalação, um script para desinstalar o K3s foi gerado durante a instalação.

A execução do script de desinstalação interrompe o K3s e todos os pods em execução e exclui o repositório de dados do cluster local, os dados xref:add-ons/storage.adoc#_setting_up_the_local_storage_provider[do Local Storage] Persistent Volume, a configuração do nó e todos os scripts e ferramentas da CLI.

Ele não remove dados de repositórios de dados externos ou criados por pods usando Volumes Persistentes do Kubernetes externos.

Se estiver planejando reintegrar um nó a um cluster existente após a desinstalação e reinstalação, certifique-se de excluir o nó do cluster para garantir que a senha secreta do nó seja removida. Consulte a documentação xref:architecture.adoc#_how_agent_node_registration_works[do Registro de Nó] para obter mais informações.

== Desinstalação de servidores

Para desinstalar o K3s de um nó do servidor, execute:

[,bash]
----
/usr/local/bin/k3s-uninstall.sh
----

== Desinstalação de agentes

Para desinstalar o K3s de um nó de agente, execute:

[,bash]
----
/usr/local/bin/k3s-agent-uninstall.sh
----
