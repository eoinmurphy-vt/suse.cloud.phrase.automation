= Ferramentas de CLI
:page-languages: [en, ja, ko, zh]
:revdate: 2024-09-11
:page-revdate: {revdate}

O binário do K3s contém várias ferramentas adicionais que o ajudam a gerenciar o cluster.

|===
| Comando | Descrição

| `+k3s server+`
| Execute um nó de servidor K3s, que inicia os componentes do Kubernetes `+apiserver+`, `+scheduler+`, `+controller-manager+` e `+cloud-controller-manager+`, além de um armazenamento de dados e os componentes do agente. Consulte a xref:cli/server.adoc[documentação do comando`+k3s server+` ] para obter mais informações.

| `+k3s agent+`
| Execute o nó do agente K3s, que inicia `+containerd+`, `+flannel+`, `+kube-router+` controlador de política de rede e os componentes do Kubernetes `+kubelet+` e `+kube-proxy+`. Consulte a xref:cli/agent.adoc[documentação do comando`+k3s agent+` ] para obter mais informações.

| `+k3s kubectl+`
| Execute o incorporado https://kubernetes.io/docs/reference/kubectl[comando] Esta é uma CLI para interagir com o apiserver do Kubernetes.  Se a variável de ambiente `+KUBECONFIG+` não estiver definida, isso tentará automaticamente usar o kubeconfig em `+/etc/rancher/k3s/k3s.yaml+`.

| `+k3s crictl+`
| Execute o incorporado https://github.com/kubernetes-sigs/cri-tools/blob/master/docs/crictl.md[comando] Esta é uma CLI para interagir com a interface de tempo de execução de contêiner (CRI) do Kubernetes. Útil para depuração.

| `+k3s ctr+`
| Execute o incorporado https://github.com/projectatomic/containerd/blob/master/docs/cli.md[comando] Esta é uma CLI para o containerd, o daemon de contêiner usado pelo K3s. Útil para depuração.

| `+k3s token+`
| Gerenciar tokens bootstrap. Consulte a xref:cli/token.adoc[documentação do comando`+k3s token+` ] para obter mais informações.

| `+k3s etcd-snapshot+`
| Realize backups sob demanda dos dados do cluster do K3s e faça o upload para o S3. Consulte a xref:cli/etcd-snapshot.adoc[documentação do comando`+k3s etcd-snapshot+` ] para obter mais informações.

| `+k3s secrets-encrypt+`
| Configure o K3s para criptografar segredos ao armazená-los no cluster. Consulte a xref:cli/secrets-encrypt.adoc[documentação do comando`+k3s secrets-encrypt+` ] para obter mais informações.

| `+k3s certificate+`
| Gerenciar certificados do K3s. Consulte a xref:cli/certificate.adoc[documentação do comando`+k3s certificate+` ] para obter mais informações.

| `+k3s completion+`
| Gerar scripts de conclusão de shell para o k3s

| `+k3s help+`
| Mostra uma lista de comandos ou a ajuda para um comando
|===
