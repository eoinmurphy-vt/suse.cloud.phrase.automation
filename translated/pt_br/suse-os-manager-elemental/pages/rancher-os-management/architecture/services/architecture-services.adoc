:revdate: 2025-09-08

{elemental-product-name} A funcionalidade é fornecida por meio da seguinte lista de serviços:

* xref:rancher-os-management/architecture/services/architecture-machineonboarding.adoc[Machine Onboarding]: permite que os hosts sejam provisionados para o sistema operacional e registrados no catálogo do operador {elemental-product-name}.
* xref:rancher-os-management/architecture/services/architecture-clusterdeployment.adoc[Implementação de cluster do Kubernetes]: aciona a criação de um cluster do Kubernetes no host disponível registrado no catálogo do operador {elemental-product-name}.
