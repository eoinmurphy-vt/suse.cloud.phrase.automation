= Visão geral da arquitetura
:revdate: 2025-11-06
:page-revdate: {revdate}

{elemental-product-name} é a combinação de dois projetos principais: o link:{elemental_toolkit_url}[{elemental_toolkit_name}] e o link:{elemental_operator_url}[{elemental_operator_name}].

O link:{elemental_toolkit_url}[{elemental_toolkit_name}] permite a instalação e as atualizações do sistema operacional a partir de imagens de contêineres da OCI, para que o sistema operacional possa ser distribuído e recuperado por meio de registros de contêineres.

O link:{elemental_operator_url}[{elemental_operator_name}] estende o Rancher com funcionalidades de provisionamento e gerenciamento de sistemas operacionais, aproveitando o {elemental-product-name} Toolkit.
Ele preenche a lacuna entre o link:{elemental_toolkit_url}[{elemental_toolkit_name}] e ohttps://documentation.suse.com/cloudnative/rancher-manager/latest/en/cluster-deployment/launch-kubernetes-with-rancher.html[provisionamento de clusters do] Rancher, para oferecer uma experiência perfeita, desde hosts sem sistema operacional até clusters do Kubernetes totalmente configurados.

Isso é obtido por meio de um xref:rancher-os-management/architecture/services/architecture-services.adoc[conjunto de serviços] oferecidos pelo site {elemental-product-name} que são possíveis graças aos xref:rancher-os-management/architecture/architecture-components.adoc[componentes] emprestados dos projetos {elemental-product-name} Toolkit e {elemental-product-name} Operator.
