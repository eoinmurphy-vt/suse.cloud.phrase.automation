= Variáveis de modelo de armazenamento
:revdate: 2025-08-15
:page-revdate: {revdate}

As informações coletadas nas variáveis do modelo de armazenamento definem os atributos dos discos disponíveis no sistema.

Os discos são rastreados pelo número (0, 1, 2, ...) e pelo nome do dispositivo_(sda_, _vda_, _nvme0n1_, ...).

Por exemplo, se `+${Storage/Disks/2/Name}+` for igual a `+sda+`, o modelo de disco poderá ser recuperado por meio das variáveis `+${Storage/Disks/2/Model}+` e `+${Storage/Disks/sda/Model}+`.

[NOTE]
====
Os discos rastreados incluem também unidades de disco óptico e dispositivos removíveis (por exemplo, pen drives).

== Tipos de unidade dos discos

|===
| Tipo | Descrição

| HDD
| Unidade de disco rígido

| FDD
| Unidade de disquete

| ODD
| Unidade de disco óptico

| SSD
| Unidade de estado sólido

| virtual
| unidade virtual, ou seja, dispositivos de loop

| Desconhecido
| tipo de unidade desconhecido
|===

== Tipos de StorageController dos discos

|===
| Tipo | Descrição

| IDE
| Eletrônica de acionamento integrada

| SCSI
| Interface de sistema de computador de pequeno porte

| NVMe
| Memória não volátil expressa

| MMC
| Controlador multimídia (usado para dispositivos de armazenamento de telefones celulares)

| virtio
| Controlador/driver de armazenamento virtualizado

| loop
| dispositivo de loop

| Desconhecido
| tipo de controlador desconhecido
|===
