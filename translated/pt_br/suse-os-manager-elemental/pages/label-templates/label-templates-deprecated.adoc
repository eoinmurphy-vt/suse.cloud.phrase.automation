= Variáveis obsoletas dos modelos de rótulos
:revdate: 2025-09-08
:page-revdate: {revdate}

As famílias de variáveis *SMBIOS* e *Hardware* foram descontinuadas no Elemental Operator v1.7.0.

Embora essas famílias de variáveis ainda sejam renderizadas corretamente nas versões atuais do Elemental Operator, seu suporte será removido em versões futuras do Elemental Operator.

Se ainda estiver usando essas famílias de variáveis dos modelos de rótulos antigos, mude paraxref:label-templates/label-templates.adoc#_label_templates_variables[as mais recentes disponíveis], se possível.

Para simplificar a adoção das novas variáveis, verifique as tabelas que mapeiam as variáveis obsoletas para as novas variáveis suportadas:

* <<_hardware_labels_to_new_label_templates_variables_table,Etiquetas de hardware para as variáveis dos novos modelos de etiqueta>>
* <<_smbios_labels_to_new_label_templates_variable_table,Rótulos SMBIOS para variáveis de novos modelos de rótulos>>
