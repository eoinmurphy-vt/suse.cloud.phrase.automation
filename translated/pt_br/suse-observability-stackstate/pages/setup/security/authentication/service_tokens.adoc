:description: Observabilidade do SUSE

== Visão geral

Use os xref:use/security/k8s-service-tokens.adoc[tokens de serviço] para se autenticar no SUSE Observability sem ter uma conta de usuário associada. Os tokens de serviço são a forma preferida de autenticação de CI/CD ou ao enviar dados de telemetria de agentes ou do Open Telemetry. Como criar e usar tokens de serviço está descrito na xref:use/security/k8s-service-tokens.adoc[documentação sobre tokens de serviço].
