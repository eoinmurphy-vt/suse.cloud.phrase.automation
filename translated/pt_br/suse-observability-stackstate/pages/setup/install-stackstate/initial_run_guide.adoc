== Nome de usuário e senha padrão

[tabs]
====
Kubernetes::
+
--
O SUSE Observability é configurado por padrão com a seguinte conta de administrador:

* *nome de usuário:* `+admin+`
* *senha:* Definido durante a instalação. Isso é coletado pelo script `+generate_values.sh+` e armazenado no formato hash MD5 em `+values.yaml+`
--

OpenShift::
+
--
O SUSE Observability é configurado por padrão com a seguinte conta de administrador:

* *nome de usuário:* `+admin+`
* *senha:* Definido durante a instalação. Isso é coletado pelo script `+generate_values.sh+` e armazenado no formato hash MD5 em `+values.yaml+`
--
