= Compatibilidade com o Rancher e o Kubernetes

{stackstate-product-name} O 2.6.3 é compatível com clusters Kubernetes gerenciados pelo Rancher que executam o RKE2/Kubernetes.

A matriz abaixo lista as versões compatíveis do Rancher e suas correspondentes versões compatíveis do RKE2/Kubernetes.

|===
| Versão Rancheiro | RKE2 / Kubernetes Versions

| 2.11.x
| v1.30.11+rke2r1

| 2.12.x
| v1.30.11+rke2r1

| 2.13.x
a|
v1.30.11+rke2r1

v1.31.13+rke2r1

v1.32.10+rke2r1

|===

{stackstate-product-name} pode ser instalado em clusters Kubernetes ou OpenShift usando os gráficos Helm fornecidos pela SUSE. A instalação requer o Helm v3.x, e os gráficos são compatíveis com as seguintes plataformas:

* *Kubernetes:* 1,25 a 1,33
* *OpenShift:* 4.14 a 4.19
