== Desativar o rastreamento de um protocolo de rede específico no agente de processo

Para desativar o rastreamento de um protocolo de rede específico no agente de processo, defina a configuração do gráfico do Helm `+processAgent.disabledProtocols+`. O exemplo a seguir desativa os protocolos `+http2+` e `+postgres+`:

----
--set "processAgent.disabledProtocols[0]=http2" \
--set "processAgent.disabledProtocols[1]=postgres"
----

== Desativar o agente de processo

Se o agente de processo não for compatível com seu ambiente, desative-o por meio da configuração do gráfico do Helm `+nodeAgent.containers.processAgent.enabled+`:

----
--set nodeAgent.containers.processAgent.enabled=false
----
