= Perspectiva de métricas
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: Observabilidade do SUSE

O Metrics Perspective (Perspectiva de métricas) mostra as métricas do recurso selecionado.

image::k8s/k8s-metrics-perspective.png[Perspectiva das métricas]

== Gráficos

Os gráficos mostram dados de métricas para os componentes selecionados quase em tempo real - os dados são obtidos a cada 30 segundos. Se um processo for interrompido e nenhum outro dado for recebido, o processo acabará saindo do gráfico, pois os dados se deslocam para a esquerda a cada 30 segundos, no mínimo. Se mais dados chegarem durante o intervalo de 30 segundos, eles serão inseridos em um gráfico.
