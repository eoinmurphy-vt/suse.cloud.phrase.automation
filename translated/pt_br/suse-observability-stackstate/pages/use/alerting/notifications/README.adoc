= Notificações
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: Observabilidade do SUSE

As notificações no SUSE Observability são a maneira de notificar aplicativos de terceiros quando um xref:/use/alerting/k8s-monitors.adoc[monitor] no SUSE Observability é acionado. As notificações se integram a aplicativos de terceiros, como ferramentas de gerenciamento de incidentes (como PagerDuty ou Opsgenie), sistemas de emissão de tíquetes (como JIRA, ServiceNow) ou outras plataformas de colaboração (como Slack). Outras ferramentas podem se referir às notificações como alertas.

As notificações são acionadas por alterações no estado de integridade dos monitores nos componentes do SUSE Observability. Por meio dos canais de notificação configurados, as informações de integridade, componente e monitor são enviadas a esses sistemas externos.
