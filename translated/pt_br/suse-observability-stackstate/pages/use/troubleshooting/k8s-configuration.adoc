= Configuração YAML
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: Observabilidade do SUSE

Se um pod estiver enfrentando problemas como falhas, falhas de inicialização ou configurações incorretas, a inspeção da configuração YAML poderá ajudá-lo a identificar a causa raiz. Você pode ver a configuração YAML clicando no botão *Show configuration (Mostrar configuração)*.

image::k8s/k8s-configuration.png[]

Isso gera a mesma saída que o comando `+kubectl describe+` aplicado ao serviço, pod ou outro recurso.

image::k8s/k8s-configuration-opened.png[]
