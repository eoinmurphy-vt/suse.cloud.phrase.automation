== {product_name} 是什么？

Rancher 的集群 API 扩展（**Cluster API**Extension for**Rancher）**，也称为{product_name} ，是一个 Kubernetes 操作器，提供**Rancher 管理器与集群 API（CAPI）之间的集成**，目的是为 Rancher 提供全面的 CAPI 支持。

通过{product_name} ，您可以

* 通过在**CAPI** 配置集群中安装 Rancher 集群代理，**自动将 CAPI** 集群**导入** Rancher。
* 实施**GitOps 原生**工作流，在 Rancher 中管理 Kubernetes 集群。
* 使用 CAPI 操作员**简化对 CAPI 提供商的管理**。
