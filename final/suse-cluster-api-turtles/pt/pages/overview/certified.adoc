== O que significa Provedores Certificados?

Os provedores certificados são provedores CAPI que são **ativamente testados** pelos fluxos de trabalho de CI da equipe do Turtles para validar se funcionam corretamente com a extensão CAPI para o Rancher. No entanto, como a maioria dos provedores CAPI são projetos upstream mantidos pela comunidade aberta do Kubernetes, **não podemos garantir** que nenhum problema seja introduzido à medida que novas versões do provedor são lançadas.
