== O que é {product_name}?

A extensão da API de cluster para o Rancher, também conhecida como {product_name}, é um operador do Kubernetes que fornece **integração entre o Rancher Manager e a API de cluster (CAPI)** com o objetivo de oferecer suporte completo à CAPI para o Rancher.

Com o {product_name}, você pode:

* **Importar automaticamente clusters CAPI para o Rancher**, instalando o Rancher Cluster Agent em clusters provisionados CAPI.
* Implemente um fluxo de trabalho **nativo do GitOps** para gerenciar clusters do Kubernetes no Rancher.
* **Gerenciamento simplificado de provedores de CAPI** usando o CAPI Operator.
