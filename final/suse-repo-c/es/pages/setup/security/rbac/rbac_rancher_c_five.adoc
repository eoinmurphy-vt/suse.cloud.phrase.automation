== Descripción general

La extensión de observabilidad Prime de SUSE Rancher utiliza RBAC de Kubernetes para conceder acceso a los usuarios de Rancher en SUSE Observability.
Si no utilizas Rancher, consulta xref:/setup/security/rbac/rbac_roles.adoc[Cómo configurar roles] en una instalación independiente.
