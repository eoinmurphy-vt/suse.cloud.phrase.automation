= Referencia a MachineInventorySelector
:revdate: 2024-09-24
:page-revdate: {revdate}

Un MachineInventorySelector selecciona MachineInventories basándose en selectores aplicados (normalmente coincidencia de patrones en valores de etiqueta de MachineInventory).

Los MachineInventorySelectors tienen dos condiciones:

* `InventoryReady`se convierte en true si el MachineInventorySelector ha encontrado un MachineInventory coincidente y se ha establecido con éxito como propietario del MachineInventory.
* `Ready`, rastrea si el selector ya adoptó una máquina e inició el proceso de aprovisionamiento de kubernetes (bootstrap de nodo).
