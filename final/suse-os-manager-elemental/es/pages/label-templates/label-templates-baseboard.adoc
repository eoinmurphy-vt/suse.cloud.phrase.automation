= Variables de la plantilla BaseBoard
:revdate: 2025-08-15
:page-revdate: {revdate}

La información recogida en las variables BaseBoard define atributos de una placa base de sistema (por ejemplo, una placa base, planar, blade de servidor u otro módulo de sistema estándar).

Subconjunto de datos de información de SMBIOS Baseboard (o Módulo) (tipo 2).

Para más información sobre SMBIOS, consulte lahttps://www.dmtf.org/sites/default/files/standards/documents/DSP0134_3.7.1.pdf[especificación].

|===
| Variable | Descripción | de

| `${BaseBoard/AssetTag}`
| Etiqueta de activos de la placa base
| 1.7.0

| `${BaseBoard/Product}`
| Nombre del producto de la placa base
| 1.7.0

| `${BaseBoard/SerialNumber}`
| Número de serie de la placa base
| 1.7.0

| `${BaseBoard/Vendor}`
| Vendedor de placas base
| 1.7.0

| `${BaseBoard/Version}`
| Revisión de la placa base
| 1.7.0
|===
