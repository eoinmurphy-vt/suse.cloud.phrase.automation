= Solución de problemas en las actualizaciones de Rancher
:revdate: 2025-07-03
:page-revdate: {revdate}

[WARNING]
====
La actualización a Rancher v2.7.2 fallará si se definen clusters {elemental-product-name}. El rancher pod se queda atascado en un bucle de bloqueo (ver https://github.com/rancher/rancher/issues/41145).
====

Tenga en cuenta que el problema sólo se presenta si se ha definido al menos un clúster {elemental-product-name}.

Para solucionar el problema, cree un recurso `dynamicschemas.management.cattle.io` vacío denominado `machineinventoryselectortemplate`:

[,shell]
----
kubectl apply -f - <<EOF
kind: DynamicSchema
apiVersion: management.cattle.io/v3
metadata:
  name: machineinventoryselectortemplate
EOF

----

El bucle de bloqueo se detendrá y la actualización de Rancher se completará con éxito.
