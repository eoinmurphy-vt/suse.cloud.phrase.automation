= Variáveis de modelo de memória
:revdate: 2025-08-15
:page-revdate: {revdate}

As informações coletadas nas variáveis do modelo Memory definem os atributos da memória (volátil) disponível no sistema.

|===
| Variável | Descrição | de

| `${Memory/Modules}`
| Módulos de memória
| 1.7.0

| `${Memory/SupportedPageSizes}`
| Tamanhos de página de memória permitidos
| 1.7.0

| `${Memory/TotalPhysicalBytes}`
| Memória total
| 1.7.0

| `${Memory/TotalUsableBytes}`
| Memória total disponível
| 1.7.0
|===
