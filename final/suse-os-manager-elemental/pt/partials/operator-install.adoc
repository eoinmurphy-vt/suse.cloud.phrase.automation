== Instalar {elemental-product-name} Operador
:revdate: 2025-09-08
:page-revdate: {revdate}

`elemental-operator` é o endpoint de gerenciamento, que executa o cluster de gerenciamento e cuida da criação de inventários, registros de máquinas e muito mais.

Usaremos o gerenciador de pacotes Helm para instalar o gráfico elemental-operator em nosso cluster.

[,console]
----
helm upgrade --create-namespace -n cattle-elemental-system --install elemental-operator-crds oci://registry.suse.com/rancher/elemental-operator-crds-chart

helm upgrade --create-namespace -n cattle-elemental-system --install elemental-operator oci://registry.suse.com/rancher/elemental-operator-chart
----

Após alguns segundos, você verá o pod do operador aparecer no namespace `cattle-elemental-system`:

[,console]
----
kubectl get pods -n cattle-elemental-system

NAME                                  READY   STATUS    RESTARTS   AGE
elemental-operator-64f88fc695-b8qhn   1/1     Running   0          16s
----

[IMPORTANT]
.Helm v3.8.0+ necessário
----
O gráfico do operador {elemental-product-name} é distribuído por meio de um registro OCI: O Helm suporta corretamente os registros baseados em OCI a partir da versão v3.8.0.
----


[CAUTION]
.Trocar a ordem de instalação dos gráficos ao atualizar a partir da versão do elemental-operator < 1.2.4
----
Ao atualizar a partir de uma versão do elemental-operator que incorpora os CRDs do {elemental-product-name} (versão < 1.2.4), a instalação do gráfico elemental-operator-crds falhará.
Você precisará atualizar o gráfico elemental-operator primeiro e só depois instalar o gráfico elemental-operator-crds.
----

=== Instalações não estáveis

Além dos gráficos do Helm listados acima, há duas outras versões disponíveis no site `non-stable`.

* *Staging:* refere-se à última versão marcada do Github. Isso está documentado nas xref:quickstarts/quickstart-ui.adoc[próximas] páginas.
* *Desenvolvimento:* refere-se à "ponta do HEAD" do Github. Esta é a versão de desenvolvimento contínuo e muda constantemente.

[tabs]
=====
Versão de teste (x86-64, ARM64 (Raspberry Pi 4))::
+
--
[,console]
----
helm upgrade --create-namespace -n cattle-elemental-system --install elemental-operator-crds oci://registry.opensuse.org/isv/rancher/elemental/staging/charts/rancher/elemental-operator-crds-chart 
helm upgrade --create-namespace -n cattle-elemental-system --install elemental-operator oci://registry.opensuse.org/isv/rancher/elemental/staging/charts/rancher/elemental-operator-chart
----
--

Versão de desenvolvimento (x86-64, ARM64 (Raspberry Pi 4))::
+
--
[WARNING]
.Lembrete
====
A versão de desenvolvimento não é recomendada para ambientes de produção. Agradecemos o feedback por meio de problemas no Slack ou no Github, mas ele pode ser instável e conter recursos experimentais que podem ser descartados sem aviso prévio.
====

[,console]
----
helm upgrade --create-namespace -n cattle-elemental-system --install elemental-operator-crds oci://registry.opensuse.org/isv/rancher/elemental/dev/charts/rancher/elemental-operator-crds-chart
helm upgrade --create-namespace -n cattle-elemental-system --install --set image.imagePullPolicy=Always elemental-operator oci://registry.opensuse.org/isv/rancher/elemental/dev/charts/rancher/elemental-operator-chart
----
--
=====
