== Voraussetzungen
:revdate: 2025-11-06
:page-revdate: {revdate}

* Ein konfigurierter Rancher-Server (v2.7.0 oder höher) (server-url gesetzt)
 ** Um den Rancher `server-url` zu konfigurieren, lesen Sie bitte die https://documentation.suse.com/cloudnative/rancher-manager/latest/en/rancher-admin/rancher-admin.html#_first_log_in[docs] Rancher
* Ein Rechner (Bare Metal oder virtualisiert) mit TPM 2.0
 ** Hinweis 1: Libvirt erlaubt das Setzen von virtuellen TPMs für virtuelle Maschinen xref:operator-operational-tasks/tpm.adoc#_add_tpm_module_to_virtual_machine[Beispiel hier]
 ** Hinweis 2: Sie können die TPM-Emulation auf Bare-Metal-Maschinen aktivieren, wenn Sie das TPM 2. xref:operator-operational-tasks/tpm.adoc#_add_tpm_emulation_to_bare_metal_machine[0-Modulbeispiel hier] vermissen
 ** Hinweis 3: Stellen Sie sicher, dass Sie UEFI (nicht BIOS) auf x86-64 verwenden, sonst bootet das ISO nicht.
 ** Hinweis 4: Es wird eine Mindestgröße des Volumes von 25 GB empfohlen. Weitere Einzelheiten finden Sie in der xref:installation/installation.adoc#_deployed_partition_table[Partitionstabelle{elemental-product-name} ] 
 ** Hinweis 5: Die Anforderungen an CPU und RAM hängen von der installierten Kubernetes-Version ab, zum Beispiel https://documentation.suse.com/cloudnative/k3s/latest/en/installation/requirements.html#_hardware[] K3s oder https://documentation.suse.com/cloudnative/rke2/latest/en/install/requirements.html#_hardware[] RKE2
* Helm-Paketmanager (https://helm.sh/)
* Für ARM (aarch64) - Eine SD-Karte (32 GB oder mehr, muss *schnell* sein - 40MB/s Schreibgeschwindigkeit ist akzeptabel) und ein USB-Stick für die Installation
