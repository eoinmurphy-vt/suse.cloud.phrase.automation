== Bereiten Sie Ihre Kubernetes-Ressourcen vor

Die Knotenbereitstellung beginnt mit einer `MachineRegistration`, die eine Gruppe von Rechnern identifiziert, die dieselbe Konfiguration (Laufwerke, Netzwerk usw.) haben.
