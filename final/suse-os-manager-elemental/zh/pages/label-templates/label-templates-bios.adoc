= BIOS 模板变量
:revdate: 2025-08-15
:page-revdate: {revdate}

BIOS 信息。

SMBIOS BIOS 信息数据（类型 0）的子集。

有关 SMBIOS 的更多信息，请参阅https://www.dmtf.org/sites/default/files/standards/documents/DSP0134_3.7.1.pdf[SMSBIOS 规范]。

|===
| 可变 | 说明 | 从

| `${BIOS/Date}`
| BIOS 发布日期
| 1.7.0

| `${BIOS/Vendor}`
| BIOS 供应商
| 1.7.0

| `${BIOS/Version}`
| BIOS 版本号
| 1.7.0
|===
