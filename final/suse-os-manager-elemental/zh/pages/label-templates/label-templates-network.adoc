= 网络模板变量
:revdate: 2025-08-15
:page-revdate: {revdate}

网络模板变量中收集的信息定义了系统中可用网络接口的属性。

网卡可按编号（0、1、2、... ）和接口名称_（eth0_、_virbr0_、... ）跟踪。

IPv4Addresses 和 IPv6Addresses 按编号（0、1、... ）跟踪，第一个 IPv4 和 IPv6 地址条目（即 IPv4Addresses.0 和 IPv6Addresses.0）在 IPv4Address 和 IPv6Address 变量中跟踪。

例如，如果`${Network/NICs/2/Name}` 等于`eth0` ，则可通过`${Network/NICs/2/MacAddress}` 和`${Network/NICs/eth0/MacAddress}` 变量检索网络接口 Mac 地址。

`eth0` (第一）IPv4 地址可通过`${Network/NICs/eth0/IPv4Address}` 和`${Network/NICs/eth0/IPv4Addresses/0}` 变量检索。`eth0` IPv4 二级地址可以用`${Network/NICs/eth0/IPv4Addresses/1}` 代替。
