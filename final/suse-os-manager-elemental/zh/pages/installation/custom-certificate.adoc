= 如何添加自定义证书
:revdate: 2024-09-24
:page-revdate: {revdate}

前提条件`.pem` 格式的证书

目标在全系统范围内提供自定义证书

[NOTE]
.用于系统级服务使用的证书。
====
Kubernetes 工作负载应在容器镜像中携带证书。
====

要安装自定义证书，我们需要

* 将`.pem` 文件复制到 `/etc/pki/trust/anchors/`
* 运行 `update-ca-certificates`

相应的`cloud-config` 代码段如下所示：

[,yaml]
----
write_files:
  - path: /etc/pki/trust/anchors/my-custom-certificate.pem
    permission: 0444
    content: |-
      -----BEGIN CERTIFICATE-----
      ...
      -----END CERTIFICATE-----
runcmd:
  - update-ca-certificates
----

(为简洁起见，实际证书内容略去不提）
