= Dépannage des mises à jour de Rancher
:revdate: 2025-07-03
:page-revdate: {revdate}

[WARNING]
====
La mise à jour vers Rancher v2.7.2 échouera si des clusters {elemental-product-name} sont définis. Le pod rancher est bloqué dans une boucle de crash (voir https://github.com/rancher/rancher/issues/41145).
====

Notez que le problème n'est présent que si au moins un cluster {elemental-product-name} est défini.

Pour contourner le problème, créez une ressource `dynamicschemas.management.cattle.io` vide nommée `machineinventoryselectortemplate`:

[,shell]
----
kubectl apply -f - <<EOF
kind: DynamicSchema
apiVersion: management.cattle.io/v3
metadata:
  name: machineinventoryselectortemplate
EOF

----

La boucle de crash s'arrêtera et la mise à jour de Rancher s'achèvera avec succès.
