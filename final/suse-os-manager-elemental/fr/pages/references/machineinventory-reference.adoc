= Inventaire des machines
:revdate: 2025-07-03
:page-revdate: {revdate}

Lorsqu'un nouvel hôte s'enregistre avec succès, le site {elemental_operator_name} crée une ressource MachineInventory représentant cet hôte particulier.
Le MachineInventory stocke le hash TPM de l'hôte suivi, récupéré au cours du processus d'enregistrement, et permet d'exécuter des commandes arbitraires (plans) sur la machine.

Un inventaire de machines est soumis à deux conditions :

* `AdoptionReady`qui indique que la machine a été adoptée par un sélectionneur pour faire partie d'un groupe.
* `Ready`qui indique que la machine a été enregistrée et approvisionnée avec un système d'exploitation {elemental-product-name}.
