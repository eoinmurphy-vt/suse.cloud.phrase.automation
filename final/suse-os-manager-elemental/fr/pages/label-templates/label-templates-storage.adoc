= Variables du modèle de stockage
:revdate: 2025-08-15
:page-revdate: {revdate}

Les informations collectées dans les variables du modèle de stockage définissent les attributs des disques disponibles dans le système.

Les disques sont suivis à la fois par numéro (0, 1, 2, ...) et par nom de périphérique_(sda_, _vda_, _nvme0n1_, ...).

Par exemple, si `${Storage/Disks/2/Name}` est égal à `sda`, le modèle de disque peut être récupéré via les variables `${Storage/Disks/2/Model}` et `${Storage/Disks/sda/Model}`.

[NOTE]
====
Les disques suivis comprennent également les lecteurs de disques optiques et les dispositifs amovibles (par exemple, les clés USB).
====

== Types de disques

|===
| Type | Description

| HDD
| Disque dur

| FDD
| Lecteur de disquettes

| ODD
| Lecteur de disque optique

| SSD
| Lecteur à semi-conducteurs

| virtual
| lecteur virtuel, c'est-à-dire dispositifs en boucle

| Inconnu
| type de lecteur inconnu
|===

== Types de contrôleurs de stockage des disques

|===
| Type | Description

| IDE
| Électronique d'entraînement intégrée

| SCSI
| Interface de petit système informatique

| NVMe
| Mémoire non volatile Express

| MMC
| Contrôleur multimédia (utilisé pour les dispositifs de stockage des téléphones portables)

| virtio
| Contrôleur/pilote de stockage virtualisé

| boucle
| dispositif en boucle

| Inconnu
| type de contrôleur inconnu
|===
