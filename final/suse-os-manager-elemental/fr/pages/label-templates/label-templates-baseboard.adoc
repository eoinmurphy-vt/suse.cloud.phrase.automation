= Variables du modèle BaseBoard
:revdate: 2025-08-15
:page-revdate: {revdate}

Les informations collectées dans les variables BaseBoard définissent les attributs d'une carte mère (par exemple, une carte mère, une carte planaire, une lame de serveur ou un autre module système standard).

Sous-ensemble de données d'information de la carte de base (ou du module) SMBIOS (type 2).

Pour plus d'informations sur SMBIOS, voir lahttps://www.dmtf.org/sites/default/files/standards/documents/DSP0134_3.7.1.pdf[spécification SMSBIOS sur le site].

|===
| Variable | Description | de

| `${BaseBoard/AssetTag}`
| Étiquette de la carte mère
| 1.7.0

| `${BaseBoard/Product}`
| Nom du produit de la carte mère
| 1.7.0

| `${BaseBoard/SerialNumber}`
| Numéro de série de la carte mère
| 1.7.0

| `${BaseBoard/Vendor}`
| Fournisseur de la carte mère
| 1.7.0

| `${BaseBoard/Version}`
| Révision de la carte mère
| 1.7.0
|===
