= Variables du modèle de mémoire
:revdate: 2025-08-15
:page-revdate: {revdate}

Les informations collectées dans les variables du modèle de mémoire définissent les attributs de la mémoire (volatile) disponible sur le système.

|===
| Variable | Description | de

| `${Memory/Modules}`
| Modules de mémoire
| 1.7.0

| `${Memory/SupportedPageSizes}`
| Tailles de page de mémoire autorisées
| 1.7.0

| `${Memory/TotalPhysicalBytes}`
| Mémoire totale
| 1.7.0

| `${Memory/TotalUsableBytes}`
| Mémoire totale disponible
| 1.7.0
|===
