= ラベルテンプレートの非推奨変数
:revdate: 2025-09-08
:page-revdate: {revdate}

*SMBIOS*および*ハードウェア*変数ファミリは、Elemental Operator v1.7.0で非推奨となりました。

これらの変数ファミリは、現在のElemental Operatorバージョンでも正しくレンダリングされますが、将来のElemental Operatorリリースではサポートが削除されます。

古いラベルテンプレートの変数ファミリーを使用している場合は、可能であればxref:label-templates/label-templates.adoc#_label_templates_variables[新しい]ものに変更してください。

新しい変数の採用を簡単にするために、非推奨の変数と新しくサポートされる変数の対応表を確認してください：

* <<_hardware_labels_to_new_label_templates_variables_table,新しいラベルテンプレートの変数にハードウェアラベルを追加する>>
* <<_smbios_labels_to_new_label_templates_variable_table,SMBIOSラベルを新しいラベルテンプレートの変数に追加する>>

== ハードウェア・ラベルを新しいラベル・テンプレートの変数テーブルに追加する。
