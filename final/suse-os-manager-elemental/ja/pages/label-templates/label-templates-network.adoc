= ネットワークテンプレート変数
:revdate: 2025-08-15
:page-revdate: {revdate}

ネットワーク・テンプレート変数で収集された情報は、システムで利用可能なネットワーク・インターフェースの属性を定義します。

NICは、番号（0、1、2、... ）とインターフェース名_（eth0_、_virbr0_、... ）の両方で追跡される。

IPv4AddressesとIPv6Addressesは番号（0、1、... ）で追跡され、最初のIPv4アドレスエントリとIPv6アドレスエントリ（すなわち、IPv4Addresses.0とIPv6Addresses.0）は、IPv4Address変数とIPv6Address変数で追跡される。

例えば、`${Network/NICs/2/Name}` が`eth0` と等しい場合、`${Network/NICs/2/MacAddress}` と`${Network/NICs/eth0/MacAddress}` の両方の変数を使ってネットワーク・インターフェイスのマック・アドレスを取得することができます。

`eth0` (最初の) IPv4アドレスは、`${Network/NICs/eth0/IPv4Address}` と`${Network/NICs/eth0/IPv4Addresses/0}` 変数の両方で取得できる。セカンダリ`eth0` IPv4アドレスは、代わりに`${Network/NICs/eth0/IPv4Addresses/1}` 。
