= {elemental-product-name} オペレーター・ヘルムチャート
:revdate: 2025-07-03
:page-revdate: {revdate}

{elemental_operator_url}[{elemental_operator_name}]は、{elemental-product-name} バージョンを管理し、エッジまたはベアメタルのインストールを支援するマシンインベントリを維持する責任を負う。

関連するチャートは、https://helm.sh[Helm]パッケージマネージャを使用して、https://rancher.com/docs/rancher/v2.6/[]Rancherクラスタ上のelemental-operatorデプロイメントをブートストラップする。

== 前提条件

* Rancher Manager バージョン v2.6
* Helmクライアントバージョンv3.8.0以上

== ヘルムチャート情報

[,console]
----
helm pull oci://registry.suse.com/rancher/elemental-operator-chart
helm show all oci://registry.suse.com/rancher/elemental-operator-chart
----

== インストールチャート

[,console]
----
helm install --create-namespace -n cattle-elemental-system elemental-operator-crds \
             oci://registry.suse.com/rancher/elemental-operator-crds-chart
helm install --create-namespace -n cattle-elemental-system elemental-operator \
             oci://registry.suse.com/rancher/elemental-operator-chart
----

このコマンドは、Kubernetesクラスタにelemental-operatorをデフォルト設定でデプロイする。

_以下の<<_configuration,構成を>>参照のこと。_

_コマンドについてはhttps://helm.sh/docs/helm/helm_install/[helm install]を参照。_

== アンインストールチャート

[,console]
----
helm uninstall -n cattle-elemental-system elemental-operator
----

これにより、チャートに関連するすべてのKubernetesコンポーネントが削除され、リリースが削除されます。

_コマンドについてはhttps://helm.sh/docs/helm/helm_uninstall/[helm uninstall]を参照。_

== アップグレードチャート

[,console]
----
helm upgrade -n cattle-elemental-system \
             --install elemental-operator \
             oci://registry.suse.com/rancher/elemental-operator-chart
----

_コマンドについてはhttps://helm.sh/docs/helm/helm_upgrade/[helm upgrade]を参照してください。_
