== 制定政策

到目前为止，您已经使用与开发机器相同的操作系统和架构编译目标构建了策略。

现在要将策略构建为 WebAssembly 二进制文件，即`.wasm` 文件。

使用命令

[subs="+attributes",console]
----
make policy.wasm
----

该命令在发布模式下使用 WebAssembly 编译目标构建代码。

构建过程会生成一个`policy.wasm` 文件：

[subs="+attributes",console]
----
$ file policy.wasm
policy.wasm: WebAssembly (wasm) binary module version 0x1 (MVP)
----
