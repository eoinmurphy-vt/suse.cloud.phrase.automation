目前，.NET Core 对 WebAssembly WASI 平台提供试验性支持。
这是从https://github.com/SteveSandersonMS/dotnet-wasi-sdk[`dotnet-wasi-sdk`]项目。

[NOTE]
====

编写或运行 .NET Core 代码不需要安装 Windows。
所有操作都可以在 Linux 或 MacOS 机器上完成。
====


== 当前状态

政策作者可以使用以下资源：

* https://github.com/kubewarden/policy-sdk-dotnet[{project-name}.NET Core SDK]：它提供了一套对象和函数，可简化编写策略的过程。
* https://github.com/kubewarden/policy-sdk-dotnet/tree/main/example[{short-project-name} 政策范例]：这是一个工作政策范例。

没有已知的限制。
SDK 支持编写验证策略和变异策略。

可以使用https://www.nuget.org/packages/KubernetesClient.Models[`KubernetesClient.Models`]库来处理 Kubernetes 对象。
