== OpenTelemetry、度量和跟踪依赖项

在撰写本报告时，https://opentelemetry.io[OpenTelemetry]协议栈正在不断改进。不过，它还不稳定，未宣布的、向后不兼容的更改仍时有发生。{short-project-name} 开发人员努力跟踪 OpenTelemetry 协议栈的变化，并做出相应调整。该项目针对已知工作范围的 OpenTelemetry、度量和跟踪堆栈进行测试。
