:revdate: 2026-03-18
:page-revdate: {revdate}

:current-version: {page-origin-branch}

{project-name} 实现对https://www.sigstore.dev/[] Sigstore 项目的支持。
这样就可以为您的集群实施 "安全供应链"。

安全供应链的部分功能是确保集群中运行的所有容器镜像都经过签名和验证。
这就证明了它们来自于它们所声明的作者，没有经过任何篡改。
如需进一步阅读，请查看有关xref:/howtos/security-hardening/secure-supply-chain.adoc[我们如何为政策本身实施安全供应链的]文档。

Sigstore 签名存储在容器注册表中，就在被签名的 OCI 对象旁边。
它们可以是容器映像，也可以是更通用的 OCI 工具，如{short-project-name} 策略。
对象签名后，其签名将作为 Sigstore 创建的 OCI 对象的层存储。
需要检查容器签名的策略需要检查这些层，并需要提取签名层来检查签名本身。

获取和操作这些 OCI 层需要在 WebAssembly 客户（策略）之外进行。
因此，这项工作由 WebAssembly 运行时{short-project-name} 的`policy-server` 或`kwctl` 完成。

{short-project-name} 策略的不同语言 SDK 可以解决这个问题。
它们提供验证容器映像、{short-project-name} 策略、Helm 图表和其他类型 OCI 工件的功能。
这确保了验证的安全和经过测试的代码路径。

从注册表中提取数据并对签名进行加密验证可能会耗费大量时间和计算量，因此 WebAssembly 运行时（PolicyServer，`kwctl` ）会确保对签名提取和验证计算进行缓存。
缓存条目会在 60 秒后自动过期，以防止提供过期数据。

== 缓存

签名验证结果缓存一分钟。

SDK 提供以下类似功能：

* {blank}
+
[subs="+attributes",rust]
----
verify_pub_keys_image(
    image_url: string,
    vector_of_pub_keys: vector<string>,
    vector_of_sigstore_annotations: Vector<(key, value: string)>
    )
    returns (is_trusted: bool, digest_of_verified_image: string)
----

* {blank}
+
[subs="+attributes",rust]
----
verify_keyless_exact_match(
    image_url: string,
    vector_of_tuples_issuer_and_subject: vector<(issuer, subject: string)>,
    vector_of_sigstore_annotations: vector<(key, value: string)>
    )
    returns (is_trusted: bool, digest_of_verified_image: string)
----

这两个函数都验证图像已签名并满足约束条件。

[NOTE]
====
成功后，函数会返回已验证图像的摘要。
现在，策略有责任确保容器是根据该摘要实例化的，而不是根据可能与校验和摘要不匹配（从而受到损害）的标签实例化的。
====


== 举例说明

{short-project-name} 团队https://github.com/kubewarden/verify-image-signatures[提供了一个验证器策略]，可对所有容器执行 Sigstore 签名。
它使用 Rust 和 Rust SDK 构建。
该策略可确保容器已签名，并可选择对请求进行更改。突变会将经过验证的校验和添加到图像的标签中。
具体信息请查看文档。

这项政策可以满足您的所有需求，但如果您想要不同的用户体验，当然也可以在它或任何其他 SDK 的基础上进行构建。

== WaPC 协议合同

如果您正在实施自己的语言 SDK，这些是验证签名的策略可以使用的功能：

=== waPC 功能 - `v2/verify`

==== `SigstorePubKeyVerify` 输入

[,hcl]
----
{
  type: "SigstorePubKeyVerify",

  # **mandatory**: image URI to verify
  "image": string,
  "pub_keys": [
    # PEM-encoded public keys
    string
    ],
  # optional:
  "annotations": [
      # signature annotations
      {
        "key": string,
        "value": string
      },
    ]
}
----

==== `SigstorePubKeyVerify` 产量

[,hcl]
----
{
   # true if image verified
   "is_trusted": boolean,
   # digest of verified image
   "digest": string
}
----

==== `SigstoreKeylessVerify` 输入

[,hcl]
----
{
  type: "SigstoreKeylessVerify",

  # mandatory: image URI to verify
  "image": string,
  "keyless": [
    # list of (issuer, subject) tuples
    {
      # OIDC issuer
      "issuer": string,
      # signature subject (mail, CI URL, ...)
      "subject": string
    }
  ],
  # optional:
  "annotations": [
    # signature annotations
    {
      "key": string,
      "value": string
    },
  ]
}
----

==== `SigstoreKeylessVerify` 产量

}

==== `SigstoreCertificateVerify` 输入

[,hcl]
----
{
  type: "SigstoreCertificateVerify",

  # mandatory: image URI to verify
  "image": string,
  # PEM-encoded certificated used to
  # verify the signature.
  # The certificate is UTF-8 encoded.
  # It's an array of bytes of the unicode code pointers of a PEM encoded
  # certificate string.
  "certificate": [byte(int), ..., byte(int)],
  # Optional - certificate chain used to
  # verify the provided certificate.
  # When not specified, the certificate
  # is assumed to be trusted.
  # The certificate is UTF-8 encoded.
  # It's an array of bytes of the unicode code pointers of a PEM encoded
  # certificate string.
  "certificate_chain": [
    [byte(int), ..., byte(int)],
    ...
    [byte(int), ..., byte(int)]
  ],
  # Require the signature layer to have
  # a Rekor bundle.
  # Having a Rekor bundle allows further
  # checks to be performed, e.g. ensuring
  # the signature has been produced during
  # the validity time frame of the cert.
  # Recommended to set to `true`
  require_rekor_bundle: bool,
  # Optional:
  "annotations": [
    # signature annotations
    {
      "key": string,
      "value": string
    },
  ]
}
----

==== `SigstoreCertificateVerify` 产量

[,hcl]
----
{
   # true if image verified
   "is_trusted": boolean,
   # digest of verified image
   "digest": string
}
----
