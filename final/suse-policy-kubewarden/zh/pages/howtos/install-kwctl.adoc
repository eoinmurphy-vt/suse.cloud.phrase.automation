= 安装 kwctl ({project-name} CLI)
:revdate: 2026-03-18
:page-revdate: {revdate}
:title: 安装 kwctl
:description: 按照不同操作系统的安装说明安装 kwctl（{project-name} CLI），然后启用 shell 完成。
:keywords: ["{project-name}", kubewarden, kubernetes, install kwctl, install, kwctl]
:doc-persona: kubewarden-all
:doc-type: 如何
:doc-topic: install-kwctl

`kwctl` 是{project-name} 的命令行界面（CLI）工具。以下是某些操作系统的安装说明。

[tabs]
====
Linux::
+
--
为 Linux 安装

. 自制啤酒
+
如果您在 Linux 上使用 Homebrew 软件包管理器，那么
+
[source,console]
----
brew install kwctl
----
+
验证安装：
+
[source,console]
----
kwctl --version
----

. Zypper (openSUSE)
+
[source,console]
----
zypper install kwctl
----

. AUR
+
如果你使用的是 Arch Linux 或基于 Arch 的发行版，可以从 AUR 安装 kwctl。
+
.. 使用 AUR 助手（耶）：
+
[source,console]
----
yay -S kwctl
----
+
.. 使用 makepkg：
+
[source,console]
----
# Clone the AUR package
git clone https://aur.archlinux.org/kwctl.git
cd kwctl

# Build and install the package
makepkg -si
----
+
.. 验证安装：
+
[source,console]
----
kwctl --version
----

. 手动安装
+
.. 下载最新版本的`kwctl` for Linux：
+
[source,console]
----
curl -LO https://github.com/kubewarden/kwctl/releases/latest/download/kwctl-linux-x86_64.zip
----
+
对于 ARM64 系统（如 Raspberry Pi），请使用：
+
[source,console]
----
curl -LO https://github.com/kubewarden/kwctl/releases/latest/download/kwctl-linux-aarch64.zip
----
+
.. 从下载的`.zip` 文件中提取文件：
+
[source,console]
----
unzip kwctl-linux-x86_64.zip
----
+
这将提取以下文件：
+
* `kwctl-linux-x86_64`:`kwctl` 二进制
* `kwctl-linux-x86_64.sig`:用于验证二进制文件的签名文件
* `kwctl-linux-x86_64.pem`:用于验证签名的证书文件
+
.. 将二进制文件移至 PATH 目录中，重命名为`kwctl` 并使其可执行。
+
.. 验证安装：
+
[source,console]
----
kwctl --version
----

Windows::
+
--
为 Windows 安装

. 下载`kwctl` ：
    .. 打开浏览器，进入https://github.com/kubewarden/kwctl/releases/latest[{short-project-name} 发布页面]。
    .. 下载`kwctl-windows-x86_64.zip` 文件。

. 从下载的压缩文件中提取文件。它将包括
+
* `kwctl-windows-x86_64.exe`：`kwctl` 二进制。
* `kwctl-windows-x86_64.sig`：用于验证二进制文件的签名文件。
* `kwctl-windows-x86_64.pem`：用于验证签名的证书文件。

. 将二进制文件从`kwctl-windows-x86_64.exe` 重命名为`kwctl.exe` ，以方便使用。

. 将二进制文件移至`PATH` 环境变量覆盖的位置。

. 验证安装。打开一个新的命令提示符或 PowerShell 窗口，检查`kwctl` 的安装情况：
+
[source,console]
----
kwctl --version
----

== 安装外壳

`kwctl` CLI 有`--shell` 选项，可为 shell 生成 shell 完成命令。您可以使用该命令的输出将补全功能整合到 shell 中。

[source,console]
----
kwctl completions --shell [bash|elvish|fish|powershell|zsh]
----
