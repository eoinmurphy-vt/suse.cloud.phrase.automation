您可以通过以下方式为{short-project-name} 文档做出贡献：

* 改进现有内容
* 创建新内容
* _(Upcoming_) 帮助翻译文件

== 开始

ifeval::["{build-type}" == "community"]

欢迎大家提交拉取请求，提出修改建议。您还可以通过 GitHub 问题向https://github.com/kubewarden/docs[`kubewarden/docs` 仓库]报告错误或拼写错误。

参与文献工作：

* 熟悉https://github.com/kubewarden/docs[文档库]和https://docusaurus.io/[静态网站生成器]。
* Docusaurus 使用 Markdown 语法，并使用https://mdxjs.com/[MDX]作为解析器。
* 了解如何xref:howtos/contribution-guide/suggesting-an-improvement.adoc[提出改进建议]并开启拉取请求。

endif::[]

ifeval::["{build-type}" == "product"]

欢迎大家提交拉取请求，提出修改建议。您还可以通过 GitHub 问题向https://github.com/rancher/kubewarden-product-docs[`rancher/kubewarden-product-docs` 仓库]报告错误或拼写错误。

参与文献工作：

* 熟悉https://github.com/rancher/kubewarden-product-docs[文档库]和https://antora.org/[静态网站生成器]。
* Antora 使用 AsciiDoc 语法。
* 了解如何xref:howtos/contribution-guide/suggesting-an-improvement.adoc[提出改进建议]并开启拉取请求。

endif::[]
