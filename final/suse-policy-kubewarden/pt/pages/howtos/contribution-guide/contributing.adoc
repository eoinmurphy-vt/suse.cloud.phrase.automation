_{short-project-name} aceita sugestões de aprimoramento de todos os colaboradores, sejam eles novos ou experientes._

Você pode contribuir com a documentação do {short-project-name}:

* Melhorar o conteúdo existente
* Criação de novos conteúdos
* _(A ser lançado_) Ajudar na tradução da documentação

== Primeiros passos

ifeval::["{build-type}" == "community"]

Todos são bem-vindos para sugerir alterações enviando uma solicitação pull. Você também pode relatar um bug ou erro de digitação com a ajuda de um problema no GitHub para ohttps://github.com/kubewarden/docs[repositório]

Para se envolver com a documentação:

* Familiarize-se com o https://github.com/kubewarden/docs[repositório de documentação] e com o https://docusaurus.io/[].
* O Docusaurus utiliza a sintaxe do Markdown e usa o https://mdxjs.com/[MDX]como analisador.
* Entenda como xref:howtos/contribution-guide/suggesting-an-improvement.adoc[sugerir uma melhoria] e abrir um pull request.

endif::[]

ifeval::["{build-type}" == "product"]

Todos são bem-vindos para sugerir alterações enviando uma solicitação pull. Você também pode relatar um bug ou erro de digitação com a ajuda de um problema no GitHub para ohttps://github.com/rancher/kubewarden-product-docs[repositório]

Para se envolver com a documentação:

* Familiarize-se com o https://github.com/rancher/kubewarden-product-docs[repositório de documentação] e com o https://antora.org/[].
* O Antora usa a sintaxe do AsciiDoc.
* Entenda como xref:howtos/contribution-guide/suggesting-an-improvement.adoc[sugerir uma melhoria] e abrir um pull request.

endif::[]
