= Instalação do kwctl ({project-name} CLI)
:revdate: 2026-03-18
:page-revdate: {revdate}
:title: Instalação do kwctl
:description: Instale o kwctl (a CLI do {project-name} ) seguindo as instruções de instalação para vários sistemas operacionais e, em seguida, ative a conclusão do shell.
:keywords: ["{project-name}", kubewarden, kubernetes, install kwctl, install, kwctl]
:doc-persona: kubewarden-all
:doc-type: howto
:doc-topic: install-kwctl

`kwctl` é a ferramenta de interface de linha de comando (CLI) para {project-name}. Abaixo estão as instruções de instalação para alguns sistemas operacionais.

[tabs]
====
Linux::
+
--
Instalar para Linux

. Homebrew
+
Se você usa o gerenciador de pacotes Homebrew no Linux, então:
+
[source,console]
----
brew install kwctl
----
+
Verifique a instalação:
+
[source,console]
----
kwctl --version
----

. Zypper (openSUSE)
+
[source,console]
----
zypper install kwctl
----

. AUR
+
Se estiver usando o Arch Linux ou uma distribuição baseada no Arch, você poderá instalar o kwctl a partir do AUR.
+
.. Usando um ajudante AUR (yay):
+
[source,console]
----
yay -S kwctl
----
+
.. Usando o makepkg:
+
[source,console]
----
# Clone the AUR package
git clone https://aur.archlinux.org/kwctl.git
cd kwctl

# Build and install the package
makepkg -si
----
+
.. Verifique a instalação:
+
[source,console]
----
kwctl --version
----

. Instalação manual
+
.. Faça o download da versão mais recente do `kwctl` para Linux:
+
[source,console]
----
curl -LO https://github.com/kubewarden/kwctl/releases/latest/download/kwctl-linux-x86_64.zip
----
+
Para sistemas ARM64 (por exemplo, Raspberry Pi), use:
+
[source,console]
----
curl -LO https://github.com/kubewarden/kwctl/releases/latest/download/kwctl-linux-aarch64.zip
----
+
.. Extraia os arquivos do arquivo `.zip` baixado:
+
[source,console]
----
unzip kwctl-linux-x86_64.zip
----
+
Isso extrai os seguintes arquivos:
+
* `kwctl-linux-x86_64`: O binário `kwctl` 
* `kwctl-linux-x86_64.sig`: Um arquivo de assinatura para verificar o binário
* `kwctl-linux-x86_64.pem`: Um arquivo de certificado para verificar a assinatura
+
.. Mova o binário para um diretório em seu PATH, renomeie para `kwctl` e torne-o executável.
+
.. Verifique a instalação:
+
[source,console]
----
kwctl --version
----

Windows::
+
--
Instalar para Windows

. Download `kwctl`:
    .. Abra seu navegador e acesse https://github.com/kubewarden/kwctl/releases/latest[a página de versões]
    .. Faça o download do arquivo `kwctl-windows-x86_64.zip`.

. Extraia os arquivos do arquivo zip baixado. Ele conterá:
+
* `kwctl-windows-x86_64.exe`: o binário `kwctl`.
* `kwctl-windows-x86_64.sig`: um arquivo de assinatura para verificar o binário.
* `kwctl-windows-x86_64.pem`: um arquivo de certificado para verificar a assinatura.

. Renomeie o arquivo binário de `kwctl-windows-x86_64.exe` para `kwctl.exe` para facilitar o uso.

. Mova o binário para um local coberto por sua variável de ambiente `PATH`.

. Verifique a instalação. Abra um novo prompt de comando ou uma janela do PowerShell e verifique a instalação do `kwctl`:
+
[source,console]
----
kwctl --version
----

== Instalar complementos de shell

A CLI do `kwctl` tem a opção `--shell` para gerar comandos de conclusão de shell para seu shell. Você pode usar a saída desse comando para integrar as conclusões em seu shell.

[source,console]
----
kwctl completions --shell [bash|elvish|fish|powershell|zsh]
----
