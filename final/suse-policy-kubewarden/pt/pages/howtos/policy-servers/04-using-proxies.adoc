Os PolicyServers respeitam as variáveis de ambiente padrão para lidar com configurações de proxy. O servidor de políticas usa o proxy configurado ao fazer download de políticas.
Isso inclui operações Sigstore ao verificar políticas e imagens.

Essas variáveis são:

* `HTTP_PROXY` ou `http_proxy`: servidor proxy para solicitações HTTP
* `HTTPS_PROXY` ou `https_proxy`: servidor proxy para solicitações HTTPS
* `NO_PROXY` ou `no_proxy`: lista de hosts separados por vírgula a serem excluídos do proxy

Consulte xref:reference/CRDs.adoc#_policyserverspec[`PolicyServer.spec.env`]na referência do CRD para definir essas variáveis de ambiente.
