== Servidor de políticas

O componente Policy Server inicializa e executa políticas.
Ao receber solicitações do servidor da API do Kubernetes, ele encaminha a solicitação para a política, retornando a resposta fornecida pela política ao servidor da API do Kubernetes.

==== `kubewarden_policy_evaluations_total`

===== Bagagem

|===
| Rótulo | Descrição

| `policy_name`
| Nome da apólice

| `resource_name`
| Nome do recurso avaliado

| `resource_kind`
| Tipo do recurso avaliado

| `resource_namespace`
| Namespace do recurso avaliado. Não estará presente se o recurso tiver escopo de cluster.

| `resource_request_operation`
| Tipo de operação: `CREATE`, `UPDATE`, `DELETE`, `PATCH`, `WATCH`...

| `accepted`
| A solicitação foi aceita?

| `mutated`
| A solicitação sofreu alguma mutação?

| `error_code`
| Código de erro retornado pela política se for rejeitada, se houver.
  Não está presente se a apólice não fornecer um.
|===
