{project-name} As políticas não podem fazer solicitações de rede no contexto de execução do WebAssembly.

As operações de rede podem ser feitas aproveitando uma série de recursos expostos pelo host.

== Pesquisa de host DNS

Essa função executa uma pesquisa de DNS a partir do FQDN fornecido pela política.

=== Armazenamento em cache

Os resultados da pesquisa são armazenados em cache por 1 minuto.

=== Protocolo de comunicação

Esta é a descrição do protocolo waPC usado para expor esse recurso:

==== Função waPC - `v1/dns_lookup_host` input

[,hcl]
----
# hostname - JSON encoded string
string
----

==== Função waPC - `v1/dns_lookup_host` output

[,hcl]
----

{
  # list of IPs
  "ips": [string]
}
----

Todos os endereços IP associados ao FQDN fornecido serão retornados como cadeias de caracteres na resposta.
As entradas IPv4 e IPv6 são retornadas como parte da mesma resposta.

Por exemplo, ao solicitar o resumo do manifesto da imagem`busybox:latest`, os payloads seriam:

* Carga útil de entrada: `"google.com"`
* Carga útil de saída: `{ "ips": ["127.0.0.1"]}`
