https://www.rust-lang.org/[Rustは]、WebAssemblyモジュールを生成できる最も成熟したプログラミング言語です。
WebAssemblyはRustの世界では一級市民なので、Rustエコシステムのツールやクレートの多くは、箱から出してすぐに動作する。

{project-name} https://crates.io/crates/kubewarden-policy-sdk[]Rust提供し、ポリシーを書くプロセスを簡素化する。
また、https://github.com/kubewarden/rust-policy-template[のテンプレート・プロジェクトも]あり、このユーティリティを使ったポリシー・プロジェクトの足場となる。https://github.com/cargo-generate/cargo-generate[`cargo-generate`]ユーティリティを使用する。

この文書では、これらのプロジェクトを使用して、Rustを使用して{short-project-name} ポリシーを記述する方法を示します。
{short-project-name} のRust SDKの詳細はカバーしていない。
詳細はhttps://docs.rs/kubewarden-policy-sdk/0.1.0[クレートドキュメントに]記載されている。

== Rustの依存関係を取得する

https://github.com/rust-lang/rustup[rustupを使って]Rustコンパイラとそのツールをインストールする。
rustuphttps://rust-lang.github.io/rustup/installation/index.html[インストールドキュメントを]参照。

`rustup` をインストールしたら、WebAssembly System Interface (WASI) ターゲットを追加します：

[subs="+attributes",console]
----
rustup target add wasm32-wasip1
----

== OSXの依存関係

`cargo-generate` を使用するには、Xcodeツールセットを追加する必要があります。
Xcodeを通してインストールされていない場合は、以下のコマンドで必要な依存関係を得ることができる：

[subs="+attributes",console]
----
xcode-select --install
----
