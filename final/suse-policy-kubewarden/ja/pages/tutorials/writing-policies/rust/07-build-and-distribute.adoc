== 政策の構築

ここまでは、開発マシンと同じオペレーティング・システムとアーキテクチャのコンパイル・ターゲットでポリシーを構築してきた。

今度は、ポリシーをWebAssemblyバイナリ、`.wasm` ファイルとしてビルドする番だ。

コマンドを使う：

[subs="+attributes",console]
----
make policy.wasm
----

このコマンドは、WebAssemblyコンパイル・ターゲットを使用して、リリース・モードでコードをビルドします。

ビルドは`policy.wasm` ：

[subs="+attributes",console]
----
$ file policy.wasm
policy.wasm: WebAssembly (wasm) binary module version 0x1 (MVP)
----
