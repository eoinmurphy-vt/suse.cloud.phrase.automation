== バイナリ

== OCIアーティファクト

Open Container Initiative (OCI)の成果物については、プロジェクトは`ghcr.io` 。 https://github.com/orgs/kubewarden/packages.

- https://github.com/orgs/kubewarden/packages/policies[`kubewarden/packages/policies/*`]
  はポリシー・ワズムOCIの成果物である。
- https://github.com/orgs/kubewarden/packages/charts[`kubewarden/packages/charts/*`]
  はOCIの成果物としてのヘルムチャートである。SLSAの基準に従っている。
- OCI イメージとして公開されているコンテナイメージ：`kubewarden-controller` `policy-server`,`audit-scanner`,`kubectl`.
- e2eテストで使用された成果物を含む`packages/tests/*` 。

== HTTPS 経由のヘルムチャート
