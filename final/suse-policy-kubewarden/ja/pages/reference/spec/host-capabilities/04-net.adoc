{project-name} ポリシーは、WebAssembly実行コンテキスト内からネットワーク要求を行うことはできません。

ネットワーク操作は、ホストが公開する一連の機能を活用することで行うことができる。

== DNSホスト検索

この関数は、ポリシーで指定されたFQDNから始まるDNSルックアップを実行します。

=== キャッシング

検索結果は1分間キャッシュされる。

=== 通信プロトコル

これは、このケイパビリティを公開するために使用されるwaPCプロトコルの説明である：

==== waPC機能 -`v1/dns_lookup_host` 入力

[,hcl]
----
# hostname - JSON encoded string
string
----

==== waPC機能 -`v1/dns_lookup_host` 出力

[,hcl]
----

{
  # list of IPs
  "ips": [string]
}
----

与えられたFQDNに関連するすべてのIPアドレスが、レスポンスに文字列として返される。
IPv4とIPv6の両方のエントリーが同じ応答の一部として返される。

たとえば、`busybox:latest` 画像のマニフェスト・ダイジェストを要求する場合、ペイロードは次のようになる：

* 入力ペイロード： `"google.com"`
* 出力ペイロード： `{ "ips": ["127.0.0.1"]}`
