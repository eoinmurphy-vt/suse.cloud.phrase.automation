= Instalación de kwctl ({project-name} CLI)
:revdate: 2026-03-18
:page-revdate: {revdate}
:title: Instalación de kwctl
:description: Instale kwctl (la CLI de {project-name} ) siguiendo sus instrucciones de instalación para varios sistemas operativos y, a continuación, active la finalización del shell.
:keywords: ["{project-name}", kubewarden, kubernetes, instalar kwctl, instalar, kwctl]
:doc-persona: kubewarden-todos
:doc-type: cómo
:doc-topic: install-kwctl

`kwctl` es la herramienta de interfaz de línea de comandos (CLI) para {project-name}. A continuación encontrará instrucciones de instalación para algunos sistemas operativos.

[tabs]
====
Linux::
+
--
Instalación para Linux

. Elaboración casera
+
Si utilizas el gestor de paquetes Homebrew en Linux, entonces:
+
[source,console]
----
brew install kwctl
----
+
Verifique la instalación:
+
[source,console]
----
kwctl --version
----

. Zypper (openSUSE)
+
[source,console]
----
zypper install kwctl
----

. AUR
+
Si utilizas Arch Linux, o una distribución basada en Arch, puedes instalar kwctl desde la AUR.
+
.. Utilizando un ayudante AUR (yay):
+
[source,console]
----
yay -S kwctl
----
+
.. Usando makepkg:
+
[source,console]
----
# Clone the AUR package
git clone https://aur.archlinux.org/kwctl.git
cd kwctl

# Build and install the package
makepkg -si
----
+
.. Verifica la instalación:
+
[source,console]
----
kwctl --version
----

. Instalación manual
+
.. Descargue la última versión de `kwctl` para Linux:
+
[source,console]
----
curl -LO https://github.com/kubewarden/kwctl/releases/latest/download/kwctl-linux-x86_64.zip
----
+
Para sistemas ARM64 (por ejemplo, Raspberry Pi), utilice:
+
[source,console]
----
curl -LO https://github.com/kubewarden/kwctl/releases/latest/download/kwctl-linux-aarch64.zip
----
+
.. Extraiga los archivos del archivo `.zip` descargado:
+
[source,console]
----
unzip kwctl-linux-x86_64.zip
----
+
Esto extrae los siguientes archivos:
+
* `kwctl-linux-x86_64`: El binario `kwctl` 
* `kwctl-linux-x86_64.sig`: Un archivo de firma para verificar el binario
* `kwctl-linux-x86_64.pem`: Un archivo de certificado para verificar la firma
+
.. Mueva el binario a un directorio de su PATH, renómbrelo a `kwctl` y hágalo ejecutable.
+
.. Verifica la instalación:
+
[source,console]
----
kwctl --version
----

Windows::
+
--
Instalación para Windows

. Descargar `kwctl`:
    .. Abra su navegador y vaya a la https://github.com/kubewarden/kwctl/releases/latest[página de lanzamientos]
    .. Descargue el archivo `kwctl-windows-x86_64.zip`.

. Extraiga los archivos del archivo zip descargado. Contendrá:
+
* `kwctl-windows-x86_64.exe`: el binario `kwctl`.
* `kwctl-windows-x86_64.sig`un archivo de firma para verificar el binario.
* `kwctl-windows-x86_64.pem`un archivo de certificado para verificar la firma.

. Cambie el nombre del archivo binario de `kwctl-windows-x86_64.exe` a `kwctl.exe` para facilitar su uso.

. Mueva el binario a una ubicación cubierta por su variable de entorno `PATH`.

. Verifica la instalación. Abra una nueva ventana del símbolo del sistema o de PowerShell y compruebe la instalación de `kwctl`:
+
[source,console]
----
kwctl --version
----

== Instalar terminaciones de shell

El `kwctl` CLI tiene la opción `--shell` para generar comandos de finalización de shell para su shell. Puede utilizar la salida de este comando para integrar terminaciones en su shell.

[source,console]
----
kwctl completions --shell [bash|elvish|fish|powershell|zsh]
----
