Los PolicyServers respetan las variables de entorno estándar para tratar con configuraciones proxy. El servidor de políticas utiliza el proxy configurado al descargar las políticas.
Esto incluye las operaciones de Sigstore al verificar las políticas y las imágenes.

Estas variables son:

* `HTTP_PROXY` o `http_proxy`: servidor proxy para peticiones HTTP
* `HTTPS_PROXY` o `https_proxy`: servidor proxy para peticiones HTTPS
* `NO_PROXY` o `no_proxy`: lista separada por comas de hosts que se excluirán del proxy

Consulte xref:reference/CRDs.adoc#_policyserverspec[`PolicyServer.spec.env`]en la referencia CRD para configurar estas variables de entorno.
