https://www.rust-lang.org/[Rust] es el lenguaje de programación más maduro que puede generar módulos WebAssembly.
WebAssembly es un ciudadano de primera clase en el mundo Rust, por lo que muchas de las herramientas y crates del ecosistema Rust funcionan desde el primer momento.

{project-name} proporciona un Rust https://crates.io/crates/kubewarden-policy-sdk[SDK] que simplifica el proceso de escribir políticas.
También hay un https://github.com/kubewarden/rust-policy-template[proyecto de plantilla] para proporcionar andamiaje para un proyecto de política utilizando la https://github.com/cargo-generate/cargo-generate[] utilidad.

Esta documentación muestra cómo utilizar estos proyectos para escribir políticas {short-project-name} utilizando Rust.
No cubre los detalles de {short-project-name}'s Rust SDK.
Los detalles se encuentran en la https://docs.rs/kubewarden-policy-sdk/0.1.0[documentación de la caja].

== Obtener las dependencias de Rust

Instale el compilador Rust y sus herramientas utilizandohttps://github.com/rust-lang/rustup[rustup].
Consulte la rustup https://rust-lang.github.io/rustup/installation/index.html[documentación de instalación de].

Una vez que haya instalado `rustup` añada el objetivo WebAssembly System Interface (WASI):

[subs="+attributes",console]
----
rustup target add wasm32-wasip1
----

== Dependencias de OSX

Para utilizar `cargo-generate` es necesario añadir el conjunto de herramientas Xcode.
Si no se instala a través de Xcode el siguiente comando le da las dependencias necesarias:

[subs="+attributes",console]
----
xcode-select --install
----
