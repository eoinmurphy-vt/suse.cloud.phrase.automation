Le compilateur officiel Go peut produire des binaires WebAssembly, pour une exécution en dehors d'un navigateur, depuis la version 1.21.

Il existe un autre compilateur Go qui peut construire des binaires WebAssembly utilisables par {project-name}.
Ce projet de compilateur est https://tinygo.org/[] TinyGo :

____
TinyGo apporte le langage de programmation Go aux systèmes embarqués et au web moderne en créant un nouveau compilateur basé sur LLVM.

Vous pouvez compiler et exécuter des programmes TinyGo sur plus de 94 cartes de microcontrôleurs différentes, telles que le BBC micro:bit et l'Arduino Uno.

TinyGo peut également produire du code WebAssembly (Wasm) qui est très compact.
Vous pouvez compiler des programmes pour les navigateurs web, ainsi que pour les serveurs et les environnements informatiques périphériques qui prennent en charge la famille d'interfaces WebAssembly System Interface (WASI).
____

Le projet {short-project-name} propose actuellement d'utiliser TinyGo pour deux raisons :

* les binaires sont plus petits
* la prise en charge de https://wapc.io[waPC] par la possibilité d'exporter des fonctions vers le moteur d'exécution

== Limites de TinyGo

TinyGo ne supporte pas encore toutes les fonctionnalités de Go, voir laTinyGo language support https://tinygo.org/lang-support/[page] pour voir l'état actuel du projet.
Actuellement, sa plus grande limite est l'absence d'un paquetage `reflect` entièrement pris en charge.
Cela signifie que les types d'API officiels de Kubernetes Go (par exemple : `k8s.io/api/core/v1`) ne se compilent pas.

{short-project-name} doivent traiter des données JSON telles que les paramètres de la politique et la demande reçue par Kubernetes.

Malgré les limites actuelles de TinyGo, il est toujours facile d'écrire des politiques de validation sur {short-project-name}.

== Outillage

L'écriture des politiques {short-project-name} nécessite une version de TinyGo supérieure à `v0.28.1`.
Toutefois, pour obtenir les meilleurs résultats, il convient d'utiliser la dernière version.

[CAUTION]
====
L'utilisation de versions antérieures de TinyGo entraîne des erreurs d'exécution dues à la prise en charge limitée de la réflexion Go.
====


Ces bibliothèques Go sont utiles lors de l'écriture d'une politique {short-project-name}:

* https://github.com/kubewarden/policy-sdk-go[{short-project-name} Go SDK]:
Fournit des structures et des fonctions réduisant la quantité de code nécessaire.
Il fournit également des aides au test.
* https://github.com/kubewarden/k8s-objects[Types de Kubernetes Go]:
Les https://github.com/kubernetes/kubernetes/tree/master/staging/src/k8s.io[types officiels Kubernetes Go de]ne peuvent pas être utilisés avec TinyGo.
Ce module fournit tous les types de Kubernetes d'une manière conviviale pour TinyGo.
* https://github.com/tidwall/gjson[gjson]:
Il s'agit d'un langage d'interrogation permettant de naviguer rapidement dans les documents JSON et d'extraire des données.
Cette bibliothèque n'utilise pas le paquetage `encoding/json` fourni par Go's `stdlib`, elle est donc utilisable avec TinyGo.
* https://github.com/deckarep/golang-set[mapset]:
Fournit une implémentation Go de la structure de donnéeshttps://en.wikipedia.org/wiki/Set_(abstract_data_type)[Set.]
Cette bibliothèque réduit la quantité de code à écrire car les opérations telles que Set `union`, `intersection`, `difference` sont des opérations courantes dans les politiques {short-project-name}.

Enfin, le projet {short-project-name} fournit unhttps://github.com/kubewarden/go-policy-template[modèle de projet de politique Go]que vous pouvez utiliser pour créer des politiques Go {short-project-name}.

== Obtenir les dépendances de TinyGo

La façon la plus simple d'obtenir TinyGo est d'utiliser les images des conteneurs en amont.
Les versions officielles sont disponibles àhttps://hub.docker.com/r/tinygo/tinygo[l'adresse], tandis que les versions de la branche de développement sont automatiquement poussées à l'https://hub.docker.com/r/tinygo/tinygo-dev[].

Si nécessaire, essayez la page dehttps://tinygo.org/getting-started/[démarrage]de TinyGo pour plus d'informations.
