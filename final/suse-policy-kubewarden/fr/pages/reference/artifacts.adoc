== Artéfacts de l'OCI

Pour les artefacts de l'Open Container Initiative (OCI), le projet publie tout dans `ghcr.io` sous https://github.com/orgs/kubewarden/packages.

- https://github.com/orgs/kubewarden/packages/policies[`kubewarden/packages/policies/*`]
  sont les artefacts Wasm OCI de la politique.
- https://github.com/orgs/kubewarden/packages/charts[`kubewarden/packages/charts/*`]
  sont les diagrammes de Helm en tant qu'artefacts de l'OCI. Ils suivent les normes de la SLSA.
- les images de conteneurs publiées en tant qu'images OCI : `kubewarden-controller`, `policy-server`, `audit-scanner`, `kubectl`.
- il y a aussi `packages/tests/*` qui contient des artefacts utilisés dans les tests e2e.
