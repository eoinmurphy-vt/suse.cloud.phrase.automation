{project-name} organise la documentation du projet comme suit :

* xref:/introduction.adoc[Introduction]
* xref:/quick-start.adoc[Démarrage rapide]
* _Didacticiels - apprentissage_par l'exemple.
* _Explications -_concepts _sous-jacents_.
* _Comment faire pour_accomplir les tâches de {short-project-name}.
* _Référence_
* Organisation - cette page
* xref:/personas.adoc[Personas -]à qui s'adresse {short-project-name}?
* xref:/disclosure.adoc[Sécurité]
* xref:/glossary.adoc[Glossaire]

La majeure partie du matériel se trouve dans les quatre sections suivantes : _didacticiels_, _explications_,_modes d'emploi_ et _références_. Il peut y avoir des chevauchements, car les sections ne s'excluent pas mutuellement. Le projet {short-project-name} s'efforce de suivre le cadre de documentation delink:https://diataxis.fr/[Diátaxis].

== Navigation

Le menu de gauche indique toujours l'emplacement du document en cours.
Le menu de droite affiche les sections de la page en cours. Il existe également une fonction de "fil d'Ariane" en haut du panneau de contenu principal de chaque page.
