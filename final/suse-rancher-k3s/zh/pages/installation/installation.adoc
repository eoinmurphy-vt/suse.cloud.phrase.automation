= 安装
:page-languages: [EN, JA, KO, ZH]
:revdate: 2025-09-11
:page-revdate: {revdate}

本节包含在各种环境中安装 K3 的说明。在开始安装 K3s 之前，请确保您已xref:installation/requirements.adoc[满足要求]。

xref:installation/configuration.adoc[配置选项]为您安装 K3 时的可用选项提供指导。

xref:installation/private-registry.adoc[专用注册表配置]包括使用`registries.yaml` 配置容器映像注册表验证和镜像。

xref:installation/registry-mirror.adoc[嵌入式注册表镜像]显示如何启用嵌入式分布式图像注册表镜像，以实现节点间图像的点对点共享。

xref:installation/airgap.adoc[Air-Gap Install]详细介绍了如何在无法直接访问互联网的环境中安装 K3。

xref:installation/server-roles.adoc[管理服务器角色]》详细介绍了如何使用专用`control-plane` 或`etcd` xref:installation/server-roles.adoc[服务器]设置 K3。

xref:installation/packaged-components.adoc[管理打包组件]》详细介绍了如何禁用打包组件，或使用自动部署清单安装自己的组件。

xref:installation/uninstall.adoc[卸载 K3s]详细说明了如何从主机上卸载 K3s。
