= Instalación
:page-languages: [en, ja, ko, zh]
:revdate: 2025-09-11
:page-revdate: {revdate}

Esta sección contiene instrucciones para instalar el K3 en distintos entornos. Asegúrese de que cumple los xref:installation/requirements.adoc[requisitos] antes de empezar a instalar el K3s.

xref:installation/configuration.adoc[Opciones de configuración] ofrece orientación sobre las opciones disponibles al instalar K3s.

xref:installation/private-registry.adoc[Configuración del Registro Privado] cubre el uso de `registries.yaml` para configurar la autenticación y réplica del registro de imágenes de contenedor.

xref:installation/registry-mirror.adoc[Embedded Registry Mirror] muestra cómo habilitar la réplica del registro de imágenes distribuido embebido, para compartir imágenes entre nodos de igual a igual.

xref:installation/airgap.adoc[Air-Gap Install] detalla cómo configurar el K3 en entornos que no tienen acceso directo a Internet.

xref:installation/server-roles.adoc[Managing Server Roles] detalla cómo configurar K3s con servidores dedicados `control-plane` o `etcd`.

xref:installation/packaged-components.adoc[Gestión de componentes] empaquetados detalla cómo desactivar componentes empaquetados o instalar los suyos propios mediante manifiestos de despliegue automático.

xref:installation/uninstall.adoc[Desinstalación de K3s] detalla cómo eliminar K3s de un host.
