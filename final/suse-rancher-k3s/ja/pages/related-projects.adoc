= 関連プロジェクト
:page-languages: [en、ja、ko、zh]。
:revdate: 2025-03-31
:page-revdate: {revdate}

K3sの配布を実施するプロジェクトは、コミュニティを拡大するために歓迎される。これらのプロジェクトは、次のようなことに役立っている：

* K3の能力と応用の可能性を探る。
* 異なるプラットフォームや環境にK3の機能を拡張する。
* 大規模な高可用性（HA）K3sクラスタの作成プロセスを簡素化します。

== k3s-ansible

マルチノードのK3sクラスタをブートストラップしようとしているユーザーで、ansibleに精通している方は、https://github.com/k3s-io/k3s-ansible[]k3s-io/k3s-ansibleリポジトリを見てください。このansible playbooksのセットは、ノードにK3をインストールする便利な方法を提供し、インストールプロセスよりもクラスタの設定に集中できるようにします。

== k3sup

K3sクラスタをセットアップするプロセスを簡素化するもう一つのプロジェクトはhttps://github.com/alexellis/k3sup[] k3supである。このプロジェクトはgolangで書かれており、ノードへのsshアクセスだけが必要です。また、組み込みのetcdだけでなく、外部のデータストアでK3をデプロイする便利な方法も提供する。

== autok3s

もう1つのプロビジョニングツールであるhttps://github.com/cnrancher/autok3s[autok3sは]、クラウドプロバイダー、VM、ローカルマシンの範囲にわたってk3sクラスタをプロビジョニングするためのGUIを提供する。このツールは、K3sクラスターを証明するためのグラフィカルなインターフェイスを好むユーザーに便利です。

== ヘッツナーK3S

Hetzner Cloudにデプロイしているユーザーは、link:https://github.com/vitobotta/hetzner-k3s[hetzner-k3sを見て]ください。Crystalで書かれたこのCLIツールは、Hetzner Cloud上でK3sクラスタをセットアップするために必要な追加ステップを処理します。
