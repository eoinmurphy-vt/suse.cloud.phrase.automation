= Installation
:page-languages: [en, ja, ko, zh]
:revdate: 2025-09-11
:page-revdate: {revdate}

Dieser Abschnitt enthält Anweisungen für die Installation von K3s in verschiedenen Umgebungen. Bitte stellen Sie sicher, dass Sie die xref:installation/requirements.adoc[Anforderungen] erfüllen, bevor Sie mit der Installation von K3s beginnen.

Unter xref:installation/configuration.adoc[Konfigurationsoptionen] finden Sie Hinweise zu den Optionen, die Ihnen bei der Installation von K3s zur Verfügung stehen.

xref:installation/private-registry.adoc[Private Registry Configuration] behandelt die Verwendung von `registries.yaml` zur Konfiguration der Authentifizierung und Spiegelung der Container-Image-Registry.

xref:installation/registry-mirror.adoc[Embedded Registry Mirror] zeigt, wie man den eingebetteten, verteilten Image Registry Mirror für den Peer-to-Peer-Austausch von Images zwischen Knoten aktiviert.

xref:installation/airgap.adoc[Air-Gap-Installation] beschreibt die Einrichtung von K3s in Umgebungen, die keinen direkten Zugang zum Internet haben.

Die xref:installation/server-roles.adoc[Verwaltung von Serverrollen] beschreibt, wie Sie K3s mit dedizierten `control-plane` oder `etcd` Servern einrichten.

Unter xref:installation/packaged-components.adoc[Verwalten von Paketkomponenten] erfahren Sie, wie Sie Paketkomponenten deaktivieren oder Ihre eigenen Komponenten mithilfe von automatisch bereitgestellten Manifesten installieren können.

Die xref:installation/uninstall.adoc[Deinstallation von K3s] beschreibt, wie man K3s von einem Host entfernt.
