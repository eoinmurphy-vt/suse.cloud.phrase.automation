== Voraussetzungen

Die aktuelle Version des KI-Assistenten erfordert die folgenden Komponenten:

* https://helm.sh/docs/[Helm installiert]
* Rancher Prime/Community Manager 2.13 oder höher

Stellen Sie sicher, dass Sie die Rechte haben, CRD (cluster_admin) im Host-Cluster bereitzustellen.

== Installieren Sie {project-name}

Die Installation von {short-project-name} ist ein zweistufiger Prozess:

* Setzen Sie den Agenten und das MCP über das bereitgestellte Helm-Chart ein.
* Setzen Sie die Rancher UI-Erweiterung ein.

=== Installieren Sie den Agenten und das MCP im lokalen Cluster

. Erstellen Sie eine `values.yaml`-Dateieinstellung (für Ollama):
+
[source,yaml]
----
ollamaLlmModel: "gpt-oss:20b"
ollamaUrl: "http://ollama:11434"
activeLlm: "ollama"
----
+
[NOTE]
====

Sie können diese Einstellungen und Anbieter später in den „globalen Einstellungen -> KI-Assistent-Tabs“ von Rancher ändern.

====
. Installieren Sie das rancher-ai-agent Chart
+
[source,bash,subs="attributes+"]
----
helm install rancher-ai-agent \
  --namespace cattle-ai-agent-system \
  --create-namespace \
  -f values.yaml {oci-reg}
----

=== Installieren Sie die UI-Erweiterung

Öffnen Sie die Rancher Manager UI und navigieren Sie zur Seite 'Erweiterungen'.

. Klicken Sie im Rancher Manager auf Erweiterung in der Menüleiste.
+
image::menubar-extension.png[Erweiterungsschaltfläche in der Menüleiste]
+
. Verwenden Sie das Drei-Punkte-Menü in der oberen rechten Ecke und wählen Sie 'Rancher-Repositories hinzufügen' aus.
+
image::rancher-repositories.png[Menü zur Verwaltung von Rancher-Repositories]
+
. Wählen Sie 'Offizielles Rancher-Erweiterungs-Repository hinzufügen', um das Repository hinzuzufügen.
+
image::add-extensions-repositories.png[Offizielles Rancher-Repository hinzufügen]
+
. Klicken Sie auf 'Hinzufügen'.
+
. Suchen Sie die Karte des KI-Assistenten und klicken Sie auf 'Installieren'.
. Wählen Sie eine Version aus (oder verwenden Sie standardmäßig die neueste) und klicken Sie auf 'Installieren'.
+
image::config-repo-install.png[Wählen Sie eine Version aus oder verwenden Sie die neueste.]
+
. Sobald die Erweiterung installiert ist, klicken Sie auf die Schaltfläche 'Neu laden', die
 oben auf der Seite erscheint.
