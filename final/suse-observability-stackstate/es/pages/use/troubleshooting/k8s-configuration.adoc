= Configuración YAML
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: Observabilidad de SUSE

Si un pod experimenta problemas como bloqueos, fallos al arrancar o errores de configuración, inspeccionar la configuración YAML puede ayudarle a identificar la causa raíz. Puede ver la configuración YAML pulsando el botón *Mostrar configuración*.

image::k8s/k8s-configuration.png[]

Esto da el mismo resultado que el comando `kubectl describe` aplicado al servicio, pod u otro recurso.

image::k8s/k8s-configuration-opened.png[]
