= Variables del cuadro de mandos
:revdate: 2025-11-24
:page-revdate: {revdate}
:description: Utilice variables para crear cuadros de mando dinámicos.

Configurando variables para un cuadro de mando, puede crear cuadros de mando con widgets dinámicos. Las variables permiten reutilizar una misma definición de cuadro de mando para distintos casos de uso. 

== Visualización de las variables del cuadro de mandos
En la barra de navegación superior, utilice el menú desplegable situado junto al botón "Guardar..." y haga clic en "Editar variables del cuadro de mandos".

image::dashboards/open-variables-drawer.png[Añadir todos los elementos anclados]

=== Variables incorporadas
Hay variables que están integradas en {stackstate-product-name}. Los nombres de estas variables están reservados, y no puedes dar a tus propias variables los mismos nombres.

Para ver todas las variables incorporadas, haga clic en el acordeón titulado "Variables PromQL de SUSE Observability".

== Añadir una nueva variable

. Abra el cajón "Editar variables del cuadro de mandos". 
. Haz clic en el botón "Añadir variable" para abrir un segundo cajón en el que podrás configurar la nueva variable.

image::dashboards/variable-drawer-example.png[Una variable de ejemplo]

Los campos de configuración de una variable son los siguientes:

|===
| Nombre del campo | Descripción

| Nombre
| El nombre identificador de la variable. Este es el nombre que puede <<using-variables, utilizar en campos compatibles>>.

| Mostrar etiqueta
| Un nombre amigable para la variable que se muestra en el cuadro de mandos cuando se <<selecting-variables, selecciona la variable>>. Esto le ayuda a identificar la variable.

| Descripción
| Una descripción para entender el propósito de la variable. Esta descripción se muestra en el cajón "Editar variables del cuadro de mandos".

| Tipo
| Tipo de la variable.
|===

== Tipos de variables
=== Variable de texto

Una variable de texto contiene texto simple. Puede establecer un valor inicial y hacer que la variable sea de sólo lectura.

=== Variable de lista

Una variable de lista tiene valores posibles definidos por el campo "Fuente". Cada tipo de fuente proporciona campos adicionales que permiten acotar los posibles valores de la variable.

|===
| Tipo de fuente | Campos fuente

| Nombres de etiquetas métricas
| Introduzca cero o más selectores de serie para refinar qué nombres de etiquetas métricas están presentes.

| Valores métricos de las etiquetas
| Introduzca un nombre de etiqueta métrica, y cero o más selectores de serie para refinar qué valores de etiqueta métrica están presentes.

| Métrica PromQL
| Introduzca una consulta PromQL personalizada y un nombre de etiqueta. Los valores posibles resultantes son los valores para el nombre de etiqueta dado.

| Lista estática
| Introduzca uno o más pares de etiqueta y valor para construir una lista estática de posibles valores.
|===

A medida que cambie el tipo de fuente y sus campos relacionados, la vista previa de la parte derecha del cajón se actualizará con los nuevos valores posibles.


==== Ajustes comunes de las variables de lista

Todas las fuentes de variables de lista tienen también los siguientes ajustes comunes:

|===
| Nombre del campo | Descripción

| Ordenar
| Ordena los valores de la lista. Por ejemplo, alfabética o numéricamente.

| Permitir valores múltiples
| Permitir a los usuarios seleccionar más de un valor a la vez.

| Activar "Incluir todo"
| Añade un elemento para seleccionar todos los valores a la lista que ve el usuario. Opcionalmente, puede especificar un valor personalizado para este elemento o dejar que se genere automáticamente. 
|===

== Utilización de variables [[using-variables]]
Una vez definida una variable, puede utilizarse para rellenar diversos campos del cuadro de mandos:

* Otras variables
* Nombre y descripción del widget
* Campos de enlace del widget: URL, Nombre y Tooltip
* Campos de consulta del widget: Expresión PromQL, Alias y Paso Mínimo
* Editor del widget Markdown

Para utilizar una variable en un campo compatible, utilice el nombre de la variable encerrado en `${}` (por ejemplo, `$\{my_variable}`). Se interpola el valor de la variable.

=== Cómo se interpolan las variables
Las variables de texto se interpolan como la cadena que se escribe en el cuadro de texto de la variable.

Del mismo modo, las variables de lista que no permiten valores múltiples ni "Incluir todo" se interpolan como el valor del elemento seleccionado de la lista.

Las variables de lista que permiten múltiples valores se interpolan como un grupo de captura de expresiones regulares de los valores seleccionados (por ejemplo `(value_1|value_2)`) independientemente de si se selecciona una o varias opciones.

Las variables de lista que tengan la opción "Incluir todo" activada y seleccionada para esa variable interpolarán como:* El valor personalizado si está definido, o * Una expresión regular generada capturando el grupo de todos los valores posibles (por ejemplo `(value_1|value_2|value_3)`)

Puede utilizar el comparador de expresiones regulares PromQL `=~` cuando consuma variables interpoladas como grupos de captura de expresiones regulares.

== Selección de variables [[selecting-variables]]
Las variables aparecen en la parte superior del cuadro de mandos, junto al selector de intervalos de tiempo. Sólo se muestran las primeras variables. Para ver el resto de variables, utilice el desplegable "Más".

Haga clic en una variable para abrir su selector y seleccione los valores que desea mostrar en el cuadro de mandos.

A medida que selecciones diferentes valores, los widgets que utilicen la variable reaccionarán a los cambios.
