== Visión general

Utilice xref:use/security/k8s-service-tokens.adoc[los tokens de servicio] para autenticarse en SUSE Observability sin tener una cuenta de usuario asociada. Los tokens de servicio son la forma preferida de autenticarse desde CI/CD, o cuando se envían datos de telemetría desde agentes u Open Telemetry. La forma de crear y utilizar tokens de servicio se describe en la xref:use/security/k8s-service-tokens.adoc[documentación sobre tokens de servicio].
