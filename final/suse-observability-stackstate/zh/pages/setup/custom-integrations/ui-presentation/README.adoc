本节记录了作为*StackPacks 2.0* 的一部分引入的*用户界面呈现*模型。

== 概述

这里记录的组件展示模型声明了组件（如 pod、虚拟机、服务）在{stackstate-product-name} 用户界面中的展示方式。
它只关注*演示方面的问题*，如命名、图标、表格、概览和高亮页面、过滤器以及菜单项的位置。

与现有的较为零散的方法相比，该模型的目标是提供一种可组合性和可扩展性更强的方法来定义组件的用户界面行为。
