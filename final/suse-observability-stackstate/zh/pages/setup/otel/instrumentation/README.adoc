= 阅读说明
:description: SUSE 可观察性

开放式遥测提供了https://opentelemetry.io/docs/languages/[为多种语言]，用于（自动）检测您的应用程序。仪器使您的应用程序能够向 SUSE Observability 发送跟踪、度量和日志。自动监测功能可捕获流行库和框架的跟踪、指标和日志，除在应用程序中包含 SDK 外，无需任何进一步的手动干预。

Open Telemetry 的 SDK 比这里记录的要多得多。如果这里没有记录您的语言，您可以使用https://opentelemetry.io/docs/languages/[opentelemetry.io]上的文档，并结合 SUSE Observability 所需的xref:/setup/otel/instrumentation/sdk-exporter-config.adoc[SDK 输出程序配置]。
