= Unités prises en charge pour les graphiques
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: SUSE Observability

Cette page contient toutes les unités prises en charge par les graphiques. Pour chaque unité, la description de l'unité est d'abord indiquée, suivie du nom de l'unité à utiliser dans les API de SUSE Observability. Par exemple, pour afficher une valeur en "notation scientifique", utilisez `sci` comme valeur pour l'unité dans un lien métrique.

Le rendu de l'unité s'ajuste automatiquement à la taille de la valeur. Par exemple, si l'unité pour un graphique est `bytes`, la valeur est interprétée comme `bytes(IEC)`. Une valeur de `100` sera rendue comme `100B`, mais une valeur de `5242880` (`5*1024*1024` sera affichée comme `5MiB`.
