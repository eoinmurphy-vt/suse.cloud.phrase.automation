:description: SUSE Observability

== Vue d'ensemble

Utilisez les xref:use/security/k8s-service-tokens.adoc[jetons de service] pour vous authentifier auprès de SUSE Observability sans avoir de compte utilisateur associé. Les jetons de service sont le moyen privilégié pour s'authentifier à partir de CI/CD, ou lors de l'envoi de données télémétriques à partir d'agents ou d'Open Telemetry. La création et l'utilisation des jetons de service sont décrites dans la xref:use/security/k8s-service-tokens.adoc[documentation sur les jetons de service].
