= LEIAME
:description: Observabilidade do SUSE

A telemetria aberta fornece https://opentelemetry.io/docs/languages/[SDKs para muitas linguagens] para instrumentar (automaticamente) seu aplicativo. A instrumentação permite que seu aplicativo envie rastreamentos, métricas e registros para o SUSE Observability. A instrumentação automática captura rastreamentos, métricas e registros de bibliotecas e estruturas populares sem nenhuma intervenção manual adicional, exceto a inclusão do SDK no seu aplicativo.

A Open Telemetry tem muito mais SDKs do que os documentados aqui. Se o seu idioma não estiver documentado aqui, você poderá usar a documentação em https://opentelemetry.io/docs/languages/[opentelemetry.io] em combinação com a xref:/setup/otel/instrumentation/sdk-exporter-config.adoc[configuração do exportador do SDK] necessária para o SUSE Observability.
