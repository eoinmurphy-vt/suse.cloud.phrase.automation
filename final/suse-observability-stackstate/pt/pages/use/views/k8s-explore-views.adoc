== Visão geral

Em geral, o conceito de xref:/use/views/k8s-view-structure.adoc[visualização] no SUSE Observability permite monitorar uma área do cenário de TI que você salvou anteriormente. Mas, muitas vezes, quando você precisa investigar um subconjunto de uma determinada visualização, não quer perder o escopo dessa visualização. É aqui que as *visualizações de exploração* entram em ação.

Para manter intacto o escopo de uma determinada visualização (por exemplo, `my view`), todas as ações investigativas aplicadas a um elemento de topologia ou seleção de elementos (por exemplo, componentes, relações, grupos) serão automaticamente abertas em uma visualização de exploração temporária, sob a visualização da qual você começou (por exemplo, `my view / explore`).

Exemplos de *ações investigativas* que serão abertas automaticamente em uma visualização de exploração:

* O * botão +* foi clicado em um componente na xref:/use/views/k8s-topology-perspective.adoc[perspectiva da topologia] para mostrar os vizinhos ocultos desse componente
* Uma *ação rápida* foi executada em um componente
* Um elemento de topologia (componente, relação, grupo) foi *clicado duas vezes* na xref:/use/views/k8s-topology-perspective.adoc[perspectiva de topologia] para investigá-lo
* O link ou botão *'Explore...'* foi clicado em um elemento de topologia (componente, relação, grupo) para investigá-lo

[TIP]
====
As ações investigativas executadas a partir de uma visualização de exploração existente *não* abrirão uma nova visualização de exploração - elas mudarão o escopo da visualização de exploração em que você está.
====


== Manuseio

Quando uma visualização de exploração é criada automaticamente, seu escopo original também é definido. Isso é útil se você quiser redefinir a visualização de exploração para o escopo original depois de fazer algumas alterações nela.

Embora uma visualização de exploração deva ser temporária, ela pode ser salva como uma xref:/use/views/k8s-custom-views.adoc[visualização personalizada] se você quiser monitorá-la ou revisá-la mais tarde.

Se você não quiser salvar uma visualização de exploração, basta sair dela ou voltar para a visualização inicial usando as trilhas de navegação na barra de navegação superior.

== Estrutura

As exibições de exploração têm uma estrutura idêntica à das xref:/use/views/k8s-custom-views.adoc[exibições personalizadas]: elas têm os mesmos xref:/use/views/k8s-view-structure.adoc#_filters[filtros] e as mesmas xref:/use/views/k8s-view-structure.adoc#_perspectives[perspectivas].
