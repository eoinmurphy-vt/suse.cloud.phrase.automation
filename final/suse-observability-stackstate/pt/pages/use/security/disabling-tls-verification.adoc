== Agente de observabilidade do SUSE

Para desativar a validação SSL para o xref:/use/security/self-signed-certificates.adoc#_suse_observability_agent[SUSE Observability Agent], você precisa configurar o valor da string Helm conforme mostrado abaixo:

[,bash]
----
--set-string 'global.skipSslValidation'=true \
----

== CLI do SUSE Observability

Para desativar a validação SSL na xref:/use/security/self-signed-certificates.adoc#_suse_observability_cli[CLI do SUSE Observability], você precisa especificar o sinalizador `--skip-ssl` com os comandos `sts CLI`. Isso pode ser feito usando um dos dois métodos abaixo:

*Método 1*: 

Especifique o sinalizador `--skip-ssl` no arquivo de certificado CA codificado por PEM para configuração persistente:

[,bash]
----
sts context save \
 --name staging \
 --url https://staging.internal \
 --api-token YOUR_API_TOKEN \
 --skip-ssl
----

*Método 2*:

Especifique o sinalizador `--skip-ssl` em qualquer comando da CLI que seja usado para validação de certificado único:

[,bash]
----
# Using certificate file path
sts agent list \
 --url https://staging.internal \
 --api-token YOUR_API_TOKEN \
 —-skip-ssl


# Using base64-encoded certificate data
sts settings list \
 --url https://staging.internal \
 --api-token YOUR_API_TOKEN \
 —-skip-ssl
----

[WARNING]
====
O uso do sinalizador `--skip-ssl` ignorará as configurações `STS_CA_CERT_PATH` e `STS_CA_CERT_BASE64_DATA`. Use com cuidado, pois isso reduz a segurança.
====

== Extensão da interface do usuário do Rancher para o SUSE Observability

Para desativar a validação SSL da extensão Rancher UI, você precisa alterar a configuração `agent-tls-mode`. 

Para alterar o valor da configuração por meio da interface do usuário, navegue até a página Configurações globais e localize a configuração `agent-tls-mode` na parte inferior da página. Depois de alterar a configuração por meio da interface do usuário, os clusters são verificados para que a condição `AgentTlsStrictCheck` seja definida como "True" antes de permitir a solicitação. Isso evita interrupções decorrentes de uma incompatibilidade de certificados.
