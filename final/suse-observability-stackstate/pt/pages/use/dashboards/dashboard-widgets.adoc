= Widgets do painel
:revdate: 2025-11-24
:page-revdate: {revdate}
:description: Visualize dados de seus recursos com vários widgets.

Os widgets são os blocos de construção de seu painel. Eles visualizam dados, geralmente de um ou vários recursos do Kubernetes. Os widgets também podem mostrar informações de texto estático aos visualizadores do painel.

== Adição de um novo widget

Você pode adicionar um novo widget de uma das duas maneiras:

* Clique no botão *Add Widget* no canto superior direito de um painel.

* Em um painel vazio, clique na seção *Add Widget (Adicionar widget* ) no meio da tela. Abre-se uma gaveta onde você pode configurar o novo widget.

image::dashboards/add-a-widget.png[Diferentes maneiras de adicionar um widget]

== Adição de um widget existente

Ao visualizar um componente individual em {stackstate-product-name}, você pode adicionar gráficos diretamente a um painel usando o menu "more actions" (mais ações) e selecionando "Add to dashboard" (adicionar ao painel).

image::dashboards/add-existing-widget.png[Adição de um widget existente]

Isso também está disponível para todos os itens fixados, por meio do mesmo menu "mais ações" em um widget individual.

Além disso, você pode adicionar todos os itens fixados a um painel, usando as "mais ações" na gaveta de itens fixados ou clicando no botão "Adicionar todos os itens ao painel" na parte inferior da gaveta.

image::dashboards/add-all-pinned-items.png[Adicionar todos os itens fixados]

Os widgets resultantes no painel têm um link para o recurso do qual foram adicionados.

== Configuração de um widget

Cada widget tem várias opções de configuração. Alguns são comuns a todos os widgets, outros são baseados no tipo de widget.

Para editar a configuração de um widget, use o menu "mais ações" e selecione "Editar widget".

image::dashboards/configure-a-widget.png[Adicionar todos os itens fixados]

=== Campos comuns de widgets

|===
| Nome do campo | Descrição

| Nome
| Um nome curto e descritivo mostrado no cabeçalho do widget.

| Descrição
| Detalhes adicionais mostrados no cabeçalho do widget atrás de uma dica de ferramenta.

| Tipo
| O <<types, tipo de visualização>>.
|===

Além desses campos, há algumas guias comuns a todos os widgets: 

|===
| Nome do registro | Descrição

| Links
| Links de recursos relacionados, exibidos abaixo do nome do widget. O primeiro link é sempre exibido. Outros links estão disponíveis em um menu suspenso.

| YAML
| A configuração YAML bruta do widget, que pode ser visualizada e editada.
|===


=== Guia Consultas

Os tipos de widgets destinados a mostrar dados de recursos terão uma guia *Consultas*. Você pode adicionar uma ou mais consultas PromQL para recuperar os dados do widget.


=== Tipos [[types]]

==== TimeSeriesChart

O site `TimeSeriesChart` visualiza dados de séries temporais em um gráfico de linhas. Cada consulta na configuração do widget produz uma linha no gráfico.

image::dashboards/time-series-chart-example.png[Exemplo de gráfico de série temporal]

Você pode definir as seguintes configurações:

* *Legenda* - A posição e o modo da legenda.
* **Visual** - Conecte valores nulos para criar uma linha contínua.
* **Eixo Y** - A unidade, o rótulo e os valores mínimo/máximo do eixo Y.
* **Limites** - Adicione linhas horizontais ao gráfico para marcar limites de valores importantes.

==== BarChart

O site `BarChart` visualiza dados de séries temporais em um gráfico de barras. Ele tem as mesmas configurações de legenda, eixo Y e limites que o `TimeSeriesChart`.

image::dashboards/bar-chart-example.png[Exemplo de gráfico de barras]

==== Gráfico de estatísticas

O site `StatChart` mostra um gráfico de linhas brilhantes e um único valor que é fácil de ler em um relance. Os limites podem ser configurados para alterar a cor do texto com base no valor.

image::dashboards/gauge-chart-example.png[Exemplo de gráfico de medidor]

Você pode definir as seguintes configurações:

* **Sparkline** - Mostrar ou ocultar o gráfico de sparkline.
* **Unidade** - A unidade do valor.
* **Decimais** - O número de decimais a serem exibidos.
* **Cálculo** - O método usado para calcular o valor a partir dos dados da série temporal.

==== GaugeChart

O site `GaugeChart` exibe um medidor com um valor no meio. A cor do medidor e o texto mudam quando o valor ultrapassa um limite.

image::dashboards/stat-chart-example.png[Exemplo de gráfico de estatísticas]

Você pode definir as seguintes configurações:

* **Unidade** - A unidade do valor.
* **Decimais** - O número de decimais a serem exibidos.
* **Max** - O valor máximo do medidor.
* **Cálculo** - O método usado para calcular o valor a partir dos dados da série temporal.

==== Markdown

O widget `Markdown` exibe texto markdown. Isso pode ser usado para adicionar explicações ou outras informações ao seu painel.

image::dashboards/markdown-chart-example.png[Exemplo de gráfico de remarcação para baixo]
