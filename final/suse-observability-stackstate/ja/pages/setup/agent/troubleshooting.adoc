== プロセスエージェントで特定のネットワークプロトコルのトレースを無効にする。

プロセスエージェントで特定のネットワークプロトコルのトレースを無効にするには、Helmのチャート設定`processAgent.disabledProtocols` 。次の例は、`http2` と`postgres` プロトコルを無効にする：

----
--set "processAgent.disabledProtocols[0]=http2" \
--set "processAgent.disabledProtocols[1]=postgres"
----

== プロセスエージェントを無効にする

お使いの環境でプロセスエージェントがサポートされていない場合は、Helmチャートの設定`nodeAgent.containers.processAgent.enabled` で無効にしてください：

----
--set nodeAgent.containers.processAgent.enabled=false
----
