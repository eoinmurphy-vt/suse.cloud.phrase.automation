= シングルパスワード
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: {stackstate-product-name} セルフホスト

{stackstate-product-name} 、xref:/k8s-suse-rancher-prime.adoc#_installation[インストール時に設定された]単一のパスワードで構成されます。これは、インストール時に提供しなければならない単一のパスワードでユーザーを認証するものである。

これは、{stackstate-product-name} を使い始めるには最も簡単な方法だが、本番環境では他のxref:/setup/security/authentication/authentication_options.adoc[認証オプションの]いずれかを使用することを推奨する。

== 外部秘密からパスワードを取得する

デフォルトのパスワードが外部のシークレットに由来するものである場合は、xref:/setup/security/external-secrets.adoc#_getting_authentication_data_from_an_external_secret[以下の手順に]従いつつ、以下のデータを記入してください：

[,yaml]
----
kind: Secret
metadata:
   name: "<custom-secret-name>"
type: Opaque
data:
  default_password: <base64 of the encoded password>
----

パスワードのエンコーディングは以下のいずれかである：* プレーンテキストのパスワードの場合 `$plain$your-password` (`$plain$` はプレインテキストを示す接頭辞) * 暗号化パスワードの場合は次のように指定する：`bcrypt-hashed-password`.パスワードの暗号化ハッシュを作成するには、`htpasswd -bnBC 10 "" <password> | tr -d ':\n'` を使用します。
