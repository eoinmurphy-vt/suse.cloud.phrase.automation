= お知らせ
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: SUSE Observability

SUSE Observabilityの通知は、SUSE Observabilityのxref:/use/alerting/k8s-monitors.adoc[モニタが]起動したときにサードパーティのアプリケーションに通知する方法です。通知は、インシデント管理ツール（PagerDutyやOpsgenieなど）、発券システム（JIRAやServiceNowなど）、その他のコラボレーションプラットフォーム（Slackなど）などのサードパーティ製アプリケーションと統合できる。他のツールでは、通知をアラートと呼ぶこともある。

通知は、SUSE Observabilityコンポーネントのモニタのヘルス状態の変化によってトリガされます。設定された通知チャネルを通じて、健全性、コンポーネント、モニター情報がこれらの外部システムに送信される。
