= ダッシュボード・ウィジェット
:revdate: 2025-11-24
:page-revdate: {revdate}
:description: 様々なウィジェットでリソースのデータを可視化。

ウィジェットはダッシュボードの構成要素です。通常は1つまたは複数のKubernetesリソースからのデータを可視化する。ウィジェットは、ダッシュボードの閲覧者に静的なテキスト情報を表示することもできる。

== 新しいウィジェットの追加

新しいウィジェットを追加するには、2つの方法があります：

* ダッシュボードの右上にある[*ウィジェットを追加]*ボタンをクリックします。

* 空のダッシュボードで、画面中央の*ウィジェット追加*セクションをクリックします。新しいウィジェットを設定できるドロワーが開きます。

image::dashboards/add-a-widget.png[ウィジェットを追加するさまざまな方法]

== 既存のウィジェットを追加する

{stackstate-product-name} で個々のコンポーネントを表示する際、"more actions "メニューを使い、"Add to dashboard "を選択することで、ダッシュボードに直接チャートを追加することができる。

image::dashboards/add-existing-widget.png[既存のウィジェットを追加する]

これは、個々のウィジェットの同じ "more actions "メニューを通して、ピン留めされたアイテムにも利用可能です。

さらに、ピン留めされたアイテムドロワーの「その他のアクション」を使用するか、ドロワーの一番下にある「すべてのアイテムをダッシュボードに追加」ボタンをクリックすることで、ピン留めされたすべてのアイテムをダッシュボードに追加することができます。

image::dashboards/add-all-pinned-items.png[すべてのピン留めアイテムを追加する]

その結果、ダッシュボードのウィジェットには、追加元のリソースへのリンクが表示されます。

== ウィジェットの設定

各ウィジェットには、多くの設定オプションがあります。すべてのウィジェットに共通するものもあれば、ウィジェットのタイプに応じたものもあります。

ウィジェットの設定を編集するには、"more actions "メニューから "Edit widget "を選択してください。

image::dashboards/configure-a-widget.png[すべてのピン留めアイテムを追加する]

=== 一般的なウィジェット・フィールド

|===
| フィールド名 | 説明

| 名称
| ウィジェットのヘッダーに表示される短い説明的な名前。

| 説明
| ツールチップの後ろのウィジェットのヘッダーに表示される追加詳細。

| タイプ
| <<types, 視覚化のタイプ>>。
|===

これらのフィールドに加えて、すべてのウィジェットに共通のタブがいくつかあります：

|===
| タブ名 | 説明

| リンク
| ウィジェット名の下に表示される関連リソースリンク。最初のリンクは常に表示される。その他のリンクはドロップダウンメニューから選択できます。

| ヤムル
| ウィジェットの生の YAML コンフィギュレーション。
|===


=== クエリタブ

リソースからのデータを表示することを意図したウィジェットタイプは、*クエリタブを*持つ。ウィジェットのデータを取得するために1つ以上のPromQLクエリを追加できます。


=== 種類 [[types]]

==== TimeSeriesChart

`TimeSeriesChart` は、時系列データを折れ線グラフで可視化する。ウィジェット構成の各クエリは、チャート上に線を生成する。

image::dashboards/time-series-chart-example.png[時系列チャートの例]

以下の設定が可能です：

* *Legend*- 凡例の位置とモード。
* **ビジュアル**- ヌル値をつなげて連続線を作成します。
* **Y Axis**- Y軸の単位、ラベル、最小/最大値。
* **しきい値**- 重要な値のしきい値を示す水平線をチャートに追加します。

==== バーチャート

`BarChart` は、時系列データを棒グラフで可視化する。凡例、Y軸、しきい値の設定は`TimeSeriesChart` と同じです。

image::dashboards/bar-chart-example.png[棒グラフの例]

==== スタットチャート

`StatChart` 、スパークラインチャートと一目で見やすい単一の値が表示される。しきい値は、値に基づいてテキストの色を変更するように設定することができます。

image::dashboards/gauge-chart-example.png[ゲージチャート例]

以下の設定が可能です：

* **Sparkline**- スパークラインチャートの表示/非表示を切り替えます。
* **Unit**- 値の単位。
* **Decimals**- 表示する小数の数。
* **計算**- 時系列データから値を計算するために使用される方法。

==== GaugeChart

`GaugeChart` 、中央に数値の入ったゲージが表示される。値がしきい値を超えると、ゲージの色とテキストが変化する。

image::dashboards/stat-chart-example.png[スタッツ・チャート例]

以下の設定が可能です：

* **Unit**- 値の単位。
* **Decimals**- 表示する小数の数。
* **Max**- ゲージの最大値。
* **計算**- 時系列データから値を計算するために使用される方法。

==== マークダウン

`Markdown` ウィジェットはマークダウンのテキストを表示します。これは、ダッシュボードに説明やその他の情報を追加するために使用できます。

image::dashboards/markdown-chart-example.png[マークダウン・チャートの例]
