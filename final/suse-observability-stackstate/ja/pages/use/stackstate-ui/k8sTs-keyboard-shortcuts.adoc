= キーボードショートカット
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: SUSE Observability

以下のキーボードショートカットを使用して、SUSE Observability UIを操作できます。

|===
| キーボードショートカット | アクション

| `CTRL` + `SHIFT` + `F`
| xref:/use/views/k8s-topology-perspective.adoc[トポロジーパースペクティブで]コンポーネントを名前で検索します。

| `CTRL` + `S`
| 以前に保存したビューの設定を上書きします。これにより、保存されたビューが更新され、xref:/use/views/k8s-filters.adoc[ビューフィルタや] xref:/use/views/k8s-topology-perspective.adoc#_visualization_settings[視覚化設定に]加えられた変更が反映されます。

| `CTRL` + `SHIFT` + `S`
| 現在のビュー設定を新しいビューとして保存します。新しいビューには、現在適用されているすべてのxref:/use/views/k8s-filters.adoc[ビューフィルタと] xref:/use/views/k8s-topology-perspective.adoc#_visualization_settings[視覚化設定が]含まれます。

| `CTRL` + `SHIFT` + `R`
| 保存されていない変更をリセットし、元のビュー、サブビュー、またはエクスプローラーモードに戻します。これにより、xref:/use/views/k8s-filters.adoc[ビューフィルタや] xref:/use/views/k8s-topology-perspective.adoc#_visualization_settings[可視化設定に]加えられた未保存の変更がすべてリセットされます。

| `SHIFT` + `1`
| ビュー、サブビュー、またはエクスプローラモードで、xref:/use/views/k8s-topology-perspective.adoc[トポロジーパースペクティブを]開きます。

| `SHIFT` + `2`
| ビュー、サブビュー、またはエクスプローラ モードで、xref:/use/views/k8s-events-perspective.adoc[イベント パースペクティブを]開きます。

| `SHIFT` + `3`
| ビュー、サブビュー、またはエクスプローラ モードで、xref:/use/views/k8s-metrics-perspective.adoc[メトリクス パースペクティブを]開きます。

| `SHIFT` + `4`
| ビュー、サブビュー、またはエクスプローラモードで、xref:/use/views/k8s-traces-perspective.adoc[トレースパースペクティブを]開きます。
|===
