= YAMLコンフィギュレーション
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: SUSE Observability

ポッドがクラッシュしたり、起動に失敗したり、コンフィギュレーションにミスがあったりする場合、YAMLコンフィギュレーションを検査することで根本的な原因を特定することができます。*Show configuration*ボタンをクリックするとYAMLの設定を見ることができます。

image::k8s/k8s-configuration.png[]

これは、`kubectl describe` コマンドをサービス、ポッド、その他のリソースに適用した場合と同じ出力を与える。

image::k8s/k8s-configuration-opened.png[]
