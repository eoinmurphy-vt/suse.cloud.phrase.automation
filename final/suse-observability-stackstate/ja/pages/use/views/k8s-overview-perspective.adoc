= 概要の視点
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: SUSE Observability

概要パースペクティブには、ビュー内のすべてのコンポーネントのリストが表示されます。ビューの種類とビュー内のコンポーネントによって、概要パースペクティブの構造は異なります。

例えば、xref:/use/views/k8s-views.adoc[kubernetesビューで]使用されるテーブル構造は、各ビューに含まれるコンポーネントタイプの最も重要なプロパティを反映します。以下のように、`services` ビューは、`pods` ビューとは異なるテーブル構造を持っています。

image::k8s/k8s-overview-perspective-table-comparison.png[オーバービュー・テーブルの構造比較]

xref:/use/views/k8s-custom-views.adoc[カスタムビューと] xref:/use/views/k8s-explore-views.adoc[エクスプローラビューの]場合、ビューに含まれる可能性のあるコンポーネントタイプは多様であるため、概要パースペクティブテーブルは、最も一般的なプロパティから構成される、汎用的な単一サイズの構造を持ちます。

image::k8s/k8s-overview-perspective-generic-table.png[概要表の一般的な構造]

概要パースペクティブのほとんどのビュー・タイプではテーブル・レイアウトが使用されますが、場合によってはカード・レイアウトも提供され、概要パースペクティブの内容を表示するさまざまなモードを切り替えることができます。

image::k8s/k8s-overview-perspective-cards-layout.png[カード・レイアウト]
