== Siehe Unterschied zur letzten Bereitstellung

Wenn nach der Anwendung von Änderungen an Ihren Kubernetes-Konfigurationen Probleme auftreten, werfen Sie einen Blick auf die letzte Bereitstellungsänderung.

image::k8s/k8s-show-diff.png[]

Ein Vergleich der aktuellen Konfiguration mit der vorherigen kann Ihnen helfen, die spezifischen Änderungen zu identifizieren, die das Problem verursacht haben könnten.

image::k8s/k8s-show-diff-opened.png[]
