= Tastaturkurzbefehle
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: SUSE Observabilität

Die folgenden Tastenkombinationen können zur Navigation in der SUSE Observability-Benutzeroberfläche verwendet werden.

|===
| Tastaturkürzel | Aktion

| `CTRL` + `SHIFT` + `F`
| Suchen Sie in der xref:/use/views/k8s-topology-perspective.adoc[Topologieperspektive] nach einer Komponente anhand ihres Namens.

| `CTRL` + `S`
| Überschreiben Sie zuvor gespeicherte Einstellungen für eine Ansicht. Dadurch wird die gespeicherte Ansicht aktualisiert, um alle Änderungen an den xref:/use/views/k8s-filters.adoc[Ansichtsfiltern] oder xref:/use/views/k8s-topology-perspective.adoc#_visualization_settings[Visualisierungseinstellungen] zu berücksichtigen.

| `CTRL` + `SHIFT` + `S`
| Speichern Sie die aktuellen Ansichtseinstellungen als neue Ansicht. Die neue Ansicht enthält alle derzeit angewandten xref:/use/views/k8s-filters.adoc[Ansichtsfilter] und xref:/use/views/k8s-topology-perspective.adoc#_visualization_settings[Visualisierungseinstellungen].

| `CTRL` + `SHIFT` + `R`
| Setzen Sie alle nicht gespeicherten Änderungen zurück und kehren Sie zur ursprünglichen Ansicht, Unteransicht oder zum Erkundungsmodus zurück. Dadurch werden alle nicht gespeicherten Änderungen an den xref:/use/views/k8s-filters.adoc[Ansichtsfiltern] oder xref:/use/views/k8s-topology-perspective.adoc#_visualization_settings[Visualisierungseinstellungen] zurückgesetzt.

| `SHIFT` + `1`
| Öffnen Sie in einer Ansicht, Unteransicht oder im Erkundungsmodus die xref:/use/views/k8s-topology-perspective.adoc[Topologieperspektive].

| `SHIFT` + `2`
| Öffnen Sie in einer Ansicht, Unteransicht oder im Erkundungsmodus die xref:/use/views/k8s-events-perspective.adoc[Ereignisperspektive].

| `SHIFT` + `3`
| Öffnen Sie in einer Ansicht, Unteransicht oder im Erkundungsmodus die xref:/use/views/k8s-metrics-perspective.adoc[Metrikperspektive].

| `SHIFT` + `4`
| Öffnen Sie in einer Ansicht, Unteransicht oder im Erkundungsmodus die xref:/use/views/k8s-traces-perspective.adoc[Perspektive "Spuren"].
|===
