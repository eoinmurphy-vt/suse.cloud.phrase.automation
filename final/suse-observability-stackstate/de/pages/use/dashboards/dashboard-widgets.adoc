= Widgets für das Armaturenbrett
:revdate: 2025-11-24
:page-revdate: {revdate}
:description: Visualisieren Sie Daten aus Ihren Ressourcen mit verschiedenen Widgets.

Widgets sind die Bausteine für Ihr Dashboard. Sie visualisieren Daten, in der Regel von einer oder mehreren Kubernetes-Ressourcen. Widgets können den Betrachtern des Dashboards auch statische Textinformationen anzeigen.

== Hinzufügen eines neuen Widgets

Sie können ein neues Widget auf eine der beiden Arten hinzufügen:

* Klicken Sie auf die Schaltfläche *Widget hinzufügen* oben rechts im Dashboard.

* Klicken Sie in einem leeren Dashboard auf den Abschnitt *Widget hinzufügen* in der Mitte des Bildschirms. Es öffnet sich eine Schublade, in der Sie das neue Widget konfigurieren können.

image::dashboards/add-a-widget.png[Verschiedene Möglichkeiten, ein Widget hinzuzufügen]

== Hinzufügen eines vorhandenen Widgets

Wenn Sie eine einzelne Komponente in {stackstate-product-name} anzeigen, können Sie Diagramme direkt zu einem Dashboard hinzufügen, indem Sie das Menü "Weitere Aktionen" verwenden und "Zum Dashboard hinzufügen" auswählen.

image::dashboards/add-existing-widget.png[Hinzufügen eines vorhandenen Widgets]

Diese Funktion ist auch für alle angehefteten Elemente verfügbar, und zwar über das gleiche Menü "Weitere Aktionen" in einem einzelnen Widget.

Außerdem können Sie alle angehefteten Elemente zu einem Dashboard hinzufügen, indem Sie die "Weitere Aktionen" in der Schublade für angeheftete Elemente verwenden oder auf die Schaltfläche "Alle Elemente zum Dashboard hinzufügen" am unteren Rand der Schublade klicken.

image::dashboards/add-all-pinned-items.png[Hinzufügen aller angehefteten Artikel]

Die daraus resultierenden Widgets im Dashboard enthalten einen Link zu der Ressource, aus der sie hinzugefügt wurden.

== Konfigurieren eines Widgets

Jedes Widget hat eine Reihe von Konfigurationsoptionen. Einige sind für alle Widgets gleich, andere hängen vom Widget-Typ ab.

Um die Konfiguration eines Widgets zu bearbeiten, verwenden Sie das Menü "Weitere Aktionen" und wählen Sie "Widget bearbeiten".

image::dashboards/configure-a-widget.png[Hinzufügen aller angehefteten Artikel]

=== Gemeinsame Widget-Felder

|===
| Name des Feldes | Beschreibung

| Name
| Ein kurzer, beschreibender Name, der in der Kopfzeile des Widgets angezeigt wird.

| Beschreibung
| Zusätzliche Details werden in der Kopfzeile des Widgets hinter einem Tooltip angezeigt.

| Typ
| Die <<types, Art der Visualisierung>>.
|===

Zusätzlich zu diesen Feldern gibt es einige gemeinsame Registerkarten für alle Widgets: 

|===
| Registerkarte Name | Beschreibung

| Links
| Links zu verwandten Ressourcen, die unterhalb des Namens des Widgets angezeigt werden. Der erste Link wird immer angezeigt. Weitere Links sind in einem Dropdown-Menü verfügbar.

| YAML
| Die rohe YAML-Konfiguration für das Widget, die eingesehen und bearbeitet werden kann.
|===


=== Registerkarte "Abfragen

Widget-Typen, die Daten aus Ressourcen anzeigen sollen, haben eine Registerkarte *Abfragen*. Sie können eine oder mehrere PromQL-Abfragen hinzufügen, um die Daten für das Widget abzurufen.


=== Typen [[types]]

==== TimeSeriesChart

Unter `TimeSeriesChart` werden Zeitreihendaten in einem Liniendiagramm dargestellt. Jede Abfrage in der Widget-Konfiguration erzeugt eine Linie im Diagramm.

image::dashboards/time-series-chart-example.png[Beispiel für ein Zeitreihendiagramm]

Sie können die folgenden Einstellungen konfigurieren:

* *Legende* - Die Position und der Modus der Legende.
* **Visuell** - Verbinden Sie Nullwerte, um eine durchgehende Linie zu erstellen.
* **Y-Achse** - Die Einheit, die Beschriftung und die Min/Max-Werte für die Y-Achse.
* **Schwellenwerte** - Fügen Sie dem Diagramm horizontale Linien hinzu, um wichtige Wertschwellen zu markieren.

==== BarChart

Unter `BarChart` werden Zeitreihendaten in einem Balkendiagramm dargestellt. Sie hat die gleichen Einstellungen für Legende, Y-Achse und Schwellenwerte wie die `TimeSeriesChart`.

image::dashboards/bar-chart-example.png[Beispiel Balkendiagramm]

==== StatChart

Die Website `StatChart` zeigt ein Sparkline-Diagramm und einen einzelnen Wert, der auf einen Blick leicht zu erkennen ist. Schwellenwerte können so konfiguriert werden, dass die Farbe des Textes auf der Grundlage des Wertes geändert wird.

image::dashboards/gauge-chart-example.png[Beispiel für ein Messdiagramm]

Sie können die folgenden Einstellungen konfigurieren:

* **Sparkline** - Blendet das Sparkline-Diagramm ein oder aus.
* **Einheit** - Die Einheit des Wertes.
* **Dezimalstellen** - Die Anzahl der anzuzeigenden Dezimalstellen.
* **Berechnung** - Die zur Berechnung des Wertes aus den Zeitreihendaten verwendete Methode.

==== GaugeChart

Auf `GaugeChart` wird eine Anzeige mit einem Wert in der Mitte angezeigt. Die Farbe der Anzeige und der Text ändern sich, wenn der Wert einen Schwellenwert überschreitet.

image::dashboards/stat-chart-example.png[Beispiel für ein statistisches Diagramm]

Sie können die folgenden Einstellungen konfigurieren:

* **Einheit** - Die Einheit des Wertes.
* **Dezimalstellen** - Die Anzahl der anzuzeigenden Dezimalstellen.
* **Max** - Der Höchstwert des Messgeräts.
* **Berechnung** - Die zur Berechnung des Wertes aus den Zeitreihendaten verwendete Methode.

==== Markdown

Das Widget `Markdown` zeigt Markdown-Text an. Damit können Sie Erklärungen oder andere Informationen zu Ihrem Dashboard hinzufügen.
