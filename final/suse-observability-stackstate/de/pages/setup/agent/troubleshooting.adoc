== Deaktivieren der Ablaufverfolgung für ein bestimmtes Netzwerkprotokoll im Prozess-Agenten

Um die Ablaufverfolgung für ein bestimmtes Netzwerkprotokoll im Prozess-Agenten zu deaktivieren, konfigurieren Sie die Helm-Diagrammeinstellung `processAgent.disabledProtocols`. Im folgenden Beispiel werden die Protokolle `http2` und `postgres` deaktiviert:

----
--set "processAgent.disabledProtocols[0]=http2" \
--set "processAgent.disabledProtocols[1]=postgres"
----

== Deaktivieren Sie den Prozess-Agenten

Wenn der Prozess-Agent in Ihrer Umgebung nicht unterstützt wird, deaktivieren Sie ihn über die Helm-Diagrammeinstellung `nodeAgent.containers.processAgent.enabled`:

----
--set nodeAgent.containers.processAgent.enabled=false
----
